# <a name="open-database-diagram-designer-visual-database-tools"></a>Открытие конструктора диаграмм баз данных (визуальные инструменты для баз данных)
Конструктор диаграмм баз данных представляет собой визуальное средство разработки и визуализации базы данных. Конструктор баз данных используется для создания, редактирования и удаления таблиц, столбцов, ключей, индексов, связей и ограничений. Для визуализации базы данных можно создать одну или несколько диаграмм, иллюстрирующих некоторые или все имеющиеся в ней таблицы, столбцы, ключи и связи.  
  
Конструктор диаграмм баз данных автоматически запускается при открытии новой или уже существующей диаграммы.  
  
### <a name="open-a-new-database-diagram"></a>Открытие новой диаграммы базы данных  
  
1.  В обозревателе объектов щелкните правой кнопкой узел диаграммы базы данных.  
  
2.  В раскрывающемся меню щелкните пункт **Новая диаграмма базы данных**.  
  
3.  В диалоговом окне **Добавить таблицу** выберите таблицы, на основе которых необходимо построить диаграмму.  
  
    > [!NOTE]  
    > Таблицы могут быть добавлены позже после выбора пункта меню **Закрыть** .  
  
4.  Меню **Диаграмма базы данных** будет добавлено в главное меню, и при этом откроется панель конструктора.  
  
### <a name="open-an-existing-database-diagram"></a>Открытие уже существующей диаграммы базы данных  
  
1.  В обозревателе объектов щелкните правой кнопкой мыши диаграмму под узлом диаграммы базы данных.  
  
2.  В раскрывающемся меню щелкните пункт **Создать диаграмму базы данных**.  
  
3.  Меню **Диаграмма базы данных** будет добавлено в главное меню и открыто с помощью панели конструктора.
