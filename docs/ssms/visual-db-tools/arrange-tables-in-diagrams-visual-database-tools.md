# <a name="arrange-tables-in-diagrams-visual-database-tools"></a>Размещение таблиц в диаграммах (визуальные инструменты для баз данных)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Для наглядного представления каждой таблицы и связей между таблицами можно быстро разместить таблицы в диаграмме базы данных.  
  
### <a name="to-arrange-tables-in-diagrams"></a>Размещение таблиц в диаграммах  
  
1.  Щелкните правой кнопкой мыши в пустом пространстве окна «Диаграмма базы данных».  
  
2.  В контекстном меню выберите **Упорядочить таблицы**.
