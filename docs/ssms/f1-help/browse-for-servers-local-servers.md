# <a name="browse-for-servers-local-servers"></a>Обзор серверов (локальные серверы)
Если при подключении к компоненту [!INCLUDE[msCoName](../../includes/msconame_md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] неизвестно точное имя [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)], в поле **Имя сервера** нажмите кнопку **Обзор**, чтобы открыть диалоговое окно **Выбор серверов** .  
  
## <a name="options"></a>Параметры  
**Выберите сервер для соединения**  
Укажите сервер, с которым нужно соединиться, выбрав экземпляр [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] , показанный в дереве. Можно показывать или скрывать участки дерева, щелкнув узел, помеченный знаком **+** или **–** .
