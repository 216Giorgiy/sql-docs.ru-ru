# <a name="browse-for-objects-object-explorer"></a>Обзор объектов (обозреватель объектов)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Отображает список объектов выбранного типа. Выберите один или несколько объектов из списка, пометив соответствующие флажки. В окне **Обзор объектов** отображаются только элементы, соответствующие действию, и элементы, на просмотр которых есть разрешение.  
  
## <a name="options"></a>Параметры  
**Флажок**  
Выберите, чтобы включить объект в список.  
  
**Значок**  
Значок, представляющий тип объектов.  
  
**Название**  
Имя объекта.  
  
**Тип**  
Тип объекта базы данных.
