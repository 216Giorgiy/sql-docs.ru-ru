# <a name="save-changes"></a>Сохранить изменения
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Выдается предостережение, что изменения перечисленных элементов не будут сохранены, пока не нажать **Да**.  
  
Это диалоговое окно может появиться при закрытии решений, планов обслуживания или других открытых файлов.  
  
### <a name="to-display-the-save-changes-dialog-box"></a>Отображение диалогового окна «Сохранить изменения»  
  
-   В меню **Файл** нажмите **Закрыть**применительно к новому открытому файлу.  
  
Изменения элементов, уже сохранявшихся ранее, будут автоматически сохранены в месте их хранения в хранилище. Если элемент еще не сохранялся, появляется диалоговое окно **Сохранить файл как** , в котором предлагается указать новое место хранения.  
  
## <a name="options"></a>Параметры  
**Сохранить изменения следующих элементов?**  
Выбранные элементы будут сохранены. Элементы можно выбирать и сбрасывать.  
  
> [!NOTE]  
> При сохранении задайте только имя объекта, а не схему и имя объекта.
