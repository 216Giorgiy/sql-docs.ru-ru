# <a name="copy-items-in-a-solution"></a>Копирование элементов в решении
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
Копировать элементы можно с помощью обозревателя решений или проводника Windows.  
  
### <a name="to-copy-items-within-solution-explorer"></a>Копирование элементов с помощью обозревателя решений  
  
1.  В обозревателе решений выберите нужный элемент.  
  
2.  В меню **Правка** выберите пункт **Копировать**.  
  
3.  В обозревателе решений выберите целевой проект.  
  
4.  В меню **Правка** выберите пункт **Вставить**.  
  
> [!NOTE]  
> Соединения копировать между проектами нельзя. При копировании в другой проект запроса со связанными соединениями эти соединения в целевой проект не копируются. В целевом проекте соединение необходимо создать вручную.
