# <a name="view-project-properties"></a>Просмотр свойств проекта
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] Свойства, связанные с проектами, можно просмотреть в окне "Свойства".  
  
### <a name="to-view-project-properties"></a>Просмотр свойств проекта  
  
1.  В обозревателе решений выберите проект.  
  
2.  Если окно "Свойства" не открыто, откройте меню **Вид** и выберите пункт **Окно "Свойства"**.  
  
3.  В окне «Свойства» можно просматривать свойства проекта.  
  
Свойства проекта, например дата создания, доступны только для чтения и не могут быть изменены в окне «Свойства».
