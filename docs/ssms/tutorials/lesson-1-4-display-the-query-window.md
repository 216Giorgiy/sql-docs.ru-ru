# <a name="lesson-1-4---display-the-query-window"></a>Занятие 1–4. Отображение окна запроса
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)] При открытии нескольких окон запросов они отображаются в виде окон документов со вкладками.  
  
1.  На главной панели инструментов нажмите кнопку **Запрос компонента Database Engine**. В диалоговом окне **Подключиться к компоненту Database Engine** нажмите кнопку **Соединить**.  
  
2.  В обозревателе объектов щелкните правой кнопкой мыши свой сервер и выберите команду **Создать запрос**. Теперь редактор запросов будет использовать сведения о соединении, полученные с зарегистрированного сервера.  
  
    Обратите внимание, что окна отображаются как окна со вкладками справа от обозревателя объектов.
