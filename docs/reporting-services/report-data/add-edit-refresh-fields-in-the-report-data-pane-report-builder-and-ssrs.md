# Добавление, изменение и обновление полей в области данных отчета (построитель отчетов и службы SSRS)
  Поля набора данных являются встроенной коллекцией имен полей, представляющей данные, возвращаемые при выполнении запроса к внешнему источнику данных.  
  
 Для внедренного набора данных полями набора данных являются поля, созданные после завершения построения запроса и закрытия панели «Конструктор запросов», а также создаваемые вычисляемые поля.  
  
 Для общего набора данных полями набора данных являются поля определения общего набора данных при его добавлении к отчету. Хотя запрос из общего набора данных на сервере отчетов всегда используется при выполнении отчета, список полей наборов данных в отчете является статичным.  
  
 Чтобы обновить список полей в отчете для соответствия текущему списку полей в запросе общего набора данных используйте кнопку **Обновить поля** . Обновление полей не влияет на вычисляемые поля, определенные в отчете.  
  
> [!NOTE]  
>  [!INCLUDE[ssRBRDDup](../../includes/ssrbrddup-md.md)]  
  
### Добавление поля запроса  
  
1.  В области данных отчета щелкните правой кнопкой мыши набор данных и выберите команду **Добавить поле запроса**.  
  
    > [!NOTE]  
    >  Если область данных отчета не видна, в меню **Вид** выберите пункт **Данные отчета**.  
  
2.  На странице **Поля** диалогового окна **Свойства набора данных** нажмите кнопку **Добавить**, затем выберите **Поле запроса**. Новая строка добавляется в сетку снизу.  
  
3.  В текстовое поле **Имя поля** введите имя поля.  
  
    > [!NOTE]  
    >  В пределах набора данных имена должны быть уникальными.  
  
4.  В текстовое поле **Источник поля** введите имя существующего поля источника данных.  
  
5.  [!INCLUDE[clickOK](../../includes/clickok-md.md)]  
  
### Добавление вычисляемого поля  
  
1.  В области данных отчета щелкните правой кнопкой мыши набор данных и выберите команду **Добавить вычисляемое поле**.  
  
2.  На странице **Поля** диалогового окна **Свойства набора данных** нажмите кнопку **Добавить**и выберите **Вычисляемое поле**. Новая строка добавляется в сетку снизу.  
  
3.  В текстовое поле **Имя поля** введите имя поля.  
  
    > [!NOTE]  
    >  В пределах набора данных имена должны быть уникальными.  
  
4.  В текстовое поле **Источник поля** введите выражение для поля. Нажмите кнопку выражения (**fx**), чтобы создать выражение.  
  
    > [!NOTE]  
    >  Выражение для вычисляемого поля не может содержать статистических выражений или ссылок на элементы отчета.  
  
5.  [!INCLUDE[clickOK](../../includes/clickok-md.md)]  
  
### Изменение поля запроса или поля набора данных  
  
1.  В области данных отчета щелкните правой кнопкой мыши поле и выберите пункт **Свойства поля**.  
  
2.  На странице **Поля** диалогового окна **Свойства набора данных** щелкните существующее поле, чтобы выбрать строку.  
  
3.  Измените имя или значение поля.  
  
4.  [!INCLUDE[clickOK](../../includes/clickok-md.md)]  
  
### Удаление поля запроса или вычисляемого поля  
  
1.  В области данных отчета разверните набор данных, чтобы отобразить коллекцию полей.  
  
2.  Щелкните правой кнопкой мыши поле, которое хотите удалить, и выберите пункт **Удалить**.  
  
### Обновление коллекции полей в области данных отчета для общего набора данных  
  
1.  В области данных отчета щелкните правой кнопкой мыши набор данных и выберите пункт **Запрос**.  
  
2.  Нажмите кнопку **Обновить поля**.  
  
     На сервере отчетов выполняется запрос общего набора данных, который возвращает коллекцию текущих полей.
