# <a name="using-my-reports-report-builder-and-ssrs"></a>Использование папки «Мои отчеты» (построитель отчетов и службы SSRS)
  На сервере отчетов, настроенном в собственном режиме, папка «Мои отчеты» представляет личную рабочую область, которую можно использовать для хранения и использования собственных отчетов. Другие папки сервера отчетов являются общедоступными, и пользователи обычно должны иметь дополнительные разрешения, чтобы добавлять или изменять содержимое. В отличие от этого папка «Мои отчеты» является рабочим пространством, которое управляется пользователем. Можно удалять и добавлять отчеты и папки и сохранять связанные отчеты с персонифицированными настройками.  
  
 По сути папка «Мои отчеты» аналогична папке «Мои документы» в файловой системе Windows. Хотя эта папка предусмотрена для всех пользователей, у каждого из них она отличается от других. За исключением администраторов сервера отчетов, другие пользователи не могут открывать содержимое папки «Мои отчеты», принадлежащей конкретному пользователю.  
  
 Возможность использования папки «Мои отчеты» не является обязательной и может быть отключена администратором сервера отчетов. Если эта возможность задействована, то папка «Мои отчеты» будет находиться в корневой папке, к которой можно получить доступ с помощью диспетчера отчетов или веб-браузера. Дополнительные сведения см. в разделе [Поиск и просмотр отчетов в диспетчере отчетов (построитель отчетов и службы SSRS)](finding-and-viewing-reports-with-a-browser-report-builder-and-ssrs.md).  
  
 На сервере отчетов, настроенном в режиме интеграции с SharePoint, отсутствует эквивалент папки «Мои отчеты». Дополнительные сведения см. в разделе [Поиск, просмотр отчетов и управление ими (построитель отчетов и службы SSRS)](../../reporting-services/report-builder/finding-viewing-and-managing-reports-report-builder-and-ssrs.md).

## <a name="ways-to-use-my-reports"></a>Способы использования папки «Мои отчеты»  
 Папка «Мои отчеты» пуста до тех пор, пока в нее не будут добавлены отчеты, папки и другие элементы. Ниже приводится несколько способов добавления содержимого в «Мои отчеты».  
  
-   Создание личного связанного отчета и сохранение его в папке «Мои отчеты». Не все отчеты доступны для связывания. Дополнительные сведения см. в разделе [Создание связанного отчета](../../reporting-services/reports/create-a-linked-report.md).  
  
-   Передача RDL-файла с определением отчета, SMDL-файла модели отчета или других файлов из файловой системы. Можно передать любой файл, но сервер отчетов обрабатывает только RDL-файлы и SMDL-файлы отчетов. Дополнительные сведения см. в разделе "Определения отчета" [документации по службам Reporting Services](http://go.microsoft.com/fwlink/?linkid=121312) в электронной документации по SQL Server и [Передача файла или отчета (диспетчер отчетов)](../../reporting-services/reports/upload-a-file-or-report-report-manager.md).  
  
-   Создание и публикация собственных отчетов в папке «Мои отчеты». Дополнительные сведения см. в разделе [Представление конструктора отчетов (построитель отчетов)](../../reporting-services/report-builder/report-design-view-report-builder.md).  
  
 Обычно разрешения для папки «Мои отчеты» дают возможность управления папкой. Тем не менее только администратор сервера отчетов определяет, какие задачи могут выполнять пользователи. Если у пользователя недостаточно разрешений для работы с папкой «Мои отчеты», следует проконсультироваться с администратором сервера отчетов.  
  
## <a name="searching-my-reports"></a>Поиск в папке «Мои отчеты»  
 При поиске в базе данных сервера отчетов содержимое папки «Мои отчеты» текущего пользователя включается в поиск, тогда как содержимое таких папок других пользователей — нет. В результаты поиска включаются только те отчеты, к которым у пользователя есть доступ.
