# <a name="how-to-find-the-product-key-for-sql-server-2017-reporting-services"></a>Как найти ключ продукта для SQL Server Reporting Services 2017

[!INCLUDE[ssrs-appliesto](../../includes/ssrs-appliesto.md)] [!INCLUDE[ssrs-appliesto-2017-and-later](../../includes/ssrs-appliesto-2017-and-later.md)] [!INCLUDE[ssrs-appliesto-not-pbirsi](../../includes/ssrs-appliesto-not-pbirs.md)]

Узнайте, как найти ключ продукта для SQL Server Reporting Services 2017 (SSRS), чтобы установить сервер в рабочей среде.

Чтобы найти ключ продукта, сначала скачайте и запустите программу установки SQL Server 2017.

1. Скачать SQL Server 2017 можно в одном из следующих мест:

    - Volume Licensing Service Center;
    - подписка MSDN;
    - розничный магазин (скачивание из Microsoft Store).

1. Запустите программу установки SQL Server 2017 и скопируйте предварительно заполненный ключ:

    ![Копирование ключа продукта для SQL Server 2017](media/find-reporting-services-product-key-ssrs/ssrs-ss2017-copy-product-key.png)

1. [Запустите программу установки служб Reporting Services](install-reporting-services.md) и вставьте ключ:

     ![Вставка ключа продукта](media/find-reporting-services-product-key-ssrs/ssrs-ssrs2017-paste-product-key.png)

Это действие необходимо выполнить только при первой установке служб SSRS 2017. При обновлении служб вводить ключ не потребуется.
