# <a name="changelog-for-the-report-viewer-controls"></a>Журнал изменений для элементов управления средства просмотра отчетов

Этот журнал изменений отслеживает выпуски элементов управления средства просмотра отчетов WinForms и WebForms.

## <a name="15900148"></a>15.900.148
В этом обновлении:
 - Исправлены ошибки, препятствующие загрузке отчетов без параметров через Server.LoadReportDefinition.

Элемент управления средства просмотра отчетов [WebForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Webforms/150.900.148)
 - Поддерживает внедрение на страницах с написанием справа налево (на страницах, которые изменяют поток текста с помощью свойства CSS *direction: rtl;*).
 - Поддерживает настройку печатного текста в диалоговом окне с помощью интерфейса локализации *IReportViewerMessages5*.
 - Улучшена поддержка специальных возможностей.

Элемент управления средства просмотра отчетов [WinForms](https://www.nuget.org/packages/Microsoft.ReportingServices.ReportViewerControl.Winforms/150.900.148)
 - Исправлены ошибки при печати, когда приложение выполняется в режиме высокого разрешения.
