# Настройки сведений об устройстве RGDI
  В следующей таблице указаны настройки сведений об устройстве для подготовки отчетов к просмотру в формате RGDI.  
  
|Настройка|Значение|  
|-------------|-----------|  
|**Страница**|Задает страницу отчета, которую необходимо подготовить к просмотру. Значение по умолчанию — **1**.|  
|**RGDIVersion**|Этот параметр не действует.|
