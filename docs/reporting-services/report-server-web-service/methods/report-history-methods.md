# <a name="report-history-methods"></a>Методы журнала отчета
  Эти методы позволяют создавать моментальные снимки журнала элементов и управлять ими.  
  
|Метод|Действие|  
|------------|------------|  
|<xref:ReportService2010.ReportingService2010.CreateItemHistorySnapshot%2A>|Создает моментальный снимок журнала элементов указанного элемента каталога.|  
|<xref:ReportService2010.ReportingService2010.DeleteItemHistorySnapshot%2A>|Удаляет отдельный моментальный снимок журнала элементов указанного элемента каталога.|  
|<xref:ReportService2010.ReportingService2010.GetItemHistoryLimit%2A>|Возвращает предельное количество моментальных снимков журнала для указанного элемента каталога.|  
|<xref:ReportService2010.ReportingService2010.GetItemHistoryOptions%2A>|Возвращает значения параметров и свойств моментального снимка журнала элемента, созданного для элемента каталога.|  
|<xref:ReportService2010.ReportingService2010.ListItemHistory%2A>|Возвращает список моментальных снимков журнала элементов и их свойства для указанного элемента каталога.|  
|<xref:ReportService2010.ReportingService2010.SetItemHistoryLimit%2A>|Указывает, сколько моментальных снимков элемента сохраняет сервер отчетов.|  
|<xref:ReportService2010.ReportingService2010.SetItemHistoryOptions%2A>|Устанавливает параметры журнала элементов, указанные при создании моментального снимка журнала элементов для указанного элемента каталога.|
