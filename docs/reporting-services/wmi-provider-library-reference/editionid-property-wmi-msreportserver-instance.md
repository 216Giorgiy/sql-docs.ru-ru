# Свойство EditionID (WMI MSReportServer_Instance)
  Возвращает выпуск сервера отчетов. Только для чтения.  
  
## Синтаксис  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## Значение свойства  
 Значение **uint** , которое содержит выпуск сервера отчетов.
