# Свойство EditionName (WMI MSReportServer_Instance)
  Возвращает имя выпуска сервера отчетов в немашинном (предназначенном для человека) формате. Только для чтения.  
  
## Синтаксис  
  
```vb  
Public Dim EditionName As String  
```  
  
```csharp  
public string EditionName;  
```  
  
## Значение свойства  
 Строка **string** , которая содержит понятное для пользователя имя выпуска сервера отчетов.
