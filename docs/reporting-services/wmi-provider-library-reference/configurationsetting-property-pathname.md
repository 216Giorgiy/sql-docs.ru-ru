# <a name="configurationsetting-property---pathname"></a>Свойство ConfigurationSetting - путь
  Задает путь установки для экземпляра сервера отчетов. Только для чтения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
public Dim PathName As String  
```  
  
```csharp  
public string PathName;  
```  
  
## <a name="property-values"></a>Значения свойств  
 Объект **String** , который представляет путь установки.
