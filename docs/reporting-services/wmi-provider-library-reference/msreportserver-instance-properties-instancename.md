# <a name="msreportserverinstance-properties---instancename"></a>Свойства MSReportServer_Instance — InstanceName
  Возвращает имя экземпляра сервера отчетов на заданном компьютере. Только для чтения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
Public Dim InstanceName As String  
```  
  
```csharp  
public string InstanceName;  
```  
  
## <a name="property-value"></a>Значение свойства  
 Объект **String** , представляющий имя экземпляра сервера отчетов.
