# <a name="configurationsetting-property---machineaccountidentity"></a>Свойство ConfigurationSetting — MachineAccountIdentity
  Возвращает удостоверение учетной записи компьютера, на котором установлен сервер отчетов.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## <a name="property-values"></a>Значения свойств  
 Значение **String** , которое содержит идентификатор учетной записи компьютера, на котором установлен сервер отчетов.
