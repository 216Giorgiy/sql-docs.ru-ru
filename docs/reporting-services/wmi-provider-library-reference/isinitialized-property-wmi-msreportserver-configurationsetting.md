# Свойство IsInitialized (WMI MSReportServer_ConfigurationSetting)
  Указывает, инициализирован ли экземпляр сервера отчетов. Только для чтения.  
  
## Синтаксис  
  
```vb  
Public Dim IsInitialized As Boolean  
```  
  
```csharp  
public Boolean DatabaseLogonName;  
```  
  
## Значения свойств  
 Объект **Boolean** . Указывает, инициализирован ли экземпляр сервера отчетов.
