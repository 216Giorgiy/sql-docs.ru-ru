# <a name="msreportserverinstance-properties---editionid"></a>Свойства MSReportServer_Instance — EditionID
  Возвращает выпуск сервера отчетов. Только для чтения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
Public Dim EditionID As UInt  
```  
  
```csharp  
public uint EditionID;  
```  
  
## <a name="property-value"></a>Значение свойства  
 Значение **uint** , которое содержит выпуск сервера отчетов.
