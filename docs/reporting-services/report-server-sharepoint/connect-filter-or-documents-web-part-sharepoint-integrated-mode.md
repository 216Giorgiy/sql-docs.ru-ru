# <a name="connect-filter-or-documents-web-part-with-a-reporting-services-report-viewer-web-part"></a>Подключение веб-часть фильтра или документов с веб-части средства просмотра отчетов Reporting Services

[!INCLUDE[ssrs-appliesto](../../includes/ssrs-appliesto.md)] [!INCLUDE[ssrs-appliesto-2016](../../includes/ssrs-appliesto-2016.md)] [!INCLUDE[ssrs-appliesto-sharepoint-2013-2016i](../../includes/ssrs-appliesto-sharepoint-2013-2016.md)] [!INCLUDE[ssrs-appliesto-not-pbirsi](../../includes/ssrs-appliesto-not-pbirs.md)]

[!INCLUDE [ssrs-previous-versions](../../includes/ssrs-previous-versions.md)]

При использовании продукта SharePoint, можно создать часть панели мониторинга или веб-страницу, включающую веб-часть фильтра или веб-часть документов и веб-части средства просмотра отчетов. Поддерживается версия [!INCLUDE[SPF2010](../../includes/spf2010-md.md)] или [!INCLUDE[SPS2010](../../includes/sps2010-md.md)]. Также поддерживаются версии [!INCLUDE[winSPServ3](../../includes/winspserv3-md.md)] и [!INCLUDE[offSPServ](../../includes/offspserv-md.md)] 2007. Подключив фильтр веб-части, пользователи выбирают значения фильтра в веб-часть фильтра можно отправить значение в параметризованный отчет на той же странице. Подключив веб-часть документов, пользователи, выбравшие отчет в библиотеке «Документы» можно просмотреть отчет в смежных веб-части средства просмотра отчетов.

> [!NOTE]
> Интеграция служб Reporting Services с SharePoint больше не доступны после SQL Server 2016.

 Веб-часть фильтра используется для передачи значений в один или несколько параметров отчета. Чтобы использовать веб-часть фильтра, отчет должен иметь параметры, определенные для нее были совместимы со значениями, тип данных и форматом, посылаемыми в веб-часть.  
  
 Веб-часть документы связан с библиотекой документов на сайте домашней. Чтобы просмотреть, добавить или удалить элементы библиотеки «Документы», выберите **Просмотреть все содержимое веб-сайта**. В меню «Библиотеки» выберите **Документы**. Для управления элементами библиотеки «Документы» можно использовать меню **Создание**, **Передача**и **Действия** .  
  
## <a name="connect-a-filter-web-part"></a>Подключение веб-части фильтра
  
1.  Откройте или создайте страницу веб-части или панели мониторинга.  
  
2.  В меню **Действия веб-сайта** выберите команду **Редактировать страницу**.  
  
3.  Нажмите кнопку **добавить веб-часть**.  
  
4.  В **все веб-части**в **Разное** категории, выберите **просмотра отчетов служб отчетов SQL Server**.  
  
5.  Нажмите кнопку **Добавить**. Веб-часть будет добавлена в верхнюю часть зоны.  
  
6.  В другой области же странице или панели мониторинга, щелкните **добавить веб-часть**.  
  
7.  В **все веб-части**в **фильтры** выберите веб-части.  
  
8.  Нажмите кнопку **Добавить**. Веб-часть будет добавлена в верхнюю часть зоны.  
  
9. В области, содержащей веб-части, выберите веб-часть **изменить** последовательно выберите пункты **подключений**, пункты **отправлять значения фильтра для**и выберите **отчетов Средство просмотра** - *имя отчета*.  
  
10. Проверьте изменения и сохраните страницу.  
  
## <a name="connect-a-documents-web-part"></a>Подключение веб-части документов  
  
1.  Откройте или создайте страницу веб-части или панели мониторинга.  
  
2.  В меню **Действия веб-сайта** выберите команду **Редактировать страницу**.  
  
3.  Нажмите кнопку **добавить веб-часть**.  
  
4.  В **все веб-части**в **списки и библиотека** выберите **документов.**  
  
5.  Нажмите кнопку **Добавить**. Веб-часть будет добавлена в верхнюю часть зоны.  
  
6.  Нажмите кнопку **Применить** в нижней части панели средств, а затем кнопку **ОК** , чтобы закрыть панель.  
  
7.  В другой области же странице или панели мониторинга, щелкните **добавить веб-часть**.  
  
8.  В **все веб-части**в **Разное** категории, выберите **просмотра отчетов служб отчетов SQL Server.**  
  
9. Нажмите кнопку **Добавить**. Веб-часть будет добавлена в верхнюю часть зоны.  
  
10. В области, содержащей веб-части, выберите веб-часть **изменить** последовательно выберите пункты **подключений**, пункты **получить определения отчетов из**и выберите  **Документы**.  
  
11. Проверьте изменения и сохраните страницу.
