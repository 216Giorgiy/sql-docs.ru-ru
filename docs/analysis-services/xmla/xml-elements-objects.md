# <a name="xml-elements---objects"></a>Элементы XML - объектов
[!INCLUDE[ssas-appliesto-sqlas-aas](../../includes/ssas-appliesto-sqlas-aas.md)]
  Протокол XML для аналитики (XMLA) используется два метода, **Discover** и **Execute**, которые предлагают стандартный способ для приложений для доступа к данным в экземпляре служб Analysis Services. Поскольку эти методы вызываются с помощью протокола SOAP, получение входных данных и формирование выходных данных происходит в формате XML.  
  
## <a name="in-this-section"></a>В этом разделе  
 Ниже описываются объекты XMLA, реализованные службами Analysis Services.  
  
|Метод|Описание|  
|------------|-----------------|  
|[Элемент DiscoverResponse &#40;XML для Аналитики&#41;](../../analysis-services/xmla/xml-elements-objects-discoverresponse.md)|Содержит информацию, возвращаемую экземпляром служб Analysis Services в ответ на [Discover](../../analysis-services/xmla/xml-elements-methods-discover.md) вызова метода.|  
|[Элемент ExecuteResponse &#40;XML для Аналитики&#41;](../../analysis-services/xmla/xml-elements-objects-executeresponse.md)|Содержит информацию, возвращаемую экземпляром служб Analysis Services в ответ на [Execute](../../analysis-services/xmla/xml-elements-methods-execute.md) вызова метода.|
