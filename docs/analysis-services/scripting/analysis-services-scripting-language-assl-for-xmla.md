# <a name="analysis-services-scripting-language-assl-for-xmla"></a>Язык ASSL (ASSL XML для Аналитики)
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]
  Приложения взаимодействуют с [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)] с помощью команд XML и описания объектов. Язык сценариев служб Analysis Services (ASSL) представляет собой фрагмент языка для описания объектов внутри этого диалекта XML.  
  
 В этом разделе справочника содержится описание синтаксиса и сведения об использовании каждого элемента в схеме языка ASSL.  
  
 Администраторы и разработчики могут использовать этот раздел справочника в качестве основных сведений о скриптах, создаваемых с помощью среды [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)], [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)], объектов AMO и других приложений, а также для помощи в создании или изменении скриптов для своих нужд.
