Дополнительные сведения см. в разделе [Использование свойств ячеек (многомерные выражения)](../../../analysis-services/multidimensional-models/mdx/using-cell-properties-mdx.md). В многомерных выражениях также поддерживается присваивание значений свойству **NON_EMPTY_BEHAVIOR** в инструкции [CREATE MEMBER](../Topic/CREATE%20MEMBER%20Statement%20\(MDX\).md).
