# <a name="use-a-template-to-create-an-action"></a>Использование шаблона для создания действия
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]**Действия** представление в конструкторе кубов содержит несколько шаблонов, которые можно использовать для создания типовых операций. Доступ к конструктору кубов можно получить из среды [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)].  
  
> [!NOTE]  
>  Пользователи получают доступ к созданной операции после повторной обработки или обновления куба. Дополнительные сведения см. в разделе [Обработка многомерной модели (службы Analysis Services)](../../analysis-services/multidimensional-models/processing-a-multidimensional-model-analysis-services.md).  
  
### <a name="to-use-a-template-to-create-an-action"></a>Чтобы использовать шаблон для создания операции  
  
-   Откройте куб, для которого необходимо создать операцию, и в разделе **Конструирование**выберите вкладку **Действия** .  
  
-   В разделе **Средства вычисления**выберите вкладку **Шаблоны** , разверните узел, в котором находится тип необходимой операции, и дважды щелкните шаблон, который хотите использовать.  
  
-   Заполните необходимую информацию для завершения создания операции по шаблону.
