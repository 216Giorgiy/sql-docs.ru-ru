# <a name="attribute-properties---bind-an-attribute-to-a-key-column"></a>Атрибут свойства - привязка атрибута к ключевому столбцу
  Данная процедура описывает изменение настройки свойства **Name** для атрибута измерения.  
  
### <a name="to-bind-an-attribute-to-a-key-column"></a>Привязка атрибута к ключевому столбцу  
  
1.  В конструкторе измерений откройте измерение с атрибутом, который необходимо переименовать.  
  
2.  На панели **Атрибуты** представления построителя измерений измените формат отображения, выбрав пункт **Дерево** или **Список**. Дополнительные сведения об изменении формата в области "Атрибуты" см. в разделе [Просмотр атрибутов в дереве, списке или сетке в конструкторе измерений](../../analysis-services/multidimensional-models/view-attributes-in-dimension-designer.md).  
  
3.  Щелкните правой кнопкой мыши атрибут, который требуется настроить, и выберите пункт **Переименовать**.  
  
4.  Введите новое имя.  
  
    > [!NOTE]  
    >  Также можно установить свойство **Name** выбранного атрибута в окне **Свойства** среды [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)].
