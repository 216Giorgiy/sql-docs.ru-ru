# <a name="file-load-and-save-event-category"></a>Категория событий «Загрузка и сохранение файлов»

|Класс событий|Идентификатор события|Description|  
|-----------------|--------------|-----------------|  
|Начало загрузки файла|90|Начало загрузки файла|  
|Окончание загрузки файла|91|Окончание загрузки файла|  
|Начало сохранения файла|92|Начало сохранения файла|  
|Окончание сохранения файла|93|Окончание сохранения файла|  
|Начало удаления страницы|94|Начало удаления страницы|  
|Окончание удаления страницы|95|Окончание удаления страницы|  
|Начало загрузки страницы|96|Начало загрузки страницы|  
|Окончание загрузки страницы|97|Окончание загрузки страницы|  
  
 Дополнительные сведения о столбцах, связанных с классами событий «Загрузка и сохранение файла», см. в разделе [File Load and Save Data Columns](../../analysis-services/trace-events/file-load-and-save-data-columns.md).
