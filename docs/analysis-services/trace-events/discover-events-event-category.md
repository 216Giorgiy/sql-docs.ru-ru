# Категория событий &#171;События обнаружения&#187;
  Категория событий «Распознавание событий» содержит классы событий, описанные в следующей таблице.  
  
|Класс событий|Идентификатор события|Description|  
|-----------------|--------------|-----------------|  
|Начало распознавания|36|Позволяет собрать все события начала распознавания с момента запуска трассировки.|  
|Завершение распознавания|38|Позволяет собрать все события завершения распознавания с момента запуска трассировки.|  
  
 Дополнительные сведения о столбцах, связанных с каждым из классов событий «Распознавание событий», см. в разделе [Discover Events Data Columns](../../analysis-services/trace-events/discover-events-data-columns.md).
