# Категория событий &#171;Определение состояния сервера&#187;
  Категория событий «Определение состояния сервера» содержит классы событий, описанные в следующей таблице.  
  
|Класс событий|Идентификатор события|Description|  
|-----------------|--------------|-----------------|  
|Начало определения состояния сервера|33|Позволяет собрать все события начала определения состояния сервера XML для аналитики с момента начала трассировки.|  
|Данные определения состояния сервера|34|Позволяет собрать все события данных определения состояния сервера XML для аналитики с момента начала трассировки. В этих событиях фиксируется содержимое ответа на запрос распознавания.|  
|Окончание определения состояния сервера|35|Позволяет собрать все события окончания определения состояния сервера XML для аналитики с момента начала трассировки.|  
  
 Дополнительные сведения о столбцах, связанных с каждым из классов событий «События запросов», см. в разделе [Discover Server State Events Data Columns](../../analysis-services/trace-events/discover-server-state-events-data-columns.md).
