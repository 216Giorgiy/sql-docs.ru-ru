# <a name="queries-events-category"></a>Категория «События запросов»
  Категория событий «События запросов» содержит классы событий, описанные в следующей таблице:  
  
|Класс событий|Идентификатор события|Description|  
|-----------------|--------------|-----------------|  
|Начало запроса|9|Позволяет собрать все события начала запросов с момента запуска трассировки.|  
|Окончание запроса|10|Позволяет собрать все события окончания запросов с момента запуска трассировки.|  
  
 Дополнительные сведения о столбцах, связанных с каждым из классов событий «События запросов», см. в разделе [Queries Events Data Columns](../../analysis-services/trace-events/queries-events-data-columns.md).
