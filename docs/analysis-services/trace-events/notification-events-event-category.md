# <a name="notification-events-event-category"></a>категория событий «События уведомления»
[!INCLUDE[ssas-appliesto-sqlas](../../includes/ssas-appliesto-sqlas.md)]Категория событий «события уведомления» содержит классы событий, описанные в следующей таблице.  
  
|Класс событий|Идентификатор события|Description|  
|-----------------|--------------|-----------------|  
|Уведомление|39|Позволяет собрать все события уведомления после запуска трассировки.|  
|Определяемые пользователем|40|Собирает определяемые пользователем события запуска команд с момента начала трассировки.|  
  
 Дополнительные сведения о столбцах, связанных с каждым из классов событий «События уведомления», см. в разделе [Notification Events Data Columns](../../analysis-services/trace-events/notification-events-data-columns.md).
