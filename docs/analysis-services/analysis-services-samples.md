## <a name="sample-solutions-and-databases"></a>Примеры решений и баз данных  

Пока больше официально не поддерживается, или обновлены Adventure Works остается одним из наиболее Инклюзивные и надежной образцы наборов данных для изучения и тестирования служб Analysis Services. Анализа служб образцы проектов баз данных, а также примеры в документации по, записи в блогах и презентациям использовать образцы данных Adventure Works 2014.

[Загрузите образцы проектов Adventure Works и баз данных, на сайте Codeplex](https://msftdbprodsamples.codeplex.com/releases/view/125550).

Для SQL Server 2016 появилась новую коллекцию Wide World Importers образца данных. Образец данных Wide World Importers можно использовать для изучения и тестирования служб Analysis Services; Однако не учебники, примеры или документации не имеется. Компания Wide World Importers набора данных и схемы не поддерживают многие из функций в службах Analysis Services. Дополнительные сведения см. в разделе [Wide World Importers](https://msdn.microsoft.com/library/mt734199).
