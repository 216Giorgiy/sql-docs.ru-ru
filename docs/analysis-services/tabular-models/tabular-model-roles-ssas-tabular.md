# <a name="tabular-model-roles-ssas-tabular"></a>Роли табличных моделей (табличные службы SSAS)
  Сведения в этом разделе и связанные с ним задачи предназначены для администраторов баз данных табличных моделей, которые будут создавать роли для развертываемых табличных моделей с помощью среды [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)]и управлять этими ролями. Сведения о ролях в табличных моделях см. в разделе [Роли (табличные службы SSAS)](../../analysis-services/tabular-models/roles-ssas-tabular.md). Сведения о создании ролей и управлении ими в проекте табличной модели с использованием диспетчера ролей в среде [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)] см. в разделе [Создание ролей и управление ими (табличные службы SSAS)](../../analysis-services/tabular-models/create-and-manage-roles-ssas-tabular.md).
