# <a name="tabular-model-roles"></a>Роли табличных моделей 
[!INCLUDE[ssas-appliesto-sqlas-aas](../../includes/ssas-appliesto-sqlas-aas.md)]
  Сведения в этом разделе и связанные с ним задачи предназначены для администраторов баз данных табличных моделей, которые будут создавать роли для развертываемых табличных моделей с помощью среды [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)]и управлять этими ролями. Дополнительные сведения о ролях в табличных моделях см. в разделе [ролей](../../analysis-services/tabular-models/roles-ssas-tabular.md). Дополнительные сведения о создании и управление ролями в проекте табличной модели с помощью диспетчера ролей в [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)], в разделе [Создание и управление ролями](../../analysis-services/tabular-models/create-and-manage-roles-ssas-tabular.md).
