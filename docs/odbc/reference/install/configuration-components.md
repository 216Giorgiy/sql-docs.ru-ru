# <a name="configuration-components"></a>Конфигурации компонентов
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC включается в операционной системе Windows. ODBC следует устанавливать только явно на более ранних версиях Windows.  
  
 Источники данных настроены с помощью установщика библиотеки DLL, который в свою очередь вызовы драйвера программой установки библиотек DLL и библиотеках переводчик установки при необходимости. Установщик DLL либо вызвать и непосредственно из панели управления или загружается и вызывается другой программой, известный как *программы администрирования*.
