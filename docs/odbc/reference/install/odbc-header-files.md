# <a name="odbc-header-files"></a>Файлы заголовка ODBC
> [!NOTE]  
>  Распространение файлов заголовка ODBC не требуется. Нет, считаются «Core».  
  
 **Заголовочные файлы**  
  
 Odbcinst.h  
  
 SQL.h  
  
 Sqlext.h  
  
 SqlTypes.h  
  
 Sqlucode.h  
  
 MSDASQL.h  
  
 Msdadc.h  
  
 Sqlspi.h (для разработки драйвера см. в [Справочник интерфейса поставщика службы ODBC (SPI)](../../../odbc/reference/syntax/odbc-service-provider-interface-spi-reference.md) подробнее) доступен начиная с Windows 8 SDK.
