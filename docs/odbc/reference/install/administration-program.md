# <a name="administration-program"></a>Программа администрирования
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC включается в операционной системе Windows. ODBC следует устанавливать только явно на более ранних версиях Windows.  
  
 Программы администрирования, а администратор ODBC, входит в состав Windows SDK и MDAC SDK. Это программы, может свободно распространяться пользователями пакета SDK. Кроме того разработчики могут создавать свои собственные программы администрирования. Как правило разработчики создавать свои собственные программы администрирования, только в том случае, если нужно сохранить полный контроль над настройки источника данных, или если они настроить источники данных непосредственно из приложения, который действует как программа администрирования. Например работы с электронными таблицами, могут разрешать пользователям добавлять и затем использовать источники данных во время выполнения.  
  
 Программа администрирования сначала загружает установщика DLL. Затем он вызывает функции в программе установки библиотеку DLL для выполнения следующих задач:  
  
-   **Добавить, изменить или удалить источники данных в интерактивном режиме.** Программа администрирования может вызвать **SQLManageDataSources**, **SQLCreateDataSource**, или **SQLConfigDataSource**.  
  
     **SQLManageDataSources** отображает диалоговое окно, с помощью которого пользователь может добавлять, изменять, или удалить источники данных и укажите параметры трассировки; эта функция вызывается при вызове установщика DLL непосредственно из панели управления. **SQLCreateDataSource** отображает диалоговое окно, с помощью которого пользователь может добавить только источники данных. **SQLConfigDataSource** передает этот вызов непосредственно к DLL-файлов установки драйвера.  
  
     Во всех случаях вызывает установщик DLL **ConfigDSN** в DLL-файлов для добавления установки драйвера, изменить или удалить источник данных. DLL-файлов установки драйвера может запрашивать у пользователя дополнительных сведений.  
  
-   **Добавить, изменить или удалить источники данных без вмешательства пользователя.** Программа администрирования вызовет **SQLConfigDataSource** в установщик DLL и передает его окно null обработки, имя источника данных, чтобы добавить, изменить или удалить и список значений для раздела реестра. Библиотеки DLL вызывает установщик **ConfigDSN** в DLL-файлов для добавления установки драйвера, изменить или удалить источник данных.  
  
-   **Добавить, изменить или удалить источник данных по умолчанию.** Источник данных по умолчанию является таким же, как любой другой источник данных, за исключением того, его имя по умолчанию. Он будет добавлен, измененных или удаленных так же, как любой другой источник данных.
