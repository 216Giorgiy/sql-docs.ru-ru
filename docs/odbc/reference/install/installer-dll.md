# <a name="installer-dll"></a>Установщик DLL
> [!NOTE]  
>  Начиная с Windows XP и Windows Server 2003, ODBC включается в операционной системе Windows. ODBC следует устанавливать только явно на более ранних версиях Windows.  
  
 Установщик DLL содержит функции для установки и удаления компонентов ODBC, Ведение реестра сведения об этих компонентах и обслуживания реестра сведения об источниках данных. Он записывается корпорацией Майкрософт и может распространяться. Полное описание функций в программе установки библиотеки DLL, в разделе [Справочник по API библиотеки DLL установщика](../../../odbc/reference/syntax/installer-dll-api-reference-function.md).
