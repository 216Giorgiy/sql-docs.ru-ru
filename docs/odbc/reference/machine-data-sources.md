# <a name="machine-data-sources"></a>Источников данных
*Источники данных компьютера* хранятся в системе с именем, определяемые пользователем. Связанный с именем источника данных необходим все сведения, диспетчер драйверов и драйвер для подключения к источнику данных. Для источника данных Xbase это может быть имя драйвера Xbase, полный путь к каталогу, содержащему файлы Xbase и некоторые параметры, которые задают драйвер использовать эти файлы, например в однопользовательском режиме, или только для чтения. Для источника данных Oracle, это может быть имя драйвера Oracle, сервера, содержащего СУБД Oracle SQL * Net строку подключения, которая идентифицирует SQL\*сетевой драйвер для использования и системный идентификатор базы данных на сервере.
