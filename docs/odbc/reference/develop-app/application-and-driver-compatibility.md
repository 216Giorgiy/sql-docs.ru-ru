# <a name="application-and-driver-compatibility"></a>Драйвер совместимости приложений и
Приложения ODBC и драйвер делятся на категории, кроме их версии. Некоторые из этих приложений несовместимы с некоторые драйверы; в других случаях тип приложения или драйвера возможно, влияют на проблемах обратной совместимости между ними.
