# <a name="diagnostic-tools"></a>Средства диагностики
Два средства может помочь в диагностике неполадок в приложениях ODBC. Оба доступны с помощью программы администрирования ODBC реализовано с помощью диспетчера драйверов. *Трассировка* служит для записи в файл журнала последовательности вызовов функций. Visual Studio Analyzer позволяет анализа информацию о взаимодействии компонентов в распределенной среде.
