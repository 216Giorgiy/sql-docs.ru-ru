# <a name="sql-statements"></a>Инструкции SQL
Приложения ODBC выполнять почти все доступа к базе данных при выполнении инструкций SQL. Формат инструкций — жестко или спрогнозированные во время выполнения, с возможностью взаимодействия или данных, связанными с источником и т. д. — зависит от потребностей приложения.
