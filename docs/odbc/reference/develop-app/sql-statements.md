# <a name="sql-statements"></a>Инструкции SQL
Приложения ODBC выполнить почти все доступ к базе данных путем выполнения инструкций SQL. Формат инструкций — жестко или созданные во время выполнения, с возможностью взаимодействия, или данные, определяемые источником и т. д. — зависит от требований приложения.
