# <a name="using-data-buffers"></a>Использование буферов данных
Буферы данных описаны три элемента информации: длина их тип, адрес и байтов. Каждый раз, когда функция необходима одна из этих элементов информации и не знаете его, он имеет аргумент, с которым приложение передает его.
