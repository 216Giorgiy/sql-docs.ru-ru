# <a name="what39s-new-in-odbc-38"></a>Какой &#39; новые возможности ODBC 3.8
Windows 8 включает обновленную версию ODBC 3.8. ODBC 3.8 в Windows 8 включает следующие компоненты:  
  
-   [Организация пулов соединений с учетом драйвера](../../odbc/reference/develop-app/driver-aware-connection-pooling.md)  
  
-   [Асинхронное выполнение (метод уведомления)](../../odbc/reference/develop-app/asynchronous-execution-notification-method.md)  
  
-   [Доступ к данным трассировки (Windows 8)](https://msdn.microsoft.com/library/windows/desktop/hh829624.aspx)  
  
-   Команды PowerShell были добавлены сведения об управлении ODBC источников, драйверы ODBC, счетчик производительности ODBC и данных Трассировка доступа к данным из командной строки.  Дополнительные сведения см. [команд PowerShell компоненты доступа к данных Windows](https://msdn.microsoft.com/library/windows/desktop/jj134064.aspx).  
  
 Windows 7 включает обновленную версию ODBC, ODBC 3.8. ODBC 3.8 включает следующие компоненты:  
  
-   Асинхронно выполняет операции подключения. Дополнительные сведения см. в разделе [асинхронное выполнение (метод опроса)](../../odbc/reference/develop-app/asynchronous-execution-polling-method.md).  
  
-   Потоковых выходных параметров. Дополнительные сведения см. в разделе [получение выходных параметров с помощью метода SQLGetData](../../odbc/reference/develop-app/retrieving-output-parameters-using-sqlgetdata.md).  
  
-   Тип данных ODBC C расширяемости. Дополнительные сведения см. в разделе [типы данных C в ODBC](../../odbc/reference/develop-app/c-data-types-in-odbc.md).  
  
 Драйвер модулей записи необходимо прочитать [обновление 3.5 драйвер 3.8 драйвер](../../odbc/reference/develop-driver/upgrading-a-3-5-driver-to-a-3-8-driver.md).  
  
 Операции асинхронного подключения могут быть использованы в ODBC 3.x и ODBC 2.x приложений с помощью драйвера ODBC 3.8.  
  
 Дополнительные сведения см. в разделе [Матрица совместимости](../../odbc/reference/develop-app/compatibility-matrix.md).
