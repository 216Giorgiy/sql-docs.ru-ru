# <a name="types-of-data-sources"></a>Типы источников данных
Существует два типа источников данных: источники данных и файловых источников данных компьютера. Несмотря на то, что оба содержат схожие сведения об источнике данных, они отличаются, как эти данные сохраняются. Из-за этих различий они используются в некоторой степени варианта.  
  
 Этот раздел содержит следующие подразделы.  
  
-   [Машинные источники данных](../../odbc/reference/machine-data-sources.md)  
  
-   [Файловые источники данных](../../odbc/reference/file-data-sources.md)
