# <a name="interval-literal-syntax"></a>Синтаксис интервале
Интервал литералов в ODBC используется следующий синтаксис.  
  
 *Интервал literal:: = ИНТЕРВАЛ* [+*&#124;*-] *интервал интервал строка квалификатор*  
  
 *Интервал строка* :: = *Квота* { *год месяц literal* &#124; *день литерал времени* } *квоты*  
  
 *год месяц literal* :: = *значение года* &#124; [*значение года* -] *значение месяцев*  
  
 *литерал для времени дня* :: = *день интервала* &#124; *интервал времени*  
  
 *день интервала* :: = *значение дней* [*значение часов* [:*значение минут*[:*значение секунд*]]]  
  
 *интервал времени* :: = *значение часов* [:*значение минут* [:*значение секунд* ]]  
  
 &#124;*значение минут* [:*значение секунд* ]  
  
 &#124;*значение секунд*  
  
 *значение года* :: = *значение даты и времени*  
  
 *значение месяцев* :: = *значение даты и времени*  
  
 *значение дней* :: = *значение даты и времени*  
  
 *значение часов* :: = *значение даты и времени*  
  
 *значение минут* :: = *значение даты и времени*  
  
 *значение секунд* :: = *секунд целое* [. [ *долей секунд*]]  
  
 *секунд целое* :: = *unsigned integer*  
  
 *доли секунд* :: = *unsigned integer*  
  
 *значение типа DateTime* :: = *unsigned integer*  
  
 *Интервал квалификатор* :: = *поле начала* TO *конец поля* &#124; *поля с одним datetime*  
  
 *поле начала* :: = *не второй поля даты и времени* [(*интервал начальные поля точности* )]  
  
 *конец поля* :: = *не второй поля даты и времени* &#124; второй [(*интервал--секунд — точность в долях*)]  
  
 *поля с одним datetime* :: = *не второй поля даты и времени* [(*интервал начальные поля точности*)] &#124; второй [(*интервал начальные поля точности*  [, (*интервал--секунд — точность в долях*)]  
  
 *поля даты и времени* :: = *не второй поля даты и времени* &#124; второй  
  
 *не второй поля даты и времени* :: = год &#124; месяц &#124; день &#124; час &#124; МИНУТУ  
  
 *интервал--секунд — точность в долях* :: = *unsigned integer*  
  
 *Интервал начальные поля точности* :: = *unsigned integer*  
  
 *Квота* :: = "  
  
 *unsigned integer* :: = *цифрой...*
