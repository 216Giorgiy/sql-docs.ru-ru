# <a name="transfer-octet-length"></a>Длина в октетах передачи
Длина столбца в октетах передачи — максимальное число байтов, возвращенных в приложение при передаче данных в тип данных C по умолчанию. Для символьных данных длина октет передачи не учитывает пространство для знака завершения null. Длина столбца в октетах передачи может отличаться от количество байтов, необходимое для хранения данных в источнике данных.  
  
 В следующей таблице показан длина передачи октетов, определенная для каждого типа данных ODBC SQL.  
  
|Идентификатор типа SQL|Длина|  
|-------------------------|------------|  
|Все символьные типы []|Определенный или максимальное значение (для типа переменной) длина столбца в байтах. Это значение совпадает со значением поля дескриптора SQL_DESC_OCTET_LENGTH.|  
|SQL_DECIMAL<br />SQL_NUMERIC|Число байтов, необходимое для хранения символов представления этих данных, если набор символов ANSI и дважды этот номер, если набор символов ЮНИКОДА. Это максимальное число цифр плюс два, так как данные возвращаются в виде строки символов и символов требуется для цифры, знак и десятичной запятой. Например передача длина столбца, определенного как NUMERIC(10,3) — 12.|  
|SQL_TINYINT|1|  
|SQL_SMALLINT|2|  
|SQL_INTEGER|4|  
|SQL_BIGINT|Число байтов, необходимое для хранения символов представления этих данных, если набор символов ANSI и дважды этот номер, если набор символов ЮНИКОДА, так как этот тип данных возвращается в виде символьной строки по умолчанию. Символьное представление состоит из 20 символов: 19 цифр и знак, если подпись или 20 символов, если без знака. Таким образом длина — 20.|  
|SQL_REAL|4|  
|SQL_FLOAT|8|  
|SQL_DOUBLE|8|  
|SQL_BIT|1|  
|Все двоичные типы []|Число байтов, необходимое для хранения определенных (для основных типов) или (для типов переменных) количество символов.|  
|SQL_TYPE_DATE<br />SQL_TYPE_TIME|6 (размер структуры SQL_DATE_STRUCT или SQL_TIME_STRUCT).|  
|SQL_TYPE_TIMESTAMP|16 (размер структуры SQL_TIMESTAMP_STRUCT).|  
|Все типы данных интервала|34 (размер структуры интервал).|  
|SQL_GUID|16 (размер структуры GUID).|  
  
 [] Если драйвер не может определить длину столбца или параметра для типов переменных, возвращается значение SQL_NO_TOTAL.
