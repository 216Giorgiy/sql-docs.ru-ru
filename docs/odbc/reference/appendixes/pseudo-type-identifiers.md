# <a name="pseudo-type-identifiers"></a>Идентификаторы псевдо типа
Для удобства программирования приложений ODBC определяет количество идентификаторов псевдо типа. Эти идентификаторы не соответствуют действительности фактических типов данных, но вместо этого разрешения в зависимости от ситуации, в существующих типов данных.
