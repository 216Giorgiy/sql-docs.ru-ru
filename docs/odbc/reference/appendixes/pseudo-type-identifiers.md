# <a name="pseudo-type-identifiers"></a>Идентификаторы псевдотипа
Для удобства программирования приложений ODBC определяет ряд идентификаторы псевдотипа. Эти идентификаторы не соответствуют фактически типами фактических данных, но вместо этого решения в зависимости от ситуации, в существующие типы данных.  
  
 Этот раздел содержит следующие подразделы.

-   [SQL_ARD_TYPE](../../../odbc/reference/appendixes/sql-ard-type.md)  
  
-   [SQL_C_TCHAR](../../../odbc/reference/appendixes/sql-c-tchar.md)
