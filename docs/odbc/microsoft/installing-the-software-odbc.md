# <a name="installing-the-software-odbc"></a>Установка программного обеспечения (ODBC)
> [!IMPORTANT]  
>  Этот компонент будет удален в будущих версиях Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемого корпорацией Oracle.  
  
 Драйвер ODBC для Oracle — один из компонентов доступа к данным. Сопровождающий других компонентов ODBC, такие как источники данных ODBC она должна быть установлена. Драйвер также можно найти в разделе «Драйверы и другие загружаемые файлы» на сайте www.Microsoft.com интерактивные службы поддержки продуктов Microsoft.  
  
 Сетевое программное обеспечение должны устанавливаться в соответствии с его собственной документации. Драйвер ODBC для Oracle особенностей нет специальной установки при условии, что поддерживается сетевое программное обеспечение.  
  
 Программное обеспечение Oracle должно устанавливаться в соответствии с его собственной документации. Драйвер ODBC для Oracle обычно особенностей нет специальной установки при условии, что драйвер поддерживает версию. Тем не менее чтобы сохранить совместимые продукты, установите драйвер ODBC для Oracle последней, чтобы убедиться, что установлена последняя версия драйвера. Oracle поддерживает общих FTP-сайта, где он инициирует передачу, помимо прочего, обновления для серверных продуктов Oracle и клиентский компонент, который поставляется вместе с серверными продуктами. Эти обновления необходимы для правильной работы нескольких продуктов и технологий корпорации Майкрософт. Дополнительные сведения об этом сайте см. в разделе [обновления программного обеспечения Oracle](../../odbc/microsoft/oracle-software-patches.md).  
  
> [!CAUTION]  
>  Установка программного обеспечения Oracle через MDAC или Windows DAC может перезаписать текущие версии компонентов MDAC. В случае возникновения проблем с помощью компонентов ODBC, переустановите компоненты MDAC.
