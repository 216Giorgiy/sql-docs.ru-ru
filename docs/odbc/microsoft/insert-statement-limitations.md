# <a name="insert-statement-limitations"></a>Ограничения инструкции INSERT
Вставленные данные усекаются справа без предупреждения, если он превышает допустимую длину, чтобы поместиться в столбец.  
  
 Попытка вставить значение, которое находится за пределами диапазона типа данных столбца вызовет значение NULL должны быть вставлены в столбец.  
  
 При использовании dBASE, Microsoft Excel, Paradox или Textdriver вставки строки нулевой длины в столбец фактически вставляет значение NULL вместо него.  
  
 При использовании драйвера Microsoft Excel, если пустая строка вставляется в столбец, пустая строка преобразуется в значение NULL; Искомая инструкции SELECT, который выполняется с пустой строкой в предложении WHERE не будет выполнено в этом столбце.  
  
 Невозможно обновить таблицу, драйвером Paradox в двух случаях:  
  
-   Если уникальный индекс не определен в таблице. Это не относится к пустую таблицу, могут обновляться по одной строке, даже если не определен уникальный индекс для таблицы. Если одна строка вставляется в пустую таблицу, не имеет уникального индекса, приложение не может создать уникальный индекс или вставлять дополнительные данные после вставки одной строки.  
  
-   Если компонент Database Engine Borland не используется, только чтение и Добавление таблицы Paradox разрешены инструкции.  
  
 При использовании драйвера текстового значения NULL представленного строковым дополняются пустым в файлах фиксированной длины, но представляются без пробелов в файлы с разделителями. Например в следующую строку, в котором содержатся три поля, второе поле имеет значение NULL:  
  
```  
"Smith:,, 123  
```  
  
 При использовании драйвера текстового значения всех столбцов можно заполняются начальных пробелов. Длина любой из строк должно быть меньше или равна 65,543 байт.
