# <a name="odbc-driver-for-oracle-conformance-levels"></a>Драйвер ODBC для Oracle уровни согласованности
> [!IMPORTANT]  
>  Этот компонент будет удален в будущих версиях Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемого корпорацией Oracle.  
  
 ODBC определяет два типа соответствия стандартам для драйверов: стандартные интерфейса API ODBC и с соответствием грамматику SQL standard. Функции, которые поддерживает драйвер ссылается интерфейса API ODBC. Соответствие SQL ссылается грамматику SQL, который поддерживает драйвер. Каждая стандартная соответствия состоит из уровней.  
  
 В этом разделе рассматривается соответствие драйвер ODBC для Oracle.
