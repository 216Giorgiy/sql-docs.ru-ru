# <a name="alter-table-statement"></a>Инструкции ALTER TABLE
При использовании Microsoft Access, dBASE или Paradoxdriver поддерживается в инструкции ALTER TABLE add и drop определение предложений ограничений таблицы. (Для драйверов Microsoft Excel или текст не поддерживаются инструкции ALTER TABLE.)  
  
> [!NOTE]  
>  При использовании драйвера Paradox без реализации СУБД Borland инструкции ALTER TABLE не поддерживаются; только чтения и добавления инструкции являются допустимыми.
