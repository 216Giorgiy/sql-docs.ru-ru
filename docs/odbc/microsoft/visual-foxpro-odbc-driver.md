# <a name="visual-foxpro-odbc-driver"></a>Драйвер ODBC для Visual FoxPro
Microsoft Visual FoxPro — это мощная среда объектно ориентированного для разработки баз данных построения и приложения. Драйвер Microsoft Visual FoxPro ODBC позволяет приложениям откройте запрос и обновления данных в более ранних версиях FoxPro через интерфейс откройте Database Connectivity (ODBC) и Visual FoxPro.  
  
 Например драйвер ODBC Microsoft Visual FoxPro позволяет выполнять следующее:  
  
-   Microsoft Query используется для запроса и обновления данных Visual FoxPro из книги Microsoft Excel.  
  
-   Создайте слияние почты письма, с помощью Visual FoxPro данных с помощью Microsoft Word.  
  
-   Запрос и обновление Visual FoxPro представлений и таблиц из Microsoft Access.  
  
-   Используйте Visual FoxPro как в хранилище данных для приложений Microsoft Visual Basic, Microsoft Visual C++ и C.
