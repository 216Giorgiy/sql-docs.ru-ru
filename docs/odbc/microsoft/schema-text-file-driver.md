# <a name="schema-text-file-driver"></a>Схема (драйвера текстового файла)
Схема включает в себя информацию о каждой таблице (текстовый файл) в источнике данных, включая формат таблицы, количество строк для сканирования для определения типов столбцов, будет ли первая строка таблицы содержит имена столбцов ли исходный файл записывается с помощью поставщиков вычислительной Техники или Кодовая страница ANSI и каждый столбец имя, тип данных и ширину.
