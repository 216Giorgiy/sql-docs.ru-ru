# <a name="schema-text-file-driver"></a>Схема (драйвер для текстовых файлов)
Схема включает в себя сведения о каждой таблицы (текстовый файл) в источнике данных, включая формат таблицы, количество строк для проверки для определения типов столбцов, является ли первая строка таблицы содержит имена столбцов, написанные исходного файла с помощью поставщиков вычислительной Техники или Кодовая страница ANSI и каждый столбец имя, тип данных и ширины.
