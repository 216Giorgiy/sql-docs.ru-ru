# <a name="odbc-driver-for-oracle-user39s-guide"></a>Драйвер ODBC для Oracle пользователя &#39; руководство
> [!IMPORTANT]  
>  Этот компонент будет удален в будущих версиях Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемого корпорацией Oracle.  
  
 Драйвер ODBC для Oracle в руководстве пользователя предназначен в дополнение к Справка программы установки, можно получить с помощью [администратор источников данных ODBC](../../odbc/admin/odbc-data-source-administrator.md).
