# <a name="configuring-the-odbc-driver-for-oracle"></a>Настройка драйвера ODBC для Oracle
> [!IMPORTANT]  
>  Этот компонент будет удален в будущих версиях Windows. Избегайте использования этого компонента в новых разработках и запланируйте изменение существующих приложений, в которых он применяется. Вместо этого используйте драйвер ODBC, предоставляемого корпорацией Oracle.  
  
 Можно выбрать производительности драйвера ODBC для Oracle, определяя среды данных и правильность настройки параметров подключения к источнику данных через [администратор источников данных ODBC](../../odbc/admin/odbc-data-source-administrator.md) диалогового окна поле или через подключения Параметры строки. Диалоговое окно предоставляет следующие элементы управления для подключения к источнику данных с помощью диалогового окна или с помощью объединение строк:  
  
-   **Вкладка DSN пользователя** перечислены имена источников данных, которые являются локальными для компьютера.  
  
-   **Вкладка DSN системы** позволяет добавить или удалить системный источник данных. Системные источники данных может осуществляться всех пользователей на локальном компьютере.  
  
-   **Вкладка "файл источника данных"** дает возможность добавления или удаления источника данных с локального компьютера. Файловые источники данных могут совместно использоваться все пользователи, имеющие один и тот же установлен драйвер.  
  
-   **Вкладка драйверов** список установленных драйверов ODBC.  
  
-   **Вкладка "Трассировка"** позволяет задать способ отслеживания вызовов функций ODBC в диспетчер драйверов ODBC. Можно настроить трассировку отдельно для каждого установленного приложения ODBC.  
  
-   **Вкладка Connection Pooling** позволяет выбрать параметры соединения для каждого установленного драйвера.  
  
-   **О вкладке** перечислены установленные файлы компонентов ODBC.  
  
 После добавления источника данных, можно использовать **администратор источников данных ODBC** диалоговое окно «», чтобы настроить доступ к источнику данных. Выберите источник данных и затем выберите одну из вкладок, чтобы изменить или просмотреть сведения.
