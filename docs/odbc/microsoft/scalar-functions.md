# <a name="scalar-functions"></a>Скалярные функции
Драйверы Microsoft ODBC системной базы данных поддерживают следующие скалярные функции:  
  
||||  
|-|-|-|  
|CONCAT|LCASE|RIGHT|  
|CONVERT|LEFT|RTRIM|  
|CURDATE|LENGTH|SUBSTRING|  
|CURTIME|LOCATE|UCASE|  
|DATABASE|LTRIM|Пользователь|  
|DAYOFMONTH|MOD|YEAR|  
|DAYOFWEEK|MONTH||  
  
 Сведения об аргументах и возвращаемых значений скалярных функций см. в разделе приложении F из *справочнике программиста ODBC.*
