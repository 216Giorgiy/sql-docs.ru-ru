# <a name="sqlprocedures-desktop-database-drivers"></a>SQLProcedures (драйверы для баз данных на настольном компьютере)
**SQLProcedures** , возвращает только строки для этих процедур, которые имеют по крайней мере один аргумент. Процедуры, которые не имеют аргументов, рассматриваются как представления.  
  
|Столбец|Комментарии|  
|------------|--------------|  
|PROCEDURE_QUALIFIER|Путь к файлу базы данных.|  
|PROCEDURE_OWNER|NULL|  
|PROCEDURE_NAME|Имя процедуры неразделенный|  
|PROCEDURE_TYPE|SQL_PT_PROCEDURE|
