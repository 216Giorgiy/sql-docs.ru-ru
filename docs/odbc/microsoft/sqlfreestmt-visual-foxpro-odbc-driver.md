# <a name="sqlfreestmt-visual-foxpro-odbc-driver"></a>SQLFreeStmt (драйвер ODBC для Visual FoxPro)
> [!NOTE]  
>  Этот раздел содержит сведения Visual FoxPro ODBC-драйвером. Общие сведения об этой функции см. в соответствующем разделе [Справочник по API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Поддержка: полный  
  
 Соответствия API ODBC: Уровень Core  
  
 Останавливает обработку, связанные с конкретным *hstmt*, закрывает все открытые курсоры, связанные с *hstmt*, отменяет ожидающие результаты и при необходимости освобождает все ресурсы, связанные с инструкцией дескриптор.  
  
 Дополнительные сведения см. в разделе [SQLFreeStmt](../../odbc/reference/syntax/sqlfreestmt-function.md) в *Справочник по программированию ODBC*.
