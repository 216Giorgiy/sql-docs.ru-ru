# <a name="sqlfreestmt-visual-foxpro-odbc-driver"></a>SQLFreeStmt (драйвер ODBC для Visual FoxPro)
> [!NOTE]  
>  Этот раздел содержит сведения по Visual FoxPro ODBC драйвера. Общие сведения об этой функции см. в соответствующем разделе [Справочник по API ODBC](../../odbc/reference/syntax/odbc-api-reference.md).  
  
 Поддержка: полный  
  
 Соответствия ODBC API: Уровень Core  
  
 Останавливает обработку, связанные с конкретным *hstmt*, закрывает все открытые курсоры, связанные с *hstmt*отменяет ожидающие результаты и при необходимости освобождает все ресурсы, связанные с инструкцией дескриптор.  
  
 Дополнительные сведения см. в разделе [SQLFreeStmt](../../odbc/reference/syntax/sqlfreestmt-function.md) в *справочнике программиста ODBC*.
