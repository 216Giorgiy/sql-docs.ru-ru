# <a name="viewing-drivers"></a>Просмотр драйверов
Для настройки источников данных, необходимо установить по крайней мере один драйвер на компьютере. **Драйверы** вкладке **администратор источников данных ODBC** диалоговом окне приводится список всех драйверов, установленных на компьютере, включая имя, версия, компания, имя файла и Дата создания файла каждого драйвера. Необходимо использовать программу установки драйвера для добавления или удаления драйверов из системы. Дополнительные сведения об изменении драйверов см. в разделе [Управление источниками данных](../../odbc/admin/managing-data-sources.md).
