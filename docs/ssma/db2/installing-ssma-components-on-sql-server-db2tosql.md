# <a name="installing-ssma-components-on-sql-server-db2tosql"></a>Установка компонентов SSMA на SQL Server (DB2ToSQL)
В этой версии SSMA нет необходимости в качестве отдельной установки на [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] пакета расширения SSMA поддерживает перенос данных и поставщики DB2, чтобы включить возможность подключения сервера к серверу.  
  
## <a name="ssma-for-db2-extension-pack"></a>SSMA для пакета расширения DB2  
Пакет расширения SSMA добавляет схему базы данных в указанном экземпляре [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)]. **Sysdb** схема содержит таблицы и хранимые процедуры, необходимые для переноса данных и определяемые пользователем функции, позволяющие воссоздавать DB2 системные функции.
