# <a name="installing-ssma-components-on-sql-server-db2tosql"></a>Установка компонентов SSMA в SQL Server (DB2ToSQL)
В этой версии SSMA нет необходимости для отдельной установки на [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] пакета расширения SSMA, которая поддерживает перенос данных и поставщики DB2, чтобы обеспечить подключение между серверами.  
  
## <a name="ssma-for-db2-extension-pack"></a>SSMA для DB2 Extension Pack  
Пакет расширений SSMA добавляет схему базы данных в указанном экземпляре [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. **Sysdb** схема содержит таблицы и хранимые процедуры, которые необходимы для переноса данных и определяемые пользователем функции, которые имитируют DB2 системные функции.
