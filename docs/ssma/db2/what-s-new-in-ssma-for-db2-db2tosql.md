# <a name="what39s-new-in-ssma--for-db2-db2tosql"></a>Какой &#39; новые возможности SSMA для DB2 (DB2ToSQL)
В этом разделе перечислены SSMA для DB2 изменения в каждом выпуске.  

## <a name="may-2016"></a>Май 2016 г.  
Май 2016 г. выпуск SSMA для DB2 содержит следующие изменения:  

1.  Добавлена поддержка SQL Server 2016
2.  Добавлено преобразование таблиц в памяти и обычным DB2 компонентов SQL Server в памяти и hekaton
3.  Добавлены преобразование DB2 элементы управления доступом к объектам политика SQL Server (Безопасность на уровне строк для DB2)
4.  Добавленный преобразования таблицы с системным управлением версиями DB2 темпоральных таблиц SQL Server
5.  Улучшенные средства синтаксического анализа DB2 и сопоставителя
6.  Удалить проверку установщика для .net 2.0
7.  Удалить ненужные *.dll Db2 установщика
8.  Исправлена «сохранить проект» и команды «Открыть проект» для консоли SSMA
9.  Команда фиксированный «securepassword» для консоли SSMA
10. Исправлена подсчет объектов для начальной загрузки
11. Исправлена ошибка в глобальных параметрах
  
## <a name="march-2016"></a>Март 2016 г.  
Март 2016 г. Предварительный выпуск SSMA для DB2 содержит следующие изменения:  
  
1.  Поддерживает миграцию на SQL Server 2016  
  
## <a name="january-2016"></a>Январь 2016 г.  
Январь 2016 г. обслуживание выпуск SSMA для DB2 содержит следующие изменения:  
  
1.  Добавлена поддержка нескольких стандартных функций  
  
2.  Ошибки синтаксического анализа основных DB2  
  
3.  Фиксированный DB2 v9 zOS поддержки (RFC 5690920)  
  
4.  Фиксированный DB2 неразрешенных идентификатор ошибок во время преобразования  
  
5.  Пункт меню журнала добавлено представление SSMA (RFC 5706203)  
  
6.  Добавлены телеметрии  
  
## <a name="november-2014"></a>Ноябрь 2014 г.  
Ноябрь 2014 г. выпуск SSMA для DB2 является первым своего рода.
