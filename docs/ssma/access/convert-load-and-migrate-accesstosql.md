# <a name="convert-load-and-migrate-accesstosql"></a>Преобразовать, загрузить и перенести (AccessToSQL)
**Преобразовать, загрузить и перенести** диалоговое окно сообщает состояние миграции при использовании кнопки Convert, загрузки и миграции или мастер миграции.  
  
После завершения миграции можно просматривать информационные, предупреждения и сообщения об ошибках, щелкнув ссылку в каждом разделе. По ссылке показывает, сколько ошибки, предупреждения и информационные сообщения доступны, например «0 ошибок предупреждений 4.»  
  
**Переноса данных для выбранных объектов** также содержит раздел страницы **отчетов** кнопки. Можно нажать эту кнопку, чтобы просмотреть отчет о миграции данных.  
  
## <a name="options"></a>Параметры  
**Отчет**  
Показывает преобразование или миграцию связанного отчета.
