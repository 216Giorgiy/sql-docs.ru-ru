# <a name="convert-load-and-migrate-accesstosql"></a>Преобразование, загрузка и перенос (AccessToSQL)
**Преобразование, загрузка и перенос** диалоговое окно сообщает о состоянии миграции при использовании кнопки Convert, загрузки и миграции или мастер миграции.  
  
После завершения миграции можно просмотреть информационное, предупреждение и сообщения об ошибках, щелкнув ссылку в каждом разделе. Эта ссылка указывает, сколько ошибки, предупреждения и информационные сообщения доступны, такие как «0 ошибок, предупреждений 4.»  
  
**Переноса данных для выбранных объектов** также содержит раздел страницы **отчетов** кнопки. Можно нажать эту кнопку, чтобы просмотреть отчет о миграции данных.  
  
## <a name="options"></a>Параметры  
**Отчет**  
Отображает связанный отчет преобразования или миграции.
