# <a name="global-settings-editor-accesstosql"></a>Глобальные параметры (редактор) (AccessToSQL)
Страница редактора **глобальные параметры** диалоговое окно, чтобы указать параметры для окон редактора кода.  
  
Чтобы открыть параметры редактора, в **средства** меню, выберите **глобальные параметры**, нажмите кнопку **графического пользовательского интерфейса** в нижней части левой панели, а затем выберите **редактор**.  
  
## <a name="options"></a>Параметры  
**Показать номера строк в редакторах**  
Отображение номеров строк в окнах редактора кода, выберите **True**.  
  
Если выбран в окне редактора кода, при изменении этого параметра, может потребоваться установить заново для текущего объекта, чтобы обновить список.
