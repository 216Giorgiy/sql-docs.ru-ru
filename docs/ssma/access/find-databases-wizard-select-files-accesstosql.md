# <a name="find-databases-wizard-select-files-accesstosql"></a>Найти мастера баз данных (выберите файлы) (AccessToSQL)
На странице Выбор файлов найти мастера баз данных перечислены файлы, которые были обнаружены во время поиска. Чтобы выбрать файл для добавления SSMA, выберите строку, содержащую имя файла. После завершения выбора файлов нажмите кнопку **Далее** для отображения страницы выбора проверки.  
  
Открытие сети сканирования для мастера файлов на **файл** меню, нажмите кнопку **найти баз данных**.  
  
Чтобы выбрать файл для добавления SSMA, установите флажок рядом с именем файла. После завершения выбора файлов нажмите кнопку **Далее** для просмотра страница завершения.  
  
## <a name="options"></a>Параметры  
**Выбрать все**  
Установите этот флажок, чтобы выбрать все базы данных в списке.  
  
**Снять выделение**  
Установите этот флажок, чтобы отменить выделение всех баз данных в списке.  
  
**Имя файла**  
Отображает имя файла базы данных Access, чтобы добавить SSMA.  
  
**Владелец**  
Отображает владельца файла.  
  
**Путь к файлу**  
Отображает путь к файлу базы данных Access.  
  
**Размер**  
Размер базы данных, в байтах.  
  
**Создан**  
Дата и время создания базы данных.  
  
**Изменен**  
Дата и время последнего обновления базы данных.
