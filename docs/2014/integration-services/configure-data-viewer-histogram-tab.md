# <a name="configure-data-viewer-histogram-tab"></a>Настройка средства просмотра данных (вкладка «Гистограмма»)
  Вкладка **Гистограмма** диалогового окна **Настройка средства просмотра данных** позволяет выбрать столбец для отображения. В гистограмме значения типа данных из одного числового столбца отображаются в виде распределения значений. Диаграмма отображает содержимое только одного буфера.  
  
## <a name="options"></a>Параметры  
 **Визуализированный столбец**  
 Выберите столбец, который будет представлен в виде гистограммы.
