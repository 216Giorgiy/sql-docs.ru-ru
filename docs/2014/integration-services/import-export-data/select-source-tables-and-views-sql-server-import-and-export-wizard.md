# <a name="select-source-tables-and-views-sql-server-import-and-export-wizard"></a>Выбор исходных таблиц и представлений (мастер импорта и экспорта SQL Server)
  Используйте **Выбор исходных таблиц и представлений** страницу, чтобы указать таблицы и представления должны быть скопированы из источника данных в место назначения.  
  
> [!NOTE]  
>  Если выбирается параметр «Копирование таблицы», необязательно копировать все столбцы таблицы. Выбрав целевой таблицы, нажмите кнопку Изменить сопоставления для отображения **сопоставления столбцов** диалоговое окно. Выберите  **\<пропустить >** в **назначения** столбец **сопоставления столбцов** диалоговое окно для столбцов, которые нужно пропустить.  
  
 Дополнительные сведения о работе этого мастера см. в разделе [SQL Server Импорт и экспорт](import-and-export-data-with-the-sql-server-import-and-export-wizard.md). Дополнительные сведения о параметрах запуска этого мастера и о разрешениях, необходимых для успешного запуска мастера см. в разделе [запустить мастер экспорта и импорта SQL Server](start-the-sql-server-import-and-export-wizard.md).  
  
 Назначение мастера импорта и экспорта SQL Server заключается в копировании данных из исходного расположения в целевое. Этот мастер может также создать целевую базу данных и целевые таблицы. Однако если нужно скопировать несколько баз данных, таблиц или других объектов базы данных, следует использовать мастер копирования баз данных. Дополнительные сведения см. в статье [Use the Copy Database Wizard](../../relational-databases/databases/use-the-copy-database-wizard.md).  
  
## <a name="options"></a>Параметры  
  
### <a name="tables-and-views-list"></a>Список «Таблицы и представления»  
 **Source**  
 С помощью флажков выберите из списка таблицы и представления для копирования в место назначения. Если выбрана исходная таблица или представление и никакие другие действия не выполняются, схема и данные из источника копируются без изменения.  
  
 **Назначение**  
 Выберите целевую таблицу из списка для каждой исходной таблицы.  
  
> [!NOTE]  
>  Если вы приостановите выполнение на этом этапе мастера, чтобы создать целевую таблицу в [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] с помощью [!INCLUDE[ssManStudio](../../includes/ssmanstudio-md.md)] или другого средства, новая таблица не сразу же отображается в списке доступных целевых таблиц. Чтобы обновить список целевых таблиц, два шага назад к **Выбор назначения** , повторно выберите целевой базы данных, а затем выполните шаг вперед на попытку **Выбор исходных таблиц и представлений**.  
  
### <a name="other-options"></a>Другие параметры  
 **Изменение сопоставлений**  
 Используйте **сопоставления столбцов** для указания целевых столбцов для получения данных из источника. Можно скопировать только подмножество столбцов, щелкнув \<пропустить > в **назначения** столбец **сопоставления столбцов** диалоговое окно для столбцов, которые нужно пропустить.  
  
 **Предварительный просмотр**  
 Просмотрите исходные данные в **данные для предварительного просмотра** диалоговое окно «», чтобы проверить его перед выполнением импорта или экспорта. **Данные для предварительного просмотра** диалоговое окно отображает до 200 строк данных.  
  
 После просмотра данных может понадобиться изменить выбранные параметры для источника данных и назначения. Для этого на странице **Выбор исходных таблиц и представлений** нажмите кнопку **Назад** , чтобы вернуться к предыдущим страницам, где можно изменить выбранные параметры.
