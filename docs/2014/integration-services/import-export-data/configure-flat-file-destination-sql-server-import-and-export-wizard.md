# <a name="configure-flat-file-destination-sql-server-import-and-export-wizard"></a>Настройка назначения «Неструктурированный файл» (мастер импорта и экспорта SQL Server)
  Используйте **настроить назначение «неструктурированный файл»** задавать параметры форматирования для назначения неструктурированного файла, а также для предварительного просмотра результатов перед продолжением.  
  
 Дополнительные сведения о работе этого мастера см. в разделе [SQL Server Импорт и экспорт](import-and-export-data-with-the-sql-server-import-and-export-wizard.md). Дополнительные сведения о режимах запуска мастера, а также разрешения, необходимые для успешного запуска мастера см. в разделе [запустить мастер экспорта и импорта SQL Server](start-the-sql-server-import-and-export-wizard.md).  
  
 Назначение мастера импорта и экспорта SQL Server заключается в копировании данных из исходного расположения в целевое. Этот мастер может также создать целевую базу данных и целевые таблицы. Однако если нужно скопировать несколько баз данных, таблиц или других объектов базы данных, следует использовать мастер копирования баз данных. Дополнительные сведения см. в статье [Use the Copy Database Wizard](../../relational-databases/databases/use-the-copy-database-wizard.md).  
  
## <a name="options"></a>Параметры  
 **Неструктурированный файл источника**  
 Имя целевого файла.  
  
 **Разделитель строк**  
 Значение выбирается из списка разделителей для строк.  
  
|Значение|Описание|  
|-----------|-----------------|  
|**{CR}{LF}**|Строка отделена с помощью комбинации символов возврата каретки / перевода строки.|  
|**{CR}**|Строка отделена с помощью символа возврата каретки.|  
|**{LF}**|Строка отделена с помощью символа перевода строки.|  
|**Точка с запятой {;}**|Строка отделена с помощью символа точки с запятой.|  
|**Двоеточие {:}**|Строка отделена с помощью символа двоеточия.|  
|**Запятая {,}**|Строка отделена с помощью символа запятой.|  
|**Табуляция {t}**|Строка отделена с помощью символа табуляции.|  
|**Вертикальная черта {&#124;}**|Строка отделена с помощью символа вертикальной черты.|  
  
 **Разделитель столбцов**  
 Значение выбирается из списка разделителей для столбцов.  
  
|Значение|Описание|  
|-----------|-----------------|  
|**{CR}{LF}**|Столбцы разделены с помощью комбинации символов возврата каретки и перевода строки.|  
|**{CR}**|Столбцы разделены с помощью символа возврата каретки.|  
|**{LF}**|Столбцы разделены с помощью символа перевода строки.|  
|**Точка с запятой {;}**|Столбцы разделены с помощью символа точки с запятой.|  
|**Двоеточие {:}**|Столбцы разделены с помощью символа двоеточия.|  
|**Запятая {,}**|Столбцы разделены с помощью символа запятой.|  
|**Табуляция {t}**|Столбцы разделены с помощью символа табуляции.|  
|**Вертикальная черта {&#124;}**|Столбцы разделены с помощью символа вертикальной черты.|  
  
 **Предварительный просмотр**  
 Просмотр в **данные для предварительного просмотра** диалоговое окно параметров результаты выбранного форматирования для назначения неструктурированного файла.  
  
 **Изменить преобразования**  
 Удаление строк, добавление строк и настройка столбцов в файле назначения с помощью **сопоставления столбцов** диалоговое окно.
