# <a name="deprecated-integration-services-features-in-sql-server-2014"></a>Устаревшие функции служб SQL Server 2014 Integration Services
  В этом разделе описываются устаревшие [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] функции, которые по-прежнему доступны в текущем выпуске [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)] [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)]. Эти функции будут удалены в следующем выпуске [!INCLUDE[ssNoVersion](../includes/ssnoversion-md.md)]. Не следует использовать устаревшие функции в новых приложениях.  
  
 В [!INCLUDE[ssISnoversion](../includes/ssisnoversion-md.md)] отсутствуют устаревшие функции служб [!INCLUDE[ssCurrent](../includes/sscurrent-md.md)].
