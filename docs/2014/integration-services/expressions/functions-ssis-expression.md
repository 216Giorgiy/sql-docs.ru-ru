# <a name="functions-ssis-expression"></a>Функции (выражение служб SSIS)
  Язык выражений включает набор функций, которые можно использовать в выражениях. Выражение может использовать только одну функцию, но обычно в выражении используется комбинация операторов и нескольких функций.  
  
 Данные функции можно разделить на следующие группы:  
  
-   Математические функции, выполняющие вычисления на основании числовых значений, переданных как параметры, и возвращающие числовые значения.  
  
-   Строковые функции, выполняющие операции над строками или входными параметрами в шестнадцетиричном виде и возвращающие строку или число.  
  
-   Функции для работы с датой и временем, выполняющие операции над значениями даты и времени и возвращающие строку, число или значение даты и времени.  
  
-   Системные функции, возвращающие сведения о выражении.  
  
 Язык выражений содержит следующие математические функции.  
  
|Компонент|Описание|  
|--------------|-----------------|  
|[ABS &#40;выражение служб SSIS&#41;](abs-ssis-expression.md)|Возвращает абсолютное положительное значение числового выражения.|  
|[EXP &#40;выражение служб SSIS&#41;](exp-ssis-expression.md)|Возвращает число «е» в степени, определяемой данным выражением.|  
|[Верхний ПРЕДЕЛ &#40;выражение служб SSIS&#41;](ceiling-ssis-expression.md)|Возвращает наименьшее целое число, большее или равное данному числовому выражению.|  
|[Функция FLOOR &#40;выражение служб SSIS&#41;](floor-ssis-expression.md)|Возвращает наибольшее целое число, меньшее или равное числовому выражению.|  
|[LN &#40;выражение служб SSIS&#41;](ln-ssis-expression.md)|Возвращает натуральный логарифм числового выражения.|  
|[ЖУРНАЛ &#40;выражение служб SSIS&#41;](log-ssis-expression.md)|Возвращает десятичный логарифм числового выражения.|  
|[POWER &#40;выражение служб SSIS&#41;](power-ssis-expression.md)|Возвращает результат возведения числового выражения в степень.|  
|[ROUND &#40;выражение служб SSIS&#41;](round-ssis-expression.md)|Возвращает числовое выражение, округленное до указанной длины или точности. , и делает это по-другому.|  
|[ЗНАК &#40;выражение служб SSIS&#41;](sign-ssis-expression.md)|Возвращает знак выражения: плюс (+), минус (-) или нуль (0).|  
|[КВАДРАТНЫЙ &#40;выражение служб SSIS&#41;](square-ssis-expression.md)|Возвращает квадрат числового выражения.|  
|[SQRT &#40;выражение служб SSIS&#41;](sqrt-ssis-expression.md)|Возвращает квадратный корень числового выражения.|

|[FINDSTRING &#40;выражение служб SSIS&#41;](findstring-ssis-expression.md)|Возвращает однократный индекс указанного вхождения символьной строки в выражение.|  
|[HEX &#40;выражение служб SSIS&#41;](hex-ssis-expression.md)|Возвращает строку, представляющую собой шестнадцатеричное значение целого числа.|

|[НИЖЕ &#40;выражение служб SSIS&#41;](lower-ssis-expression.md)|Возвращает символьное выражение после преобразования всех символов верхнего регистра в нижний.|  
|[Функция LTRIM &#40;выражение служб SSIS&#41;](trim-ssis-expression.md)|Возвращает символьное выражение после удаления начальных пробелов.|

|[РЕПЛИЦИРОВАТЬ &#40;выражение служб SSIS&#41;](replicate-ssis-expression.md)|Возвращает символьное выражение, реплицированное указанное число раз.|

|[ПРАВО &#40;выражение служб SSIS&#41;](right-ssis-expression.md)|Возвращает указанное количество символов из крайней правой части заданного символьного выражения.|  
|[RTRIM &#40;выражение служб SSIS&#41;](rtrim-ssis-expression.md)|Возвращает символьное выражение после удаления конечных пробелов.|

|[ВЕРХНЯЯ &#40;выражение служб SSIS&#41;](upper-ssis-expression.md)|Возвращает символьное выражение после преобразования символов в нижнем регистре в символы верхнего регистра.|

|Функция|Описание|  
|--------------|-----------------|

|[Функция GETDATE &#40;выражение служб SSIS&#41;](getdate-ssis-expression.md)|Возвращает текущую системную дату.|

|[МЕСЯЦ &#40;выражение служб SSIS&#41;](month-ssis-expression.md)|Возвращает целое число, представляющее месяц указанной даты.|

 Средство оценки выражений содержит следующие функции для значения NULL.  
  
|Компонент|Описание|  
|--------------|-----------------|  
|[Функция ISNULL &#40;выражение служб SSIS&#41;](null-ssis-expression.md)|Возвращает результат в виде логического выражения, в зависимости от того, имеет ли выражение значение NULL.|  
|[Значение NULL &#40;выражение служб SSIS&#41;](null-ssis-expression.md)|Возвращает значение NULL запрошенного типа данных.|  
  
 Названия выражений указаны в верхнем регистре, но эти имена обрабатываются без учета регистра. Например, значение «null» равносильно использованию значения «NULL».
