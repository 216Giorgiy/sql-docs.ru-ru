# <a name="raw-file-source-editor-columns-page"></a>Редактор источника «Необработанный файл»  (страница "Столбцы")
  Источник «Необработанный файл» считывает необработанные данные из файла. Поскольку данные представлены в собственном формате источника, их преобразование не требуется. Также практически не требуется синтаксический анализ. Дополнительные сведения см. в статье [Raw File Source](data-flow/raw-file-source.md).
