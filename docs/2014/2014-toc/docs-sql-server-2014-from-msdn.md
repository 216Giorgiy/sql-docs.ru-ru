# Документация по SQL Server 2014 перемещается в документы

В этой статье описывается перемещение предстоящих статей документации Microsoft SQL Server 2014. В этой статье записывается в конце 2018 апреля и перемещения произойдет перед 2018 июля.

## Ссылки на документацию по SQL Server 2014 на MSDN

Ниже приведены ссылки в документации SQL Server 2014 Microsoft с сохраненными на его *MSDN* веб-сайта. Перед 2018 июля Microsoft будет переместить эти статьи документации новым *Docs* веб-сайта:

- *SQL Server 2014*
    - [https://msdn.microsoft.com/library/dn197878(v=sql.10).aspx](https://msdn.microsoft.com/library/dn197878(v=sql.10).aspx)

- *Электронная документация по SQL Server 2014*
    - [https://msdn.microsoft.com/library/ms130214(v=sql.120).aspx](https://msdn.microsoft.com/library/ms130214(v=sql.120).aspx)

MSDN является initialism для *Microsoft Developer Network*.


## Перенаправителями

Устаревшие адреса MSDN HTTP будут продолжать работать по крайней мере на несколько лет после перемещения. Перенаправителями будет перехватывать запросы MSDN и пересылают их с их новых соответствующие адреса в документы. Однако после перемещения будет иметь значение true следующие элементы:

- Он будет лучше работать с нового адреса HTTP, которые переходить непосредственно к документации.
- Запустится программ-обходчиков Bing и Google delist и обходить старые адреса MSDN.


## После перемещения

После перемещения в *Docs*, адрес статьи документации по SQL Server 2014 корневой будет:

- [https://docs.microsoft.com/sql/?view=sql-server-2014](https://docs.microsoft.com/sql/?view=sql-server-2014)
