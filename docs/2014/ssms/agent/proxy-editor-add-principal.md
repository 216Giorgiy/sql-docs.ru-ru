# <a name="proxy-editor-and--add-principal"></a>Редактор прокси-сервера и добавить участника
  При помощи этой страницы можно предоставить участникам сервера доступ к учетной записи-посреднику агента [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="options"></a>Параметры  
 **Тип участника**  
 Тип участника для отображения.  
  
 **Доступные участники**  
 Список участников выбранного типа.  
  
 **Название**  
 Выберите имя участника, которому необходимо предоставить доступ, или **Имя** , чтобы предоставить доступ всем участникам в списке.
