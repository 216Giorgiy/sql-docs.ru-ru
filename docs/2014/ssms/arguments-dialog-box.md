# <a name="arguments-dialog-box"></a>диалоговое окно «Аргументы»
  В диалоговом окне **Аргументы** можно указать для средства новые или существующие аргументы. Оно появляется при запуске внешнего средства, для которого в окне **Внешние средства** установлен параметр **Запрашивать аргументы** .  
  
## <a name="options"></a>Параметры  
 **Аргументы**  
 Содержит список переменных, заданных для средства в диалоговом окне **Внешние инструменты** . Полный список стандартных аргументов и их определений см. в разделе [Arguments for External Tools](menu-help/external-tools.md). Также можно указать пользовательский аргумент.  
  
 **Командная строка**  
 Отображает значения аргументов, указанных в диалоговом окне **Аргументы** .
