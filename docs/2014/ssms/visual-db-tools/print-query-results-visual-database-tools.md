# <a name="print-query-results-visual-database-tools"></a>Печать результатов запроса (визуальные инструменты для баз данных)
  Чтобы распечатать результаты запроса, необходимо скопировать содержимое [панели результатов](visual-database-tools.md) в другую программу Windows, например в программу подготовки текстов, затем произвести в ней форматирование и распечатать результаты запроса из этой программы.  
  
 Конструктор запросов и представлений копирует результаты запроса в буфер обмена, используя символы табуляции в качестве разделителей между столбцами и символов возврата каретки, а символы новой строки в качестве разделителей между строками.  
  
### <a name="to-print-query-results"></a>Печать результатов запроса  
  
1.  Выберите строки или столбцы, которые нужно распечатать.  
  
2.  Воспользуйтесь командой **Копировать** для перемещения их в буфер обмена.  
  
3.  Переключитесь на программу Windows, при помощи которой нужно распечатать результаты.  
  
4.  Воспользуйтесь командой **Вставить** для перемещения результатов запроса из буфера обмена в программу.  
  
5.  Отформатируйте и напечатайте результаты.
