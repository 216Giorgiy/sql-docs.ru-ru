# <a name="customize-the-amount-of-information-displayed-in-diagrams-visual-database-tools"></a>Настройка объема сведений, отображаемых в диаграммах (визуальные инструменты для баз данных)
  При добавлении таблицы в диаграмму базы данных эта таблица появляется вместе с рядом отображаемых свойств. Количество отображаемых сведений можно задать в меню вариаций или создавать пользовательский набор свойств.  
  
 Возможные варианты:  
  
-   Standard  
  
-   Имена столбцов  
  
-   Ключи  
  
-   Только имя  
  
-   Другой  
  
-   Изменить пользовательский  
  
### <a name="to-choose-a-table-view"></a>Выбор табличного представления  
  
1.  Необходимо щелкнуть правой кнопкой мыши таблицу в конструкторе диаграмм баз данных.  
  
2.  В контекстном меню укажите пункт Табличное представление и выберите нужное представление.  
  
### <a name="to-create-a-custom-table-view"></a>Создание пользовательского табличного представления  
  
1.  Щелкните правой кнопкой мыши таблицу в конструкторе диаграмм баз данных.  
  
2.  В контекстном меню укажите пункт **Вид таблицы** и выберите **Изменить пользовательский**.  
  
3.  Выберите свойства для отображения в диалоговом окне **Выбор столбцов** , если в меню **Вид таблицы** выбран пункт **Пользовательский** .
