# <a name="mining-accuracy-chart-designer-data-mining"></a>Конструктор диаграммы точности интеллектуального анализа данных (интеллектуальный анализ данных)
  Используйте представление **Диаграмма точности интеллектуального анализа данных** , чтобы сравнить эффективность выполнения моделей интеллектуального анализа данных, построенных на основе выбранной структуры интеллектуального анализа данных.  
  
 Представление **Диаграмма точности интеллектуального анализа данных** содержит следующие вкладки.  
  
-   [Вкладка Selection входного &#40;интеллектуального анализа данных представление диаграммы точности&#41;](input-selection-tab-mining-accuracy-chart-view.md)  
  
     Выберите модель или модели для отображения на диаграмме точности прогнозов, диаграмме роста прибыли, точечной диаграмме или в матрице классификации. Укажите проверяемые данные и сопоставьте столбцы внешних данных с данными модели.  
  
-   [Вкладка Диаграмма точности прогнозов &#40;интеллектуального анализа данных представление диаграммы точности&#41;](lift-chart-tab-mining-accuracy-chart-view.md)  
  
     Создайте диаграмму точности прогнозов, диаграмму роста прибыли или точечную диаграмму для моделей, выделенных на вкладке `Input Selection`.  
  
-   [Вкладка «матрица классификации» &#40;интеллектуального анализа данных представление диаграммы точности&#41;](classification-matrix-tab-mining-accuracy-chart-view.md)  
  
     Отображает матрицу классификации для моделей, выбранных на `Input Selection` вкладку.  
  
-   [Вкладки перекрестной проверки &#40;интеллектуального анализа данных представление диаграммы точности&#41;](cross-validation-tab-mining-accuracy-chart-view.md)  
  
     Задайте параметры и создайте результаты перекрестной проверки для выбранной структуры интеллектуального анализа данных.  
  
 Вкладки содержат следующие средства и диалоговые окна:  
  
-   [Выбор интеллектуального анализа данных диалоговое окно «Структура» &#40;интеллектуального анализа данных представление диаграммы точности&#41;](select-mining-structure-dialog-box-mining-accuracy-chart-view.md)  
  
-   [Выберите таблицу-диалоговое окно &#40;интеллектуального анализа данных представление диаграммы точности&#41;](select-table-dialog-box-mining-accuracy-chart-view.md)  
  
-   [Диалоговое окно «вложенного соединения» &#40;интеллектуального анализа данных представление диаграммы точности&#41;](specify-nested-join-dialog-box-mining-accuracy-chart-view.md)  
  
-   [Диалоговое окно «Параметры диаграммы» прибыль &#40;интеллектуального анализа данных представление диаграммы точности&#41;](profit-chart-settings-dialog-box-mining-accuracy-chart-view.md)  
  
-   [Изменение сопоставления диалоговое окно &#40;интеллектуального анализа данных диаграммы точности&#41;](modify-mapping-dialog-box-mining-accuracy-chart.md)  
  
-   [Диалоговое окно фильтра &#40;интеллектуального анализа данных диаграммы точности&#41;](filter-dialog-box-mining-accuracy-chart.md)  
  
-   [Фильтр набора данных или диалоговое окно Фильтр модели](data-set-filter-or-model-filter-dialog-box.md)
