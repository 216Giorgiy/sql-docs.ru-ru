# <a name="mining-accuracy-chart-designer-data-mining"></a>Конструктор диаграммы точности интеллектуального анализа данных (интеллектуальный анализ данных)
  Используйте представление **Диаграмма точности интеллектуального анализа данных** , чтобы сравнить эффективность выполнения моделей интеллектуального анализа данных, построенных на основе выбранной структуры интеллектуального анализа данных.  
  
 Представление **Диаграмма точности интеллектуального анализа данных** содержит следующие вкладки.  
  
-   [Вкладка выбора ввода &#40;представление диаграммы точности интеллектуального анализа данных&#41;](input-selection-tab-mining-accuracy-chart-view.md)  
  
     Выберите модель или модели для отображения на диаграмме точности прогнозов, диаграмме роста прибыли, точечной диаграмме или в матрице классификации. Укажите проверяемые данные и сопоставьте столбцы внешних данных с данными модели.  
  
-   [Вкладка Диаграмма точности прогнозов &#40;представление диаграммы точности интеллектуального анализа данных&#41;](lift-chart-tab-mining-accuracy-chart-view.md)  
  
     Создайте диаграмму точности прогнозов, диаграмму роста прибыли или точечную диаграмму для моделей, выделенных на вкладке `Input Selection`.  
  
-   [Вкладка «матрица классификации» &#40;представление диаграммы точности интеллектуального анализа данных&#41;](classification-matrix-tab-mining-accuracy-chart-view.md)  
  
     Отображает матрицу классификации для моделей, выбранных на `Input Selection` вкладку.  
  
-   [Вкладка «перекрестная проверка» &#40;представление диаграммы точности интеллектуального анализа данных&#41;](cross-validation-tab-mining-accuracy-chart-view.md)  
  
     Задайте параметры и создайте результаты перекрестной проверки для выбранной структуры интеллектуального анализа данных.  
  
 Вкладки содержат следующие средства и диалоговые окна:  
  
-   [Выберите диалоговое окно структуры интеллектуального анализа &#40;представление диаграммы точности интеллектуального анализа данных&#41;](select-mining-structure-dialog-box-mining-accuracy-chart-view.md)  
  
-   [Диалоговое окно «таблицы» выберите &#40;представление диаграммы точности интеллектуального анализа данных&#41;](select-table-dialog-box-mining-accuracy-chart-view.md)  
  
-   [Диалоговое окно «вложенного соединения» &#40;представление диаграммы точности интеллектуального анализа данных&#41;](specify-nested-join-dialog-box-mining-accuracy-chart-view.md)  
  
-   [Диалоговое окно параметров диаграммы роста прибыли &#40;представление диаграммы точности интеллектуального анализа данных&#41;](profit-chart-settings-dialog-box-mining-accuracy-chart-view.md)  
  
-   [Изменение сопоставления диалоговое окно &#40;диаграмма точности интеллектуального анализа данных&#41;](modify-mapping-dialog-box-mining-accuracy-chart.md)  
  
-   [Диалоговое окно фильтрации &#40;диаграмма точности интеллектуального анализа данных&#41;](filter-dialog-box-mining-accuracy-chart.md)  
  
-   [Фильтр набора данных или диалоговое окно Фильтр модели](data-set-filter-or-model-filter-dialog-box.md)
