# <a name="add-reference-dialog-box-analysis-services---multidimensional-data"></a>Диалоговое окно «Добавление ссылки» (службы Analysis Services — многомерные данные)
  Диалоговое окно **Добавление ссылки** в среде [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] используется для добавления ссылки на сборку [!INCLUDE[msCoName](../includes/msconame-md.md)] .NET Framework или на другой проект разработки. Чтобы открыть диалоговое окно **Добавление ссылки**, щелкните правой кнопкой мыши папку **Сборки** проекта служб [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] в **обозревателе решений** и выберите в контекстном меню пункт **Создать ссылку на сборку**.  
  
## <a name="options"></a>Параметры  
  
|Термин|Определение|  
|----------|----------------|  
|**.NET**|На этой вкладке можно добавить ссылку на зарегистрированный компонент. На этой вкладке содержится сетка со списком зарегистрированных компонентов .NET Framework. Выберите один или несколько элементов из сетки, затем нажмите кнопку **Добавить** , чтобы добавить выбранные компоненты .NET к списку **Выбранные продукты и компоненты**. Сетка содержит следующие столбцы:<br /><br /> **Имя компонента**. Полное или понятное наименование компонента. Щелкните заголовок столбца для сортировки по названию компонента.<br /><br /> **Версия**: номер версии компонента. Щелкните заголовок столбца для сортировки по версии.<br /><br /> **Среда выполнения**: версия .NET Framework, на которой основан компонент. Щелкните заголовок столбца, чтобы отсортировать по версии среды выполнения.<br /><br /> **Путь**: имя файла компонента и путь, в котором он находится. Щелкните заголовок столбца для сортировки по пути.|  
|**Обзор**|Щелкните этот столбец для сборки, не указанной на вкладках **.NET** или **Последние** . Вкладка содержит следующие параметры:<br /><br /> **Искать в**: выберите папку из этого раскрывающегося списка. Если выбрать папку в этом списке, в основной панели отображается содержимое папки.<br /><br /> **Перейдите к последней просмотренной папке**: возвращает **папка** к предыдущему размещению.<br /><br /> **Один уровень вверх**: переходит Далее папке более высокого уровня в иерархии.<br /><br /> **Создание новой папки**: выберите, чтобы создать новую вложенную папку в папке, выбранной в **папка**.<br /><br /> **Просмотр меню**: выберите, чтобы изменить режим представления в основной панели.  Дополнительные сведения о параметрах режима **Меню "Вид"[!INCLUDE[msCoName](../includes/msconame-md.md)] см. в разделе "Обзор просмотра файлов и папок" документации**  Windows. Доступны следующие параметры:<br />Эскизы страниц<br />Плитка<br />Значки<br />Список<br />Сведения<br /><br /> **Основной панели**: отображает содержимое папки, выбранной в **папка**. Выберите один или несколько элементов и нажмите кнопку **добавить** для добавления файлов, выбранных для **выбранные продукты и компоненты**. Дополнительные сведения о параметрах и контекстном меню основной панели см. в разделе «Обзор просмотра файлов и папок» документации Windows.<br /><br /> **Имя файла**: используйте этот параметр для фильтрации файлов и папок, которые отображаются. Введите часть имени файла или имя файла целиком, которое будет использоваться фильтром. В качестве символа-шаблона можно использовать символ звездочки (\*). Чтобы выбрать несколько файлов, введите несколько имен (каждое имя должно быть заключено в парные двойные кавычки ("), и отделено пробелом). В раскрывающемся списке также можно выбрать ранее введенные файлы. Совет: Вы можете найти веб-серверы и сетевые компьютеры, введя URL-адрес или сетевой путь в **имя файла**. Например, при вводе "http://mywebsite" отобразятся файлы, доступные на веб-сайте "mywebsite", а при вводе "\\\myserver\myshare" — файлы, расположенные в папке "myshare" на сервере "myserver".<br /><br /> **Файлы типа**: используйте этот параметр для фильтрации содержимого папки или каталога, выбранного в **папка** для определенного типа файлов.|  
|**Последние**|Отображает список ссылок на компоненты, добавленные к проектам в среде [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)] последними. Выберите один или несколько элементов из этой сетки, затем нажмите кнопку **Добавить** , чтобы добавить выбранные сборки .NET Framework к списку **Выбранные продукты и компоненты**. Сетка содержит следующие столбцы:<br /><br /> **Имя компонента**. Полное или понятное наименование компонента. Щелкните заголовок столбца для сортировки по названию компонента.<br /><br /> **Версия**: номер версии компонента. Щелкните заголовок столбца для сортировки по версии.<br /><br /> **Среда выполнения**: версия .NET Framework, на которой основан компонент. Щелкните заголовок столбца, чтобы отсортировать по версии среды выполнения.<br /><br /> **Путь**: имя файла компонента и путь, в котором он находится. Щелкните заголовок столбца для сортировки по пути.|  
|**Добавить**|Нажмите, чтобы добавить компонент, выбранный на вкладках **.NET**, **Обзор**или **Последние** к списку **Выбранные проекты и компоненты**.|  
|**Удалить**|Нажмите, чтобы удалить компонент, выбранный в списке **Выбранные проекты и компоненты**.|  
|**Выбранные проекты и компоненты**|Список ссылок на компоненты, добавленных к проекту служб [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] в среде [!INCLUDE[ssBIDevStudioFull](../includes/ssbidevstudiofull-md.md)]. Выберите один или несколько элементов в сетке, затем нажмите кнопку **Удалить** , чтобы удалить из сетки выбранные компоненты. Сетка содержит следующие столбцы:<br /><br /> **Имя компонента**. Полное или понятное наименование компонента. Щелкните заголовок столбца для сортировки по названию компонента.<br /><br /> **Версия**: номер версии компонента. Щелкните заголовок столбца для сортировки по версии.<br /><br /> **Среда выполнения**: версия .NET Framework, на которой основан компонент. Щелкните заголовок столбца, чтобы отсортировать по версии среды выполнения.<br /><br /> **Путь**: имя файла компонента и путь, в котором он находится. Щелкните заголовок столбца для сортировки по пути.|
