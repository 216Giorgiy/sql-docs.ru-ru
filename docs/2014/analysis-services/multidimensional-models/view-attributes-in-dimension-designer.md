# <a name="view-attributes-in-a-tree-list-or-grid-in-dimension-designer"></a>Просмотр атрибутов в дереве, списке или сетке в конструкторе измерений
  В данной процедуре описывается изменение способа отображения атрибутов в конструкторе измерений.  
  
### <a name="to-change-the-format-of-the-attributes-pane-in-dimension-designer"></a>Изменение формата панели «Атрибуты» в построителе измерений  
  
1.  В конструкторе измерений откройте измерение, содержащее атрибуты, с которыми необходимо работать, и перейдите на вкладку «Структура измерения».  
  
2.  Щелкните в любом месте панели **Атрибуты** , чтобы установить фокус на нее **Атрибуты**.  
  
3.  Выберите меню **Измерение**или щелкните правой кнопкой мыши панель **Атрибуты** .  
  
4.  Выберите пункт **Показать атрибуты в**, а затем выберите пункт **Дерево**, **Сетка**или **Список**.
