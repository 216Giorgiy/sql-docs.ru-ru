# <a name="analysis-services-scripting-language-xml-elements-assl"></a>Элементы XML языка сценариев служб Analysis Services (ASSL)
  Хотя в схеме языка ASSL содержатся только элементы XML, с точки зрения разработчика этим элементам соответствуют следующие понятия.  
  
-   Объекты, такие как `Database`, `Cube`, и `Dimension` объектов в иерархии объектов, содержащихся в экземпляре [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssASnoversion](../../includes/ssasnoversion-md.md)].  
  
-   Коллекции объектов, например `Dimensions` и `Cubes`.  
  
-   Свойства, описывающие объекты, например `Name` и `Description`.  
  
-   Типы данных (например `Binding` и `Permission`), которые используются для определения дочерних элементов и свойств других объектов.  
  
    > [!NOTE]  
    >  В этом разделе не описываются элементы типов данных. Дополнительные сведения см. в разделе, [Analysis Services Scripting типы данных XML в &#40;ASSL&#41;](data-type/analysis-services-scripting-language-xml-data-types-assl.md).
