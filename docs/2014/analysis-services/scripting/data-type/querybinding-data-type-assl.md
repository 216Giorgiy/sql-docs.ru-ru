# <a name="querybinding-data-type-assl"></a>Тип данных QueryBinding (ASSL)
  Определяет производный тип данных, представляющий связь [DataSource](../objects/datasource-element-assl.md) элемент с [QueryDefinition](../properties/querydefinition-element-assl.md) элемента.  
  
## <a name="syntax"></a>Синтаксис  
  
```xml  
  
<QueryBinding>  
   <!-- The following elements extend TabularBinding -->  
   <DataSourceID>...</DataSourceID>  
      <QueryDefinition>...</QueryDefinition>  
</QueryBinding>  
```  
  
## <a name="data-type-characteristics"></a>Характеристики типа данных  
  
|Характеристика|Описание|  
|--------------------|-----------------|  
|Базовые типы данных|[TabularBinding](binding-data-type-assl.md)|  
|Производные типы данных|None|  
  
## <a name="data-type-relationships"></a>Связи типа данных  
  
|Связь|Элемент|  
|------------------|-------------|  
|Родительские элементы|None|  
|Дочерние элементы|[DataSourceID](../properties/id-element-assl.md), [QueryDefinition](../properties/querydefinition-element-assl.md)|  
|Производные элементы|В разделе [привязки](binding-data-type-assl.md)|
