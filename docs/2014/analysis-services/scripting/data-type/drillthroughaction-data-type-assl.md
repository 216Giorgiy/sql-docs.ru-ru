# <a name="drillthroughaction-data-type-assl"></a>Тип данных DrillThroughAction (ASSL)
  Определяет производный тип данных, представляющий действие детализации.  
  
## <a name="syntax"></a>Синтаксис  
  
```xml  
  
<DrillThroughAction>  
   <!-- The following elements extend Action -->  
   <Default>...</Default>  
   <Columns>...</Columns>  
</DrillThroughAction>  
```  
  
## <a name="data-type-characteristics"></a>Характеристики типа данных  
  
|Характеристика|Описание|  
|--------------------|-----------------|  
|Базовые типы данных|[Действие](action-data-type-assl.md)|  
|Производные типы данных|None|  
  
## <a name="data-type-relationships"></a>Связи типа данных  
  
|Связь|Элемент|  
|------------------|-------------|  
|Родительские элементы|None|  
|Дочерние элементы|[Столбцы](../collections/columns-element-assl.md), [по умолчанию](../properties/default-element-assl.md)|  
|Производные элементы|[Действие](../objects/action-element-assl.md) ([действия](../collections/actions-element-assl.md), коллекция [куба](../objects/cube-element-assl.md), или [перспективы](../objects/perspective-element-assl.md))|  
  
## <a name="remarks"></a>Примечания  
 Соответствующий элемент в объектной модели Analysis Management объекты AMO — это <xref:Microsoft.AnalysisServices.DrillThroughAction>.
