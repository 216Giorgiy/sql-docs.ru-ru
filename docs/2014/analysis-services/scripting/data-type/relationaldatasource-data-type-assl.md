# <a name="relationaldatasource-data-type-assl"></a>Тип данных RelationalDataSource (ASSL)
  Определяет производный тип данных, представляющий [DataSource](../objects/datasource-element-assl.md) основанный на реляционном источнике данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```xml  
  
<RelationalDataSource>  
   <!-- Child elements are only those inherited from DataSource -->  
</RelationalDataSource>  
```  
  
## <a name="data-type-characteristics"></a>Характеристики типа данных  
  
|Характеристика|Описание|  
|--------------------|-----------------|  
|Базовые типы данных|[Источник данных](datasource-data-type-assl.md)|  
|Производные типы данных|None|  
  
## <a name="data-type-relationships"></a>Связи типа данных  
  
|Связь|Элемент|  
|------------------|-------------|  
|Родительские элементы|None|  
|Дочерние элементы|None|  
|Производные элементы|[Источник данных](../objects/datasource-element-assl.md) ([DataSources](../collections/datasources-element-assl.md) коллекцию [базы данных](../objects/database-element-assl.md))|  
  
## <a name="remarks"></a>Примечания  
 Соответствующий элемент в модели объектов объекты управления Analysis AMO — это <xref:Microsoft.AnalysisServices.RelationalDataSource>.
