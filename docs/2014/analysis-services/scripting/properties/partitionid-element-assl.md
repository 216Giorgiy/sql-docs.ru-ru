# <a name="partitionid-element-assl"></a>Элемент PartitionID (ASSL)
  Связывает [секции](../objects/partition-element-assl.md) элемента с родительским элементом, привязка или привязка вне строки.  
  
## <a name="syntax"></a>Синтаксис  
  
```xml  
  
<PartitionBinding>  
   ...  
   <PartitionID>...</PartitionID>  
   ...  
</PartitionBinding>  
```  
  
## <a name="element-characteristics"></a>Характеристики элемента  
  
|Характеристика|Описание|  
|--------------------|-----------------|  
|Тип данных и длина|String|  
|Значение по умолчанию|None|  
|Количество элементов|1-1: обязательный элемент, который может появляться только один раз.|  
  
## <a name="element-relationships"></a>Связи элемента  
  
|Связь|Элемент|  
|------------------|-------------|  
|Родительский элемент|[PartitionBinding](../data-type/binding-data-type-assl.md)|  
|Дочерние элементы|None|
