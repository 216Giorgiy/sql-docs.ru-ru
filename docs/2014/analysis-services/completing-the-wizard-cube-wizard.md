# <a name="completing-the-wizard-cube-wizard"></a>Завершение работы мастера (мастер кубов)
  Страница **Завершение работы мастера** позволяет присвоить имя кубу, который будет создан после завершения работы мастера, и предварительно просмотреть этот куб.  
  
## <a name="options"></a>Параметры  
 **Имя куба**  
 Введите имя куба.  
  
 **Предварительный просмотр**  
 Выводит на экран структуру куба, который будет создан после завершения работы мастера.
