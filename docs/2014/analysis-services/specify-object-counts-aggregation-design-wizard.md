# <a name="specify-object-counts-aggregation-design-wizard"></a>Определение счетчиков объектов (мастер статистических схем)
  Для автоматического вычисления количества объектов в кубе или ввода оценочного числа вручную используется страница **Определение счетчиков объектов** . С помощью счетчиков объектов мастер статистических схем оценивает требования к хранилищу.  
  
## <a name="options"></a>Параметры  
 **Объекты кубов**  
 Выводит измерения и атрибуты куба. Только атрибуты, у которых нет их `AggregationUsage` свойство значение `None` в **просмотр использования статистической обработки** страницы мастера отображаются, так как только атрибуты, которые требуется указывать счетчики.  
  
 **Оценочное число**  
 Выводит оценочное число строк группы мер и оценочное число элементов атрибута в измерениях базы данных. Значения оценочного числа можно ввести вручную или вычислить. Чтобы вычислить значения этих, введите `0` в поле и нажмите кнопку **число**. Поля, где уже отображаются оценочные числа, не обновляются.  
  
 **Число секций**  
 Введите оценочное число строк группы мер и оценочное число элементов атрибута в секциях (необязательно).  
  
 **Count**  
 Вычисляет и повторно заполняет значения в столбце **Оценочное число** для всех пустых полей. Поля, где уже отображаются оценочные числа, не обновляются.
