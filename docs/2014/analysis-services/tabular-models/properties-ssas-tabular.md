# <a name="properties-ssas-tabular"></a>Свойства (табличные службы SSAS)
  Проекты табличной модели в среде [!INCLUDE[ssBIDevStudioFull](../../includes/ssbidevstudiofull-md.md)] имеют различные свойства, которые определяют поведение проекта, модели, отчетов и развертывания. Параметры свойств сохраняются в формате XML в файле Model.bim, однако все свойства, описанные в этом разделе, можно настроить в окнах **Свойства** в среде [!INCLUDE[ssBIDevStudio](../../includes/ssbidevstudio-md.md)].  
  
## <a name="related-tasks"></a>Связанные задачи  
  
|Раздел|Описание|  
|-----------|-----------------|  
|[Power просматривать свойства отчетов &#40;табличные службы SSAS&#41;](power-view-reporting-properties-ssas-tabular.md)|Данный раздел содержит описания и параметры конфигурации для табличных моделей, для работы с которыми будет выполняться подключение из [!INCLUDE[ssCrescent](../../includes/sscrescent-md.md)].|  
|[Свойства проекта &#40;табличные службы SSAS&#41;](project-properties-ssas-tabular.md)|Содержит описание свойств проекта. Свойства проекта включают файл проекта и параметры развертывания.|  
|[Свойств модели &#40;табличные службы SSAS&#41;](model-properties-ssas-tabular.md)|Содержит описание свойств модели. Во время разработки модели свойства модели определяют действия построения проекта модели, резервное копирование и базу данных рабочей области.|  
|[Свойства таблицы &#40;табличные службы SSAS&#41;](table-properties-ssas-tabular.md)|Содержит описание базовых свойств таблицы. Описанные здесь свойства отличаются от свойств таблицы в диалоговом окне «Изменение свойств таблицы», которые отображают и обеспечивают выделение и фильтрацию столбцов из источника.|  
|[Свойства столбца &#40;табличные службы SSAS&#41;](column-properties-ssas-tabular.md)|Содержит описание свойств столбца. Свойства столбца определяют тип данных столбца, формат и параметры скрытия.|  
|[Настройка моделирования данных по умолчанию и свойств развертывания &#40;табличные службы SSAS&#41;](configure-default-data-modeling-and-deployment-properties-ssas-tabular.md)|Предоставляет описания и шаги настройки свойств моделирования и развертывания, установленных по умолчанию. Свойства по умолчанию применяются к новым проектам табличной модели. При необходимости после создания проекта эти свойства в конкретном проекте модели можно изменить.|
