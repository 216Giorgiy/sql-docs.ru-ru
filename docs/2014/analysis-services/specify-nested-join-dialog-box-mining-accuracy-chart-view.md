# <a name="specify-nested-join-dialog-box-mining-accuracy-chart-view"></a>Диалоговое окно «Определение вложенного соединения» (диаграмма точности интеллектуального анализа данных)
  С помощью диалогового окна **Определение вложенного соединения** создается связь между вложенной таблицей и таблицей вариантов. Чтобы открыть диалоговое окно **Определение вложенного соединения** , выберите вложенную таблицу и нажмите кнопку **Изменить соединение**.  
  
## <a name="options"></a>Параметры  
 **Добавить связь**  
 Нажмите для открытия диалогового окна **Создание связи** .  
  
 **Удалить связь**  
 Нажмите для удаления связи, выбранной на панели **Диаграмма** .
