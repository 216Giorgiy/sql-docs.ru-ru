# <a name="completing-the-wizard-linked-object-wizard"></a>Завершение работы мастера (мастер связанных объектов)
  Чтобы просмотреть решения, принятые на предыдущих страницах мастера, воспользуйтесь страницей **Завершение работы мастера** .  
  
## <a name="options"></a>Параметры  
 **Предварительный просмотр**  
 Показывает объекты, выбранные для связывания или импорта.
