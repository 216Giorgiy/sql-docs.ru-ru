# <a name="connect-to-database-dialog-box"></a>Диалоговое окно «Соединение с базой данных»
  Используйте диалоговое окно **Соединение с базой данных** для подключения к базе данных служб [!INCLUDE[ssASnoversion](../includes/ssasnoversion-md.md)] .  
  
## <a name="options"></a>Параметры  
  
|Термин|Определение|  
|----------|----------------|  
|**Подключиться к существующей базе данных**|Выберите этот параметр для подключения к существующей базе данных. В списке существующих баз данных щелкните ту, к которой необходимо подключиться.|  
|**Создание новой базы данных**|Выберите этот параметр для создания базы данных.|  
|**Добавьте в решение**|Выберите этот параметр для добавления базы данных к текущему решению.|  
|**Закрыть решение**|Выберите этот параметр для добавления базы данных к новому решению.|
