# <a name="notification-events-event-category"></a>категория событий «События уведомления»
  В категорию событий «События уведомления» входят классы событий, описанные в следующей таблице.  
  
|Класс событий|Идентификатор события|Описание|  
|-----------------|--------------|-----------------|  
|Уведомление|39|Позволяет собрать все события уведомления после запуска трассировки.|  
|Определяемые пользователем|40|Собирает определяемые пользователем события запуска команд с момента начала трассировки.|  
  
 Дополнительные сведения о столбцах, связанных с каждым из классов событий «События уведомления», см. в разделе [Notification Events Data Columns](notification-events-data-columns.md).
