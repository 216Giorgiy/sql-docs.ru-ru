# <a name="command-events-event-category"></a>категория событий «Командные события»
  К категории «Командные события» относятся следующие классы событий.  
  
|Класс событий|Идентификатор события|Описание|  
|-----------------|--------------|-----------------|  
|Command Begin|15|Объединяет события запуска команд после начала трассировки.|  
|Command End|16|Объединяет события завершения команд после начала трассировки.|  
  
 Дополнительные сведения о столбцах, связанных с каждым классом событий «Командные события», см. в разделе [Command Events Data Columns](command-events-data-columns.md).
