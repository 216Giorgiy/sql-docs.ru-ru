# <a name="select-how-to-define-the-connection-data-source-wizard-analysis-services"></a>Выбор метода определения соединения (мастер источников данных) (службы Analysis Services)
  Страница **Выбор метода определения соединения** используется для создания или выбора подключения к данным для источника данных.  
  
## <a name="options"></a>Параметры  
 **Создать источник данных на основе существующего или нового соединения**  
 Выберите этот параметр для использования существующей строки соединения или создания новой строки соединения для объекта источника данных.  
  
 **Подключения к данным**  
 Выберите существующее подключение к данным из списка.  
  
 **Свойства подключения к данным**  
 Отображает настройки свойств соединения для подключения к данным, выбранного в пункте **Подключения к данным**.  
  
 **Новые функции...**  
 Нажмите, чтобы создать новый источник данных, используя диалоговое окно **Диспетчер соединений** .  
  
 **Удаление**  
 Выберите это параметр для удаления существующего источника данных.  
  
 **Создать источник данных на основе другого объекта**  
 Выберите этот параметр, чтобы создать новый источник данных на основе существующего объекта источника данных или существующего проекта.
