# <a name="mssqlserver12302"></a>MSSQLSERVER_12302
    
## <a name="details"></a>Сведения  
  
|||  
|-|-|  
|Название продукта|SQL Server|  
|Идентификатор события|12302|  
|Источник события|MSSQLSERVER|  
|Компонент|SQLEngine|  
|Символическое имя|HK_UNSUPPORTED_COMPUTED_COLUMNS|  
|Текст сообщения|Обновление столбцов, входящих в ограничение PRIMARY KEY, не поддерживается для '*построение*'.|  
  
## <a name="user-action"></a>Действие пользователя  
 Не обновляйте столбцы, входящие в состав ограничения первичного ключа.
