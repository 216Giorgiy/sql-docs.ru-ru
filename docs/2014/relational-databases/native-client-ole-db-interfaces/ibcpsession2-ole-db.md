# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
  IBCPSession2, интерфейс является расширением IBCPSession, предоставляющий функцию-член является альтернативой вызывающий IBCPSession::BCPColFmt для каждого столбца.  IBCPSession2, наследующий от IBCPSession и добавлением одного нового метода: [метода IBCPSession2::BCPSetBulkMode](ibcpsession2-bcpsetbulkmode.md).
