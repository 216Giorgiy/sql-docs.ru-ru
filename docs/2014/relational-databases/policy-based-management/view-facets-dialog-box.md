# <a name="view-facets-dialog-box"></a>Диалоговое окно «Просмотр аспектов»
  Это диалоговое окно предназначено для просмотра свойств аспекта управления на основе политик. Свойства аспекта определяются во время его создания и не могут быть изменены в этом диалоговом окне.  
  
## <a name="options"></a>Параметры  
 **Аспект**  
 Имя выбранного в данный момент аспекта.  
  
 **Описание**  
 Предоставляет общее описание аспекта.  
  
 **Свойства аспекта**  
 Отображает имена и настройки свойств аспекта.  
  
 **Экспортировать текущее состояние как политику**  
 Щелкните здесь, чтобы экспортировать описанное состояние свойств аспекта в виде новой политики в XML-файл.
