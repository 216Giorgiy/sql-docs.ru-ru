# <a name="subscription-properties---ltsubscriptiongt"></a>Свойства подписки — &lt;Подписка&gt;
  В этом разделе приводится сведения о диалоговом окне **Свойства подписки** .  
  
-   В статье [Свойства подписки — подписчик](subscription-properties-subscriber.md) рассматриваются свойства подписок, доступные на подписчике (только для подписок по запросу).  
  
-   В[Subscription Properties - Publisher](subscription-properties-publisher.md) рассматриваются свойства подписок, доступные на издателе (для подписок по запросу и принудительных подписок).
