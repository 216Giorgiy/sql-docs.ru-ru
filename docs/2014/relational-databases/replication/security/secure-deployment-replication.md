# <a name="secure-deployment-replication"></a>Безопасное развертывание (репликация)
  В этом разделе приводятся сведения, помогающие лучше защитить репликацию.  
  
## <a name="in-this-section"></a>в этом разделе  
 [Защита распространителя](secure-the-distributor.md)  
 Содержит описание обеспечения безопасности распространителя.  
  
 [Организация безопасности издателя](secure-the-publisher.md)  
 Содержит описание списка доступа к публикации (PAL) и рекомендуемые методы его использования.  
  
 [Защита подписчика](secure-the-subscriber.md)  
 Содержит описание соединений, устанавливаемых с подписчиками, и соглашения, касающиеся подписчиков с обновлением транзакций.  
  
 [Защита папки моментальных снимков](secure-the-snapshot-folder.md)  
 Содержит описание разрешений, необходимых для доступа к папке моментального снимка, рекомендуемые методы определения местоположения папки и рассмотрение вопросов доставки моментальных снимков по протоколу FTP.  
  
 [Защита репликации через Интернет](securing-replication-over-the-internet.md)  
 Содержит сведения о виртуальных частных сетях и веб-синхронизации.  
  
 [Рекомендации по безопасности для репликации баз данных выпуска SQL Server Compact](security-considerations-for-sql-server-compact-replication.md)  
 Объясняется обеспечение безопасности соединения между экземплярами [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)] [!INCLUDE[ssDE](../../../includes/ssde-md.md)] и [!INCLUDE[ssEW](../../../includes/ssew-md.md)].
