# <a name="deprecated-features-in-sql-server-replication"></a>Устаревшие функции репликации SQL Server
  В этом разделе описаны устаревшие компоненты репликации, по-прежнему доступные в [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)]. Эти функции будут удалены в следующем выпуске [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Не следует использовать устаревшие функции в новых приложениях.  
  
## <a name="items-deprecated-in-sql-server-2014"></a>Элементы, устаревшие в SQL Server 2014  
  
|Компонент|Описание|  
|-------------|-----------------|  
|[!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)]|Репликация поддерживается, если на каждой конечной точке [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] установлена одна из двух основных версий текущей версии [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Поэтому [!INCLUDE[ssSQL14](../../includes/sssql14-md.md)] не поддерживает репликацию в тип данных [!INCLUDE[ssVersion2005](../../includes/ssversion2005-md.md)].|  
|[!INCLUDE[ssEW](../../includes/ssew-md.md)]|Репликация поддерживается, если на каждой конечной точке [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] установлена одна из двух основных версий текущей версии [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Поэтому [!INCLUDE[ssSQL14](../../includes/sssql14-md.md)] не поддерживает репликацию в тип данных [!INCLUDE[ssEW](../../includes/ssew-md.md)].|
