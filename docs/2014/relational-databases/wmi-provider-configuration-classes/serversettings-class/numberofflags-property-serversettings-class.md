# <a name="numberofflags-property-serversettings-class"></a>Свойство NumberOfFlags (класс ServerSettings)
  Возвращает число общих флагов, связанных с экземпляром [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object  
.NumberOfFlags [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *object*  
 Объект [класса ServerSettings](serversettings-class.md) , представляющий параметры сервера в экземпляре [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Значение `uint32`, определяющее число общих флагов, связанных с экземпляром [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
