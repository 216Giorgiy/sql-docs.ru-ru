# <a name="sqlserviceadvancedproperty-class"></a>Класс SqlServiceAdvancedProperty
  [Класс SqlServiceAdvancedProperty](sqlserviceadvancedproperty-class.md) представляет дополнительное свойство службы, на который ссылается [класс SqlService](../sqlservice-class/sqlservice-class.md) объекта.  
  
 [Свойство AdvancedProperties (класс SqlService)](../sqlservice-class/advancedproperties-property-sqlservice-class.md) ссылается на массив [класс SqlServiceAdvancedProperty](sqlserviceadvancedproperty-class.md) объектов.  
  
 [Запуск и остановка служб](http://technet.microsoft.com/library/ms174886\(v=sql.105\).aspx) класс представляет свойства, уникальные для службы. Эти свойства не входят в список свойств, связанных с [класс SqlService](http://technet.microsoft.com/library/ms186497.aspx) класса. [Класс SqlServiceAdvancedProperty](http://technet.microsoft.com/library/ms182447.aspx) класс позволяет представлять строковые, числовые или логические свойства. Этот класс можно использовать для просмотра уникальных свойств заданной службы.
