# <a name="clientnetworkprotocolproperty-class"></a>Класс ClientNetworkProtocolProperty
  Класс [ClientNetworkProtocolProperty](clientnetworkprotocolproperty-class.md) представляет свойства сетевого протокола клиента.  
  
 При работе с классом [ClientNetworkProtocolProperty](clientnetworkprotocolproperty-class.md)могут быть выполнены следующие задачи.  
  
-   Указание значения индекса для выбора свойства сетевого протокола клиента из списка свойств.  
  
-   Проверка протокола, которому принадлежит текущее свойство.  
  
-   Проверка имени, типа и значения текущего свойства.  
  
-   Изменение строковых и числовых значений текущего свойства.  
  
-   Установка флагов для текущего свойства.
