# <a name="flagvalue-property-clientsettingsgeneralflag-class"></a>Свойство FlagValue (класс ClientSettingsGeneralFlag)
  Возвращает значение флага.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object  
.FlagValue [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *object*  
 Объект [класса ClientSettingsGeneralFlag](clientsettingsgeneralflag-class.md) , представляющий универсальный флаг для параметров сервера.  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Логическое значение, указывающее состояние флага.
