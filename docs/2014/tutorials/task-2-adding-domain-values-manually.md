# <a name="task-2-adding-domain-values-manually"></a>Задача 2. Добавление значений домена вручную
  В этой задаче добавляется новое значение для **страны** домена вручную. В разделе [изменение значений домена](http://msdn.microsoft.com/library/hh510408.aspx) Дополнительные сведения о полях на этой странице.  
  
1.  Нажмите кнопку **страны** домена в **список доменов**.  
  
2.  На правой панели переключитесь **значения домена** вкладки.  
  
3.  Нажмите кнопку **добавить новое значение домена** на панели инструментов в области справа.  
  
     ![Добавить новое значение домена — кнопка панели инструментов](../../2014/tutorials/media/et-addingdomainvaluesmanually-01.jpg "добавить новое значение домена — кнопка панели инструментов")  
  
4.  Тип **США** для **значение** поле и нажмите клавишу **ввод**. Вы можете увидеть, что по умолчанию **тип** равно **исправьте** (зеленый флажок). Тип может быть присвоено **ошибка** (красный крест) или **недопустимый** (оранжевый треугольник), и можно ввести правильное значение в **исправить на** поля.  
  
     ![США для значения домена](../../2014/tutorials/media/et-addingdomainvaluesmanually-02.jpg "США для значения домена")
