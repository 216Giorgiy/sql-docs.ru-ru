# <a name="task-2-uploading-supplier-data-to-mds-using-mds-add-in-for-excel"></a>Задача 2. Передача данных поставщика в MDS с помощью надстройки MDS для Excel
  В этой задаче очищенные и поставщика данных для публикации **MDS** с помощью **надстройки MDS для Excel**. Создать сущность с именем **Supplier** в **поставщики** модель, созданная на предыдущем занятии. Сущность будет иметь атрибуты для каждого столбца в файле Excel. Атрибуты кода и имени сущности Supplier соответствуют **SupplierID** и **имя поставщика** столбцов в Excel.  
  
1.  Откройте **очищены и сопоставлены Suppliers.xls** в **Excel**.  
  
2.  Нажмите клавишу **CTRL + A** чтобы выбрать все данные. Это **важные** выбрать все данные в электронной таблице.  
  
3.  Нажмите кнопку **основных данных** в строке меню.  
  
4.  Нажмите кнопку **создать сущность** кнопки на ленте.  
  
     ![Excel — вкладка основных данных — кнопка «Создать сущность»](../../2014/tutorials/media/et-ulingsdtomdsusingmdsaddinforexcel-01.jpg "Excel — вкладка основных данных — кнопка «Создать сущность»")  
  
5.  В **Управление соединениями** диалоговое окно, если отсутствует подключение к **локальный сервер MDS** под **существующие подключения**, выполните следующие действия:  
  
    1.  Выберите **создайте новое соединение,** и нажмите кнопку **New** кнопки.  
  
    2.  В **Добавление нового соединения** диалоговом **локальный сервер MDS** для **описание** и **http://localhost/MDS** для  **Адрес сервера MDS**и нажмите кнопку **ОК** чтобы закрыть диалоговое окно.  
  
6.  В **Управление соединениями** выберите **локальный сервер MDS** (http://localhost/MDS), нажмите кнопку **тестирования** для проверки соединения. Нажмите кнопку **ОК** в окне сообщения.  
  
7.  Нажмите кнопку **Connect** для подключения к серверу MDS.  
  
8.  В **создать сущность** выберите **поставщики** для **модели**.  
  
9. Убедитесь, что **VERSION_1** выбран для **версии**.  
  
10. Введите **Supplier** для **новое имя сущности**.  
  
11. Выберите **SupplierID** для **столбец, содержащий уникальный идентификатор** поля (также можно создать код автоматически). По сути, выполняется сопоставление **SupplierID** столбца в **Excel** для **кода** атрибут **Supplier** сущности.  
  
12. Выберите **имя поставщика** для **столбец, содержащий имена** поля. По сути, выполняется сопоставление **имя поставщика** столбца в **Excel** для **имя** атрибут **Supplier** сущности. **Кода** и **имя** атрибуты являются обязательными атрибутами сущности в MDS.  
  
     ![Сущности диалоговое окно создания](../../2014/tutorials/media/et-ulingsdtomdsusingmdsaddinforexcel-02.jpg "создать сущность-диалоговое окно")  
  
13. Нажмите кнопку **ОК** создание сущности в MDS, опубликуйте основные данные в сущность и закрыть **создать сущность** диалоговое окно.  
  
14. Теперь вы увидите новый лист **Supplier**, — имя сущности, добавлены в листе Excel и в верхней части листа вы увидите, что лист подключен к серверу MDS. Обратите внимание, что исходный лист (под названием **Sheet1**) все еще существует.  
  
     ![Excel — вкладки Supplier и Sheet1](../../2014/tutorials/media/et-ulingsdtomdsusingmdsaddinforexcel-03.jpg "Excel — вкладки Supplier и Sheet1")  
  
     ![Excel — отображаются сведения о соединении с MDS](../../2014/tutorials/media/et-ulingsdtomdsusingmdsaddinforexcel-04.jpg "Excel — отображаются сведения о соединении с MDS")  
  
15. Сохранить **Excel** открыть.
