# <a name="task-1-prerequisite-removing-supplier-data-in-mds"></a>Задача 1 (предварительно). Удаление сведений о поставщике данных в MDS
  В этой задаче будут удалены данные о поставщиках, хранящихся в MDS. Вы отправили данные вручную с помощью **надстройки MDS для Excel** на предыдущем занятии. Пакет служб SSIS, который будет создан на этом занятии, будет автоматически загружать данные в MDS. Поэтому перед тестированием пакета служб SSIS необходимо удалить данные поставщиков из MDS, удалить производную иерархию, удалить сущности поставщиков и состояний, а также создать сущность поставщика без данных.  
  
1.  Запустите **диспетчера основных данных** , перейдя к **http://localhost/MDS** или веб-сайт и приложение, указанное при настройке MDS. Если **диспетчера основных данных** открыть, щелкните **SQL Server 2012 Master Data Services** в верхней части, чтобы переключиться на **Домашняя страница**.  
  
2.  Нажмите кнопку **системное администрирование** в **административных задач** раздела.  
  
3.  Наведите указатель мыши **управление** на меню и выберите пункт **производные иерархии**. Необходимо удалить производную иерархию **SuppliersInState** перед удалением сущностей в **поставщики** модели.  
  
4.  Выберите **SuppliersInState** из **производной иерархии** списке и нажмите кнопку **X (Delete)** на панели инструментов.  
  
5.  Нажмите кнопку **ОК** для подтверждения удаления.  
  
6.  Наведите указатель мыши **управление** на меню и выберите пункт **сущности**.  
  
7.  Нажмите кнопку **Supplier** и нажмите кнопку **удаления (X)** кнопку на панели инструментов, чтобы удалить сущность. Нажмите кнопку **ОК** в окнах сообщений.  
  
8.  Повторите предыдущий шаг, чтобы удалить **состояние** сущности.  
  
9. Не закрывайте **диспетчера основных данных**.  
  
10. Переключитесь в окно Excel, в котором **Cleansed and Matched Suppliers.xls** файл открыт. Переключитесь в **Sheet1** вкладку внизу.  
  
11. Выберите только **первая строка с заголовками**. Не выбирайте другие строки. Нам необходимо создать сущности на основании столбцов Excel, не загружая при этом данные. Именно поэтому следует выбрать только первую строку с заголовками.  
  
12. Нажмите кнопку **основных данных** в строке меню.  
  
13. Нажмите кнопку **создать сущность** на ленте.  
  
14. В **Управление соединениями** диалоговое окно, если отсутствует подключение к **локальный сервер MDS** под **существующие подключения**, выполните следующие действия:  
  
    1.  Выберите **создайте новое соединение,** и нажмите кнопку **New** кнопки.  
  
    2.  В диалоговом окне Добавление нового соединения введите **локальный сервер MDS** для **описание** и **http://localhost/MDS** для **адрес сервера MDS**, и нажмите кнопку **ОК** чтобы закрыть диалоговое окно.  
  
15. В **Управление соединениями** выберите **локальный сервер MDS** (http://localhost/MDS), нажмите кнопку **тестирования** для проверки соединения. Нажмите кнопку **ОК** в окне сообщения.  
  
16. Нажмите кнопку **Connect** для подключения к серверу MDS.  
  
17. В **создать сущность** диалогового окна поле, выполните следующие действия:  
  
    1.  Убедитесь, что **диапазон** равно **$1: $1**.  
  
    2.  Выберите **поставщики** для **модели**.  
  
    3.  Выберите **VERSION_1** для **версии**.  
  
    4.  Тип **Supplier** для **новое имя сущности**.  
  
    5.  Выберите **SupplierID** для **кода**.  
  
    6.  Выберите **имя поставщика** для **имя**.  
  
    7.  Нажмите кнопку **ОК** создать сущность и закрыть диалоговое окно.  
  
18. Закрыть **Excel** и **не сохранять** файл.  
  
19. В **диспетчера основных данных**, обновите Интернет-браузер и убедитесь, что **Supplier** сущность отображается в списке.  
  
20. Переключитесь в **Домашняя страница** , щелкнув **SQL Server 2012 Master Data Services** вверху.  
  
21. Убедитесь, что **поставщики** выбрана модель **модели** и **VERSION_1** выбран для **версии**.  
  
22. Нажмите кнопку **Браузер**. Обратите внимание, что **Supplier** сущности со всеми атрибутами создается с **значения не**.
