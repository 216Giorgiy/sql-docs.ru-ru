# <a name="task-5-exporting-cleansing-results-to-an-excel-file"></a>Задача 5. Экспорт результатов очистки в файл Excel.
  В этой задаче вы экспортируете результаты операции очистки в файл Excel. В разделе [этап экспорта](http://msdn.microsoft.com/library/hh213061.aspx#Export) Дополнительные сведения см.  
  
1.  В области справа выберите **Excel** для **конечный тип**.  
  
2.  Нажмите кнопку **Обзор**, укажите имя выходного файла, как **Cleansed Supplier List.xls**, а затем нажмите кнопку **откройте**.  
  
3.  Выберите **только данные** для **вывода** формат, чтобы экспортировать только очищенные данные. Второй параметр, **данным и очистке**, позволяет экспортировать сведения об очистке вместе с очищенными данными. **Стандартный формат** позволяет применить все форматы вывода, заданные в домене и значениям этого домена. В этом учебнике формат вывода не был задан ни для одного домена.  
  
     ![Страница результатов очистки экспорта](../../2014/tutorials/media/et-exportingcleansingresultstoanexcelfile.jpg "очистки экспорта страница «результаты»")  
  
4.  Нажмите кнопку **Экспорт** для экспорта данных. Не нажимайте кнопку **Готово** еще.  
  
5.  Нажмите кнопку **закрыть** на **Экспорт** диалоговое окно.  
  
6.  Нажмите кнопку **Готово** , чтобы завершить работу. Если вы забыли экспортировать результаты перед нажатием кнопки **Готово**, нажмите кнопку **открыть проект качества данных** главной странице **клиента DQS**выберите **поставщика очистки Список** из списка проектов и нажмите кнопку **Далее** в нижней части экрана, чтобы получить **Экспорт** этапа процесса очистки. Вы также можете перейти на **управление и просмотр результатов** , нажав кнопку **обратно** кнопки.  
  
7.  Откройте **Cleansed Supplier List.xls** и выполните следующие действия:  
  
    1.  Убедитесь, что адреса электронной почты, заканчивающиеся на adventure-work.com (без символа «s»), отсутствуют, выполнив поиск «adventure-work.com» на листе.  
  
    2.  Убедитесь, что имеется не **США** значение в **страны** столбца.  
  
    3.  Поиск **Лос-Анджелес** и убедитесь, что **состояние** равно **ЦС**.  
  
    4.  Убедитесь, что условия не **Co.**, **Corp.**, и **Inc.**.  
  
    5.  Удалить **проверки адреса** столбца из электронной таблицы и сохраните файл excel. Этот дополнительный столбец соответствует составному домену проверки адреса.
