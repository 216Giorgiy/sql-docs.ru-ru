# <a name="task-3-optional-reviewing-the-subscription-views"></a>Задача 3 (дополнительно). Просмотр представлений подписки
  В этой задаче вы ознакомитесь с созданием представлений SQL с помощью среды SQL Server Management Studio.  
  
1.  Запустите **SQL Server Management Studio**. Нажмите кнопку **запустить** , выберите элемент **все программы**, нажмите кнопку **Microsoft SQL Server 2012**, а затем нажмите кнопку **SQL Server Management Studio**.  
  
2.  В **соединение с сервером** задайте **тип сервера** для **СУБД**, тип **имя сервера** (или выберите **() Local)** и выберите соответствующую **проверки подлинности**и нажмите кнопку **Connect**.  
  
3.  В **обозревателя объектов** области, разверните **баз данных**, разверните **MDS**и разверните **представления**.  
  
4.  Подтвердите, что вы видите **mdm. Поставщики** представление в списке.  
  
     ![SQL Server Management Studio - mdm. Поставщики представление](../../2014/tutorials/media/et-reviewingthesubscriptionviews.jpg "SQL Server Management Studio - mdm. Просмотр поставщиков")
