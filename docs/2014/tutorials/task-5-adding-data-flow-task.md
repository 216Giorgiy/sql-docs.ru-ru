# <a name="task-5-adding-data-flow-task"></a>Задача 5. Добавление задачи потока данных
  В этой задаче вы добавите задачу потока данных в поток управления пакетом служб SSIS.  
  
1.  Перетаскивание **задачи потока данных** из **область элементов служб SSIS** для **поток управления** в конструкторе служб SSIS. Если вы не видите **область элементов служб SSIS**, щелкните в любом месте **поток управления** щелкните **SSIS** в строке меню и нажмите кнопку **область элементов служб SSIS**.  
  
     ![Поток данных перетащите задачи из области элементов служб SSIS](../../2014/tutorials/media/et-addingdataflowtask.jpg "потока данных перетащите задачи из области элементов служб SSIS")  
  
2.  Щелкните правой кнопкой мыши **задачи потока данных** в **поток управления** и нажмите кнопку **переименование**.  
  
3.  Тип **Receive, очистка, соответствия и исправление данных поставщика** и нажмите клавишу **ввод**.  
  
4.  Дважды щелкните **задачи потока данных** переключение **потока данных** вкладки.
