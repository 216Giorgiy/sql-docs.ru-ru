# <a name="task-4-setting-domain-rules"></a>Задача 4. Задание правил домена
  В этой задаче вы создаете правило для **адрес электронной почты** домена, чтобы проверить, оканчивается ли адрес электронной почты **@adventure-works.com**. В разделе [создания правила домена](http://msdn.microsoft.com/library/hh510397.aspx) Дополнительные сведения на странице.  
  
1.  Нажмите кнопку **адрес электронной почты** в **список доменов**.  
  
2.  Переключитесь в **правил домена** вкладку на правой панели.  
  
     ![Добавление новой кнопки панели инструментов правило домена](../../2014/tutorials/media/et-settingdomainrules-01.jpg "Добавление новой кнопки панели инструментов правила домена")  
  
3.  В области справа щелкните **добавить новое правило домена** на панели инструментов (см. рисунок) для добавления правила.  
  
4.  Тип **проверки по электронной почте** для **имя правила** и нажмите клавишу **ввод**. **Active** флажок должен быть установлен по умолчанию. Этот элемент управления позволяет временно отключить правило.  
  
5.  В **построить правило** области, нажмите кнопку **стрелка вниз**и выберите **значение заканчивается на**.  
  
6.  Тип **@adventure-works.com** в текстовом поле и нажмите клавишу **ВКЛАДКА**. Можно добавить дополнительные условия, нажав кнопку **добавить новое условие в выбранное предложение** кнопки панели инструментов в **построить правило** области.  
  
     ![Правило проверки электронной почты](../../2014/tutorials/media/et-settingdomainrules-02.jpg "правило проверки электронной почты")  
  
7.  Нажмите кнопку **запустить выбранное правило домена к тестовым данным** на панели инструментов на правой панели, чтобы проверить правило на основе образцов данных.  
  
     ![Запустить правило домена на кнопки панели инструментов данных теста](../../2014/tutorials/media/et-settingdomainrules-03.jpg "запустить правило домена на кнопки панели инструментов данных теста")  
  
8.  В **проверка доменного правила** диалоговое окно, нажмите кнопку **добавляет новый проверочный термин для правила домена** на панели инструментов.  
  
     ![Проверить диалоговое окно правила домена](../../2014/tutorials/media/et-settingdomainrules-04.jpg "проверить правило домена-диалоговое окно")  
  
9. Тип **frank7@adventure-works.com** (допустимое значение) в **адрес электронной почты** столбца.  
  
10. Повторите два предыдущих шага, чтобы добавить **joe2@adventure-work.com** (без использования недопустимое значение элемента ").  
  
11. Нажмите последнюю кнопку (**проверить правило домена на всех терминах**) на панели инструментов, чтобы проверить входные данные для правила.  
  
     ![Проверить правило домена на панели инструментов кнопку все условия](../../2014/tutorials/media/et-settingdomainrules-05.jpg "проверить правило домена на панели инструментов кнопку все условия")  
  
12. Обратите внимание, что первая запись отображается как допустимый элемент, а вторая — как недопустимый элемент.  
  
     ![Результаты проверки правила домена](../../2014/tutorials/media/et-settingdomainrules-06.jpg "результаты проверки правила домена")  
  
13. Нажмите кнопку **закрыть** закрыть **проверка доменного правила** диалоговое окно.
