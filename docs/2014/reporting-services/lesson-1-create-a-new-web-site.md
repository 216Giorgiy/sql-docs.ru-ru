# <a name="lesson-1-create-a-new-web-site"></a>Урок 1. Создание нового веб-сайта
  На этом занятии показано, как создать новый проект веб-сайта с использованием шаблона веб-сайта ASP.NET для Visual C#.  
  
### <a name="to-create-a-new-website"></a>Создание нового веб-сайта  
  
1.  Откройте Microsoft Visual Studio 2010 или Microsoft Visual Studio 2012.  
  
2.  В меню **Файл** укажите пункт **Создать**и выберите **Веб-сайт**.  
  
3.  В диалоговом окне **Создание веб-сайта** на панели **Установленные шаблоны** выберите **Visual C#** , а затем **Веб-сайт ASP.NET**.  
  
4.  В поле **Расположение** укажите каталог проекта и нажмите кнопку **ОК**.  
  
 Откроется проект веб-сайта.
