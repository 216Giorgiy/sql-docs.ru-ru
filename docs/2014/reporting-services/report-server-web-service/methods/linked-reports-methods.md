# <a name="linked-reports-methods"></a>Методы связанных отчетов
  Эти методы позволяют создавать и управлять связанными элементами.  
  
|Метод|Действие|  
|------------|------------|  
|<xref:ReportService2010.ReportingService2010.CreateLinkedItem%2A>|Добавляет новый связанный элемент в базу данных сервера отчетов.|  
|<xref:ReportService2010.ReportingService2010.GetItemLink%2A>|Возвращает имя элемента, описание которого используется для указанного связанного элемента.|  
|<xref:ReportService2010.ReportingService2010.ListDependentItems%2A>|Возвращает список элементов, связанных с указанным элементом.|  
|<xref:ReportService2010.ReportingService2010.SetItemLink%2A>|Указывает элемент каталога, используемый для описания существующего связанного элемента.|
