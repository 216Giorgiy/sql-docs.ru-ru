# <a name="report-designer-f1-help"></a>Справка F1 конструктора отчетов
  Этот раздел содержит справку F1 для [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] [!INCLUDE[ssRSnoversion](../../includes/ssrsnoversion-md.md)] мастеров, представлений и диалоговым окнам конструктора отчетов. Открыть справку F1 во время работы с диалоговым окном можно, нажав клавишу F1 или кнопку справки, расположенную в диалоговом окне.
