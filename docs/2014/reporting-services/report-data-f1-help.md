# <a name="report-data-f1-help"></a>Справка F1 по данным отчета
  Этот раздел содержит информацию о параметрах, которые можно задавать для источников данных, общих источников данных и баз данных, а также о конструкторах запросов, доступных в конструкторах отчетов.  
  
 Чтобы получить доступ к этой информации, в диалоговом окне **Свойства отчета** , выберите **Справка**.
