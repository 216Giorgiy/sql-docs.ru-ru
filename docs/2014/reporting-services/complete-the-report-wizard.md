# <a name="complete-the-report-wizard"></a>Завершение работы мастера отчетов
  На этой странице можно указать имя отчета и просмотреть параметры, которые должны быть применены к отчету.  
  
## <a name="options"></a>Параметры  
 **Имя отчета**  
 Введите имя отчета. Это имя также отображается в заголовке отчета. Заголовок отчета можно также изменить в режиме конструктора.  
  
 **Сводка по отчету**  
 Просмотр параметров, выбранных на предыдущих шагах мастера отчетов. Если необходимо изменить какой-либо параметр, нажимайте кнопку **Назад** до тех пор, пока не будет найдена страница, на которой это параметр находится.  
  
 **Предварительный просмотр отчета**  
 Выберите этот параметр, чтобы просмотреть созданный отчет. В противном случае отчет будет открыт в режиме конструктора.
