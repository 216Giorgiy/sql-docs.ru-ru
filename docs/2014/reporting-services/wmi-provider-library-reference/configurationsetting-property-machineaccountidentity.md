# <a name="machineaccountidentity-property-wmi"></a>Свойство MachineAccountIdentity (WMI)
  Возвращает удостоверение учетной записи компьютера, на котором установлен сервер отчетов.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
Public Dim MachineAccountIdentity As String  
```  
  
```csharp  
public string MachineAccountIdentity;  
```  
  
## <a name="property-values"></a>Значения свойств  
 Объект `String` значение, содержащее идентификатор учетной записи компьютера, установленной на сервере отчетов.
