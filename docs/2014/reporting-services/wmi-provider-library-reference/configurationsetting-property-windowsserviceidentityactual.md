# <a name="windowsserviceidentityactual-property-wmi-msreportserverconfigurationsetting"></a>Свойство WindowsServiceIdentityActual (WMI MSReportServer_ConfigurationSetting)
  Идентификатор, под которым выполняется служба Windows сервера отчетов. Только для чтения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
Public Dim WindowsServiceIdentityActual As String  
```  
  
```csharp  
public string WindowsServiceIdentityActual;  
```  
  
## <a name="property-values"></a>Значения свойств  
 Объект `String` объект, содержащий идентификатор которой запущена служба Windows сервера отчетов.
