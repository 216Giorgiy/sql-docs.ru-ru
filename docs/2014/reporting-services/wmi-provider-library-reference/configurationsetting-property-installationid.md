# <a name="installationid-property-wmi-msreportserverconfigurationsetting"></a>Свойство InstallationID (WMI MSReportServer_ConfigurationSetting)
  Это свойство возвращает уникальный идентификатор для определенного экземпляра сервера отчетов.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
public Dim InstanceID As String  
```  
  
```csharp  
public string InstanceID;  
```
