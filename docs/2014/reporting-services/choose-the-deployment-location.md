# <a name="choose-the-deployment-location"></a>Выбор места развертывания
  Эта страница мастера отчетов предназначена для выбора сервера и папки развертывания проекта сервера отчетов.  
  
## <a name="options"></a>Параметры  
 **Сервер отчетов**  
 Например, введите URL-адрес виртуального каталога сервера отчетов, http://localhost/reportserver. Это виртуальный каталог сервера отчетов, а не диспетчера отчетов. По умолчанию сервер отчетов устанавливается в виртуальный каталог reportserver.  
  
 **Папка развертывания**  
 Введите имя папки назначения для публикации отчетов, содержащихся в проекте сервера отчетов. По умолчанию папке присваивается имя проекта. Если на сервере отчетов папка не существует, то конструктор отчетов создаст ее при публикации отчетов.
