# <a name="open-a-trace-file-sql-server-profiler"></a>открыть файл трассировки (приложение SQL Server Profiler)
  В этом разделе объясняется, как открыть файл трассировки с помощью приложения [!INCLUDE[ssSqlProfiler](../../includes/sssqlprofiler-md.md)].  
  
### <a name="to-open-a-trace-file"></a>Открытие файла трассировки  
  
1.  В меню **Файл** выберите команду **Открыть**и щелкните пункт **Файл трассировки**.  
  
2.  В диалоговом окне **Открытие файла** выберите нужный файл данных трассировки.
