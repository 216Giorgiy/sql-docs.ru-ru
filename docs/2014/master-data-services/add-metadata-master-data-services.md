# <a name="add-metadata-master-data-services"></a>добавление метаданных (службы Master Data Services)
  В [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)], определяемые пользователем метаданные для добавления **метаданные** модели для описания объектов модели.  
  
## <a name="prerequisites"></a>предварительные требования  
 Для выполнения этой процедуры:  
  
-   Необходимо иметь разрешение на доступ к функциональной области **Обозреватель** .  
  
-   Для **метаданные** модели, необходимо иметь **добавить** разрешение на метаданные сущности или тип конечного элемента, необходимо добавить.  
  
-   Версия должна быть в состоянии **Открыта**.  
  
### <a name="to-add-metadata"></a>Добавление метаданных  
  
1.  На [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)] домашней странице **модель** выберите **метаданные** модели.  
  
2.  Выберите версию из списка **Версия** .  
  
3.  Нажмите кнопку **Браузер**.  
  
4.  В строке меню наведите курсор на **сущностей** и выберите сущность метаданных, которые вы хотите добавить.  
  
5.  Нажмите кнопку **Add Member** кнопки. **Добавление нового элемента** страница будет отображена в правой части этой страницы.  
  
6.  Введите новое имя, код и описание, а затем нажмите кнопку **ОК**.
