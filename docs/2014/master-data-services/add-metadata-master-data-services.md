# <a name="add-metadata-master-data-services"></a>добавление метаданных (службы Master Data Services)
  В [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)], добавить определяемые пользователем метаданные для **метаданных** модели для описания объектов модели.  
  
## <a name="prerequisites"></a>предварительные требования  
 Для выполнения этой процедуры:  
  
-   Необходимо иметь разрешение на доступ к функциональной области **Обозреватель** .  
  
-   Для **метаданных** модели, необходимо иметь **добавить** разрешения на метаданные сущности или тип конечного элемента, вы хотите добавить.  
  
-   Версия должна быть в состоянии **Открыта**.  
  
### <a name="to-add-metadata"></a>Добавление метаданных  
  
1.  На [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)] домашней странице **модели** выберите **метаданных** модели.  
  
2.  Выберите версию из списка **Версия** .  
  
3.  Нажмите кнопку **Браузер**.  
  
4.  На панели меню наведите указатель мыши на **сущностей** и выберите сущность метаданных, который вы хотите добавить.  
  
5.  Нажмите кнопку **Add Member** кнопки. **Добавление нового элемента** страницы будет отображаться в правой части этой страницы.  
  
6.  Введите новое имя, код и описание, а затем нажмите кнопку **ОК**.
