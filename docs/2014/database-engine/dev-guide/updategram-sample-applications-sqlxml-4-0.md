 Все примеры в этом разделе используют образец базы данных AdventureWorks в [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. Все обновления применяются к таблицам базы данных AdventureWorks.
