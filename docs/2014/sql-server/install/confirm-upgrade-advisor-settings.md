# <a name="confirm-upgrade-advisor-settings"></a>Подтверждение настроек помощника по обновлению
  Эта страница позволяет просмотреть параметры, выбранные в мастере анализа помощника по обновлению. После просмотра параметров, нажмите кнопку **запуска** чтобы начать анализ.  
  
## <a name="options"></a>Параметры  
 **Параметры помощника по обновлению**  
 Позволяет просмотреть все данные, введенные для каждого из компонентов, а также параметры, относящиеся к работе помощника по обновлению в целом. Содержит параметры отчетов, в том числе выходной каталог.  
  
 **Отправлять отчеты в [!INCLUDE[msCoName](../../includes/msconame-md.md)]. Эти отчеты могут содержать личные сведения.**  
 Эта команда отправляет анализ, чтобы [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] team. Эти данные будут использованы для улучшения будущих версий продукта.  
  
 **Просмотреть заявление о конфиденциальности.**  
 Щелкните эту ссылку, чтобы прочитать [!INCLUDE[msCoName](../../includes/msconame-md.md)] политику конфиденциальности в Интернете.
