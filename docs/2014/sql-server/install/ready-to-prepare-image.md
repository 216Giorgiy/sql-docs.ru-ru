# <a name="ready-to-prepare-image"></a>Все готово для подготовки образа
  На странице **Все готово для подготовки образа** приведен сводный список компонентов, которые будут подготовлены программой установки [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .  
  
## <a name="options"></a>Параметры  
 На этой странице доступны следующие параметры.  
  
-   Чтобы продолжить установку, нажмите кнопку **Установить**.  
  
-   Чтобы изменить настройки установки, нажмите кнопку **Назад**.  
  
-   Чтобы выйти из программы установки, нажмите кнопку **Отмена**.
