# <a name="sql-server-native-soap-support-is-discontinued-in-this-version-of-sql-server"></a>Поддержка собственной реализации доступа по протоколу SOAP в SQL Server прекращается в этой версии SQL Server.
  Помощник по обновлению обнаружил использование собственных XML-веб-служб [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
## <a name="component"></a>Компонент  
 [!INCLUDE[ssDE](../../includes/ssde-md.md)]  
  
## <a name="description"></a>Описание  
 Собственные XML-веб-службы [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] в этом выпуске [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] отсутствуют.  
  
## <a name="discovering-where-you-use-native-xml-web-services"></a>Поиск мест использования собственных XML-веб-служб  
 Выяснить, в каких приложениях используются собственные XML-веб-службы, можно следующим образом.  
  
-   При запуске помощника по обновлению.  
  
-   При обновлении версии [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)].  
  
## <a name="corrective-action"></a>Действие по исправлению  
 Измените приложения, в настоящее время использующие собственные XML-веб-службы.
