# <a name="sql-server-2014-upgrade-advisor"></a>Помощник по обновлению SQL Server 2014
  [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] Помощник по обновлению анализирует конфигурацию установленных [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] компоненты, а затем создает отчет о проблемах, которые необходимо решить, чтобы обеспечить успешное обновление.  
  
 Получение файла установки помощника по обновлению (sqlua.msi) в папке "redist" установочного носителя или как части [загрузки пакета дополнительных компонентов SQL 2014](http://www.microsoft.com/download/details.aspx?id=42295).
