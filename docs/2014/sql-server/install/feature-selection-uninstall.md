# <a name="feature-selection-uninstall"></a>Выбор компонентов (удаление)
  Указать компоненты, которые нужно удалить из указанного экземпляра [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)], можно на странице «Выбор компонентов — удаление».  
  
## <a name="options"></a>Параметры  
 Можно установить любое сочетание компонентов.
