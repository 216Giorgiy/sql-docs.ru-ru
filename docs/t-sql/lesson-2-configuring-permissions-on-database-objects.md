# <a name="lesson-2-configuring-permissions-on-database-objects"></a>Занятие 2. Настройка разрешений на объекты базы данных
[!INCLUDE[tsql-appliesto-ss2008-all-md](../includes/tsql-appliesto-ss2008-all-md.md)]Предоставление пользователю доступа к базе данных состоит из трех этапов. Вначале создается имя входа. Имя входа дает пользователю возможность подключиться к компоненту [!INCLUDE[ssDEnoversion](../includes/ssdenoversion-md.md)]. Затем имя входа настраивается как пользователь в заданной базе данных. Наконец, предоставляются пользовательские разрешения на объекты базы данных. В этом занятии показаны все три шага, а также создание представления и хранимой процедуры в виде объекта.  
  
Это занятие содержит следующие разделы:

-   [Предоставление доступа к базе данных](../t-sql/lesson-2-2-granting-access-to-a-database.md)  
  
-   [Создание представлений и хранимых процедур](../t-sql/lesson-2-3-creating-views-and-stored-procedures.md)
