# <a name="json-functions-transact-sql"></a>Функции JSON (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2016-asdb-xxxx-xxx-md](../../includes/tsql-appliesto-ss2016-asdb-xxxx-xxx-md.md)]

Используйте функции, описанные на страницах в этом разделе, чтобы проверить или изменить текст JSON или для извлечения простых или сложных значений.  
  
|Функция|Description|  
|--------------|-----------------|  
|[ISJSON](../../t-sql/functions/isjson-transact-sql.md)|Проверяет, является ли строка содержит допустимые данные JSON.|  
|[JSON_VALUE](../../t-sql/functions/json-value-transact-sql.md)|Извлекает скалярное значение из строки JSON.|  
|[JSON_QUERY](../../t-sql/functions/json-query-transact-sql.md)|Извлекает объект или массив из строки JSON.|  
|[JSON_MODIFY](../../t-sql/functions/json-modify-transact-sql.md)|Обновляет значение свойства в строке JSON и возвращает обновленную строку JSON.|

 Дополнительные сведения о встроенной поддержке JSON в [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)], в разделе [данных JSON &#40; SQL Server &#41; ](../../relational-databases/json/json-data-sql-server.md).
