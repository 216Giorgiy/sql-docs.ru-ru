# <a name="transact-sql-reference-database-engine"></a>Справочник по Transact-SQL (компонент Database Engine)
[!INCLUDE[tsql-appliesto-ss2008-all-md](../includes/tsql-appliesto-ss2008-all-md.md)]

В этом разделе приводится общее представление о том, как находить и использовать разделы справки Microsoft Transact-SQL (T-SQL). T-SQL является ключом к использованию Microsoft SQL продуктов и служб. Все средства и приложения, которые взаимодействуют с базой данных SQL сделать путем отправки команды T-SQL.
