# Редактор задачи &#171;Выполнение процесса&#187; (страница &#171;Общие&#187;)
  Воспользуйтесь страницей **Общие** диалогового окна **Редактор задачи "Выполнение процесса"** для ввода имени и описания задачи "Выполнение процесса".  
  
 Дополнительные сведения об этой задаче см. в разделе [Execute Process Task](../../integration-services/control-flow/execute-process-task.md).  
  
## Параметры  
 **Название**  
 Введите уникальное имя для задачи "Выполнение процесса". Это имя используется в качестве метки для значка задачи.  
  
> [!NOTE]  
>  Имена задач в пределах пакета должны быть уникальными.  
  
 **Description**  
 Введите описание задачи "Выполнение процесса".
