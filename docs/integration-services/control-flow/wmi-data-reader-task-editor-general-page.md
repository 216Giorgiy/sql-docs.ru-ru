# Редактор задачи &#171;Модуль чтения данных WMI&#187; (страница &#171;Общие&#187;)
  Страница **Общие** диалогового окна **Редактор задачи «Модуль чтения данных WMI»** позволяет дать имя и описание задаче «Модуль чтения данных WMI».  
  
 Дополнительные сведения об этой задаче см. в разделе [WMI Data Reader Task](../../integration-services/control-flow/wmi-data-reader-task.md). Дополнительные сведения о языке запросов WQL см. в разделе документации по инструментарию управления Windows [Запросы с использованием языка запросов WQL](http://go.microsoft.com/fwlink/?LinkId=79045) в библиотеке MSDN.  
  
## Параметры  
 **Название**  
 Задайте уникальное имя задаче «Модуль чтения данных WMI». Это имя используется в качестве метки для значка задачи.  
  
> [!NOTE]  
>  Имена задач в пределах пакета должны быть уникальными.  
  
 **Description**  
 Введите описание задачи «Модуль чтения данных WMI».
