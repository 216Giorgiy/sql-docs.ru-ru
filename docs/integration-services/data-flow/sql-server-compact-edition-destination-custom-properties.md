# Пользовательские свойства назначения &#171;SQL Server Compact Edition&#187;
  Назначение « [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Compact» имеет как пользовательские, так и общие для всех компонентов потока данных свойства.  
  
 В следующей таблице описаны пользовательские свойства назначения « [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Compact». Все свойства доступны для чтения и записи.  
  
|Имя свойства|Тип данных|Description|  
|-------------------|---------------|-----------------|  
|TableName|Строковые значения|Имя целевой таблицы в базе данных [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Compact.<br /><br /> Значение этого свойства можно задать с помощью выражения свойства.|  
  
 Ввод и входные столбцы назначения « [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Compact» не имеют пользовательских свойств.  
  
 Дополнительные сведения см. в разделе [Назначение "SQL Server Compact Edition"](../../integration-services/data-flow/sql-server-compact-edition-destination.md).
