# Редактор источника &#171;Необработанный файл&#187; (страница &#171;Диспетчер соединений&#187;)
  Источник «Необработанный файл» считывает необработанные данные из файла. Поскольку данные представлены в собственном формате источника, их преобразование не требуется. Также практически не требуется синтаксический анализ. Дополнительные сведения см. в статье [Raw File Source](../../integration-services/data-flow/raw-file-source.md).
