# Подготовка к обновлению с помощью помощника по обновлению
  Приготовьтесь к обновлению баз данных OLTP и внедрению новых возможностей в [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)], скачав и запустив помощник по миграции данных Microsoft Data Migration Assistant.  
  
## <a name="download-and-install-data-migration-assistant"></a>Загрузка и установка помощника по миграции данных  
 Скачайте и установите помощник по миграции данных [здесь](http://go.microsoft.com/fwlink/?LinkID=613421). Этот инструмент не входит в комплект на установочном носителе [!INCLUDE[ssSQL15](../../includes/sssql15-md.md)] .
