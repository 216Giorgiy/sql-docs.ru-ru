## <a name="181-sqlpackage"></a>sqlpackage 18.1

Дата выпуска:&nbsp;1 февраля 2019 г.  
Сборка:&nbsp;15.0.4316.1  
Предварительный выпуск.

### <a name="features"></a>Компоненты

| Компонент | Сведения |
| :------ | :------ |
| Добавлена поддержка параметров сортировки UTF8. | &nbsp; |
| Включить некластеризованный columnstore индексы в индексированном представлении. | &nbsp; |
| Переместить в .NET Core 2.2. | &nbsp; |
| Использование памяти резервного хранилища для сравнения схем в .NET Core. | &nbsp; |
| &nbsp; | &nbsp; |

### <a name="fixes"></a>Исправления

| Fix | Сведения |
| :-- | :------ |
| Исправление производительности для использования унаследованной кратности для обратного проектирования запросов. | &nbsp; |
| Исправлена проблема производительности сравнения схемы, при создании скрипта. | &nbsp; |
| Исправлена логика обнаружения схемы о смещении для игнорирования определенных сеансов расширенных событий (xevent). | &nbsp; |
| Упорядочение для таблицы graph предопределенной импорта. | &nbsp; |
| Исправлена Экспорт внешних таблиц с разрешениями на объекты. | &nbsp; |
| &nbsp; | &nbsp; |

### <a name="known-issues"></a>Известные проблемы

Этот выпуск включает предварительные кросс платформенные сборки sqlpackage, предназначенных для .NET Core 2.2. Sqlpackage можно запустить в macOS и Linux.

| Известная проблема | Сведения |
| :---------- | :------ |
| Участники сборки и развертывания не поддерживаются. | &nbsp; |
| Старые файлы DACPAC и bacpac, использующих сериализацию данных json, не поддерживаются. | &nbsp; |
| На который указывает ссылка .dacpacs (например master.dacpac) не может разрешить из-за проблем с системами файлов с учетом регистра. | Правильным решением будет использовать имя файла ссылки (например, MASTER. BACPAC-ФАЙЛ). |
| &nbsp; | &nbsp; |
