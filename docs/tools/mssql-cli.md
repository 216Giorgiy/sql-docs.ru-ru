# <a name="mssql-cli-command-line-query-tool-for-sql-server"></a>средство командной строки запроса MSSQL cli для SQL Server
[!INCLUDE[appliesto-ss-asdb-asdw-xxx-md](../includes/appliesto-ss-asdb-asdw-xxx-md.md)]

MSSQL cli — это интерактивное средство командной строки, для выполнения запросов к SQL Server, установите его на Windows, macOS или Linux.

## <a name="install-mssql-cli"></a>Установить mssql cli

Подробные инструкции по установке, в разделе [руководство по установке](https://github.com/dbcli/mssql-cli/blob/master/doc/installation_guide.md), или если вы знаете pip, установить, запустив следующую команду:

```$ pip install mssql-cli```
