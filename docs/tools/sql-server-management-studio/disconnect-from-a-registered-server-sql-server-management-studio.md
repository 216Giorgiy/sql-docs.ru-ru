# Отключение от зарегистрированного сервера (среда SQL Server Management Studio)
  В этом разделе описывается, как отключить обозреватель объектов от сервера в любое время в [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)]. Это не приведет к отключению других компонентов среды [!INCLUDE[ssManStudioFull](../../includes/ssmanstudiofull-md.md)] , например редактора SQL.  
  
##  <a name="SSMSProcedure"></a>  
  
#### Отключение от зарегистрированного сервера  
  
-   В обозревателе объектов либо щелкните правой кнопкой мыши сервер и нажмите кнопку **Отключиться**, либо нажмите кнопку **Отключиться** на панели инструментов.
