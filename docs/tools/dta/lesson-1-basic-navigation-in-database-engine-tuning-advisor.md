# <a name="lesson-1-basic-navigation-in-database-engine-tuning-advisor"></a>Урок 1. Основы перемещения в помощнике по настройке ядра СУБД
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]Помощник по настройке ядра СУБД предоставляет графический пользовательский интерфейс (GUI) на основе способ просмотра сеансов настройки и создания отчетов с рекомендациями по настройке. На этом занятии объясняется, как запустить данный инструмент и выполнить настройку его вида. После завершения занятия пользователь будет иметь представление о различных способах запуска этого инструмента, а также узнает, как настраивать его вид для оптимизации выполнения регулярных задач по настройке.  
  
Это занятие содержит следующие разделы:  
  
-   Запуск помощника по настройке ядра СУБД  
  
-   Настройка параметров и вида средств
