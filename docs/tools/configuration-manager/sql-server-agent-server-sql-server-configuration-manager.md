# <a name="sql-server-agent-ltservergt-sql-server-configuration-manager"></a>Агент SQL Server &lt;сервера&gt; (диспетчер конфигурации SQL Server)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]Это служба, которая поддерживает [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] агента. Чтобы просмотреть свойства службы, щелкните правой кнопкой мыши в области сведений, а затем выберите пункт **Свойства**.  
  
 [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]Агент — [!INCLUDE[msCoName](../../includes/msconame-md.md)] обращение к службе Windows, выполняющая запланированные административные задачи *задания*. Дополнительные сведения об агенте [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] см. в разделе «Об агенте SQL Server» электронной документации по [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] .
