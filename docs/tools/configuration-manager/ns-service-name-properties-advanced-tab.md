# Свойства NS$&lt;имя_службы&gt; (вкладка &quot;Дополнительно&quot;)
  [!INCLUDE[ssNS](../../includes/ssns-md.md)] реализованы как служба, имеющая имя в формате **NS$***\<заданное_пользователем_имя>*. Если определены пользовательские свойства, они отображаются на этой вкладке вместе со значениями. На этой вкладке может не быть никаких свойств.
