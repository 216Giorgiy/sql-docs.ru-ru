# <a name="nsltservice-namegt-properties-advanced-tab"></a>NS$&lt;имя службы&gt; свойства (вкладка «Дополнительно»)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)][!INCLUDE[ssNS](../../includes/ssns-md.md)] реализован как служба с именем в формате **NS$***< заданное_пользователем_имя >*. Если определены пользовательские свойства, они отображаются на этой вкладке вместе со значениями. На этой вкладке может не быть никаких свойств.
