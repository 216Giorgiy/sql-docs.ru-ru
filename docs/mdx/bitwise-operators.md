# <a name="bitwise-operators"></a>Битовые операторы
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx_md](../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  Логические операторы вычисляют значения и возвращают логическое значение. В многомерных выражениях логические операторы не выполняют побитовых операций.  
  
 В многомерных выражениях поддерживаются логические операторы, перечисленные в следующей таблице.  
  
|Оператор|Description|  
|--------------|-----------------|  
|[И](../mdx/and-mdx.md)|Выполняет логическое умножение двух числовых выражений.|  
|[—](../mdx/is-mdx.md)|Выполняет логическое сравнение двух выражений объектов.|  
|[НЕ](../mdx/not-mdx.md)|Выполняет операцию логического отрицания над числовым выражением.|  
|[или](../mdx/or-mdx.md)|Выполняет логическое сложение двух числовых выражений.|  
|[ИСКЛЮЧАЮЩЕЕ ИЛИ](../mdx/xor-mdx.md)|Выполняет логическое исключение над двумя числовыми выражениями.|
