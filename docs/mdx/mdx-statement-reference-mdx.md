# <a name="mdx-statement-reference-mdx"></a>Справка по инструкциям многомерных выражений (многомерные выражения)


  Инструкции многомерных выражений делятся на три группы (см. следующую таблицу).  
  
## <a name="in-this-section"></a>в этом разделе  
  
|Раздел|Описание|  
|-----------|-----------------|  
|[Инструкции сценариев многомерных Выражений &#40;многомерных Выражений&#41;](../mdx/mdx-scripting-statements-mdx.md)|Многомерные инструкции скриптов для управления областью, контекстом и потоками в скриптах многомерных выражений.|  
|[Инструкции определения данных многомерных Выражений &#40;многомерных Выражений&#41;](../mdx/mdx-data-definition-statements-mdx.md)|Многомерные инструкции языка определения данных (DDL) для создания, сбрасывания и работы с многомерными объектами.|  
|[Инструкции языка манипулирования данными &#40;многомерных Выражений&#41;](../mdx/mdx-data-manipulation-statements-mdx.md)|Многомерные инструкции языка манипулирования данными (DML) для извлечения и манипулирования данными многомерных объектов.|
