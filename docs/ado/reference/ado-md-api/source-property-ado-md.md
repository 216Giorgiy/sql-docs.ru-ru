# <a name="source-property-ado-md"></a>Свойство Source (ADO MD)
Указывает источник данных в [ячеек](../../../ado/reference/ado-md-api/cellset-object-ado-md.md).  
  
## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения  
 Возвращает или задает **Variant**, и доступно для чтения/записи для закрыто [ячеек](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) объектов и только для чтения для открытия **ячеек** объектов. **Variant** должен содержать допустимое **строка**, например, запрос многомерных Выражений.
