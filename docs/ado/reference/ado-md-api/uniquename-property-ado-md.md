# <a name="uniquename-property-ado-md"></a>Свойство UniqueName (ADO MD)
Указывает имя однозначным для текущего объекта.  
  
## <a name="return-values"></a>Возвращаемые значения  
 Возвращает **строка** и доступно только для чтения.
