# <a name="levelname-property-ado-md"></a>Свойство LevelName (ADO MD)
Указывает имя уровня [член](../../../ado/reference/ado-md-api/member-object-ado-md.md).  
  
## <a name="return-values"></a>Возвращаемые значения  
 Возвращает **строка** и доступно только для чтения.  
  
## <a name="remarks"></a>Примечания  
 Используйте **LevelName** свойства, чтобы получить имя уровня, к которой принадлежит элемент. Это соответствует [имя](../../../ado/reference/ado-md-api/name-property-ado-md.md) свойство [уровень](../../../ado/reference/ado-md-api/level-object-ado-md.md) объекта.
