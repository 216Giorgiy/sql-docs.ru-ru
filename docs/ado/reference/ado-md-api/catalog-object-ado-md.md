# <a name="catalog-object-ado-md"></a>Объект каталога (ADO MD)
Сведения многомерной схемой (т. е кубы и базового измерения, иерархии, уровни и элементы) поставщикам многомерных данных (MDP).  
  
## <a name="remarks"></a>Замечания  
 С коллекциями и свойствами **каталога** объекта, можно сделать следующее:  
  
-   Открыть каталог, задав [ActiveConnection](../../../ado/reference/ado-md-api/activeconnection-property-ado-md.md) свойства стандартных ADO [подключения](../../../ado/reference/ado-api/connection-object-ado.md) объекта или допустимую строку соединения.  
  
-   Определить **каталога** с [имя](../../../ado/reference/ado-md-api/name-property-ado-md.md) свойства.  
  
-   Итерации кубов в каталог с помощью [CubeDefs](../../../ado/reference/ado-md-api/cubedefs-collection-ado-md.md) коллекции.
