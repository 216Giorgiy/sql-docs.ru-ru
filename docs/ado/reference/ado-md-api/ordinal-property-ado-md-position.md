# <a name="ordinal-property-ado-md-position"></a>Порядковый номер свойства (позиция ADO MD)
Уникально идентифицирует [позиции](../../../ado/reference/ado-md-api/position-object-ado-md.md) вдоль оси.  
  
## <a name="return-values"></a>Возвращаемые значения  
 Возвращает **длинные** целое число со знаком и доступно только для чтения.  
  
## <a name="remarks"></a>Замечания  
 **Порядковый номер** свойство [позиции](../../../ado/reference/ado-md-api/position-object-ado-md.md) соответствует индекс **позиции** в [позиций](../../../ado/reference/ado-md-api/positions-collection-ado-md.md) коллекции.  
  
 Ячейки можно быстро получить с помощью **порядковый номер** из **позиции** по каждой оси с [элемент](../../../ado/reference/ado-md-api/item-property-ado-md-cellset.md) свойство [ячеек](../../../ado/reference/ado-md-api/cellset-object-ado-md.md) объекта .
