# <a name="attributes-property-adox"></a>Свойства атрибутов (ADOX)
Описывает характеристики столбца.  
  
## <a name="settings-and-return-values"></a>Параметры и возвращаемые значения  
 Возвращает или задает **длинные** значение. Это значение определяет характеристики таблицы, представленного [столбца](../../../ado/reference/adox-api/column-object-adox.md) объекта. Значение может представлять собой сочетание [ColumnAttributesEnum](../../../ado/reference/adox-api/columnattributesenum.md) константы. Значение по умолчанию равно нулю (**0**), который не является ни **adColFixed** , ни **adColNullable**.
