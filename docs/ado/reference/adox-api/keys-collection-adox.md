# <a name="keys-collection-adox"></a>Коллекция ключей (ADOX)
Содержит все [ключ](../../../ado/reference/adox-api/key-object-adox.md) объектов [таблицы](../../../ado/reference/adox-api/table-object-adox.md).  
  
## <a name="remarks"></a>Примечания  
 [Append](../../../ado/reference/adox-api/append-method-adox-keys.md) метод [ключи коллекции](../../../ado/reference/adox-api/keys-collection-adox.md) уникален для ADOX. Возможные действия:  
  
-   Добавьте новый раздел в коллекцию с [Append](../../../ado/reference/adox-api/append-method-adox-keys.md) метод.  
  
 Остальные свойства и методы являются стандартными коллекциям ADO. Возможные действия:  
  
-   Доступ к ключу в коллекции с [элемент](../../../ado/reference/ado-api/item-property-ado.md) свойство.  
  
-   Возвращает число ключей, содержащихся в коллекции с [число](../../../ado/reference/ado-api/count-property-ado.md) свойство.  
  
-   Удаления ключа из коллекции с помощью [удалить](../../../ado/reference/adox-api/delete-method-adox-collections.md) метод.  
  
-   Обновление объектов в коллекции в соответствии с текущей базы данных в схеме с [обновление](../../../ado/reference/ado-api/refresh-method-ado.md) метод.  
  
 Этот раздел содержит следующий раздел.
