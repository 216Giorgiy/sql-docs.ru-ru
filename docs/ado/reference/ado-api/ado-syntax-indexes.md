# <a name="ado-syntax-indexes"></a>Синтаксис индексов ADO
Синтаксис вызова ADO методы и свойства отличается в зависимости от среды разработки. Остальная часть справочнике по языку ADO использует Microsoft Visual Basic язык программирования для демонстрации синтаксиса метода или свойства ADO. Тем не менее см. в разделах Подробные примеры синтаксиса в зависимости от языка программирования и методологии:  
  
-   [ADO для Visual C++ синтаксис индекса COM](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-for-com.md) охватывает ADO свойства и методы без использования **#import** директивы компилятора с Microsoft Visual C++.  
  
-   [ADO для индекса в Visual C++ синтаксис с #import](../../../ado/reference/ado-api/ado-for-visual-c-syntax-index-with-sharpimport.md) рассматриваются ADO свойств и методов, при использовании **#import** директивы компилятора с Microsoft Visual C++.
