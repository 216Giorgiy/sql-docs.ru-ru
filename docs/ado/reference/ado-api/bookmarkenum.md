# <a name="bookmarkenum"></a>BookmarkEnum
Указывает которого начинается операция закладки.  
  
|Константа|Значение|Описание|  
|--------------|-----------|-----------------|  
|**adBookmarkCurrent**|0|Начинает текущей записи.|  
|**adBookmarkFirst**|1|Начинается с первой записи.|  
|**adBookmarkLast**|2|Начинает последнюю запись.|
