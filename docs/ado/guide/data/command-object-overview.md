# <a name="command-object-overview"></a>Обзор объекта Command
С **команда** объекта, можно сделать следующее:  
  
-   Задать исполняемый текст команды (например, инструкции SQL или хранимой процедуры) с помощью **CommandText** свойство.  
  
-   Определить запросы с параметрами и аргументами хранимой процедуры с помощью **параметр** объектов и **параметры** коллекции.  
  
-   Выполнять команды и возвращать **записей** объекта, если это применимо, с помощью **Execute** метод.  
  
-   Укажите тип команды с помощью **CommandType** свойство перед выполнением для оптимизации производительности.  
  
-   Укажите подробные сведения о тексте команды с помощью **диалект** свойство **команда** объекта.  
  
-   Контролировать, сохраняет ли поставщик подготовленную (скомпилированную) версию перед выполнением команды с помощью **Готово** свойство.  
  
-   Задать число секунд, поставщик будет ожидать выполнения с помощью команды **CommandTimeout** свойство.  
  
-   Связать открытое соединение с **команда** , задавая его **ActiveConnection** свойство.  
  
-   Задать **имя** свойство для идентификации **команда** объекту в качестве метода для соответствующей **подключения** объекта.  
  
-   Передайте **команда** объект **источника** свойство **записей** для получения данных.  
  
-   Передайте **поток** объект, содержащий команды (например, XML-команды) для поставщика, который поддерживает его.
