# <a name="internet-publishing-scenario"></a>Сценарий публикации в Интернете
Данный пример кода демонстрирует, как использовать ADO с поставщиком данных Microsoft OLE DB для публикаций в Интернете. В этом сценарии вы создадите приложение Visual Basic, который использует **записей**, **запись**, и **поток** опубликованных объектов для отображения содержимого ресурсы с Публикации в Интернете поставщика.
