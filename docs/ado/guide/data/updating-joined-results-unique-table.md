# <a name="updating-joined-results-unique-table"></a>Обновление объединить результаты: Уникальной таблицы
ADO позволяет точно управления изменения определенной базовой таблицей в наборе записей, созданное с помощью операции СОЕДИНЕНИЯ на нескольких базовых таблицах, с помощью динамических свойств уникальной таблицы. Дополнительные сведения об использовании уникальной таблицы ссылаться программиста ADO справочные разделы, посвященные уникальной таблицы и установки обновления выполнить повторную синхронизацию динамических свойств.
