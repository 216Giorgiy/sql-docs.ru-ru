# <a name="serversettings-class"></a>Класс ServerSettings
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  [Класс ServerSettings](../../../relational-databases/wmi-provider-configuration-classes/serversettings-class/serversettings-class.md) представляет настройки сервера на экземпляр [!INCLUDE[msCoName](../../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
 [Класс ServerSettings](../../../relational-databases/wmi-provider-configuration-classes/serversettings-class/serversettings-class.md) позволяет выполнять следующие задачи:  
  
-   получать доступ к общим флагам настройки экземпляра [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)];  
  
-   задавать значения по умолчанию для экземпляра [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)];  
  
-   задавать сертификат безопасности для экземпляра [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].
