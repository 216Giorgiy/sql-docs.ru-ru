# <a name="flagname-property-serversettingsgeneralflag-class"></a>Свойство FlagName (класс ServerSettingsGeneralFlag)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Возвращает имя флага.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object.FlagName [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *объект*  
 Объект [класса ServerSettingsGeneralFlag](../../../relational-databases/wmi-provider-configuration-classes/serversettingsgeneralflag-class/serversettingsgeneralflag-class.md) , представляющий универсальный флаг для параметров сервера.  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Строковое значение, указывающее имя флага.
