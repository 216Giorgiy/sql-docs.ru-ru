# <a name="propertystrvalue-property-sqlserviceadvancedproperty-class"></a>Свойство PropertyStrValue (класс SqlServiceAdvancedProperty)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]
  Возвращает строковое значение дополнительного свойства.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object.PropertyStrValue [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *объект*  
 Объект [класс SqlServiceAdvancedProperty](../../../relational-databases/wmi-provider-configuration-classes/sqlserviceadvancedproperty-class/sqlserviceadvancedproperty-class.md) объект, представляющий дополнительное свойство.  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Строка, указывающая значение дополнительного свойства.
