# <a name="size-property-clientnetlibinfo-class"></a>Свойство Size (класс ClientNetLibInfo)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Возвращает размер клиента сетевой библиотеки.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object.Size [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *объект*  
 Объект [класса ClientNetLibInfo](../../../relational-databases/wmi-provider-configuration-classes/clientnetlibinfo-class/clientnetlibinfo-class.md) , представляющий сведения о сетевой библиотеке клиента.  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Значение uint32, определяющее размер файла сетевой библиотеки клиента в килобайтах.
