# <a name="protocoldisplayname-property-clientnetlibinfo-class"></a>Свойство ProtocolDisplayName (класс ClientNetLibInfo)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Возвращает или задает имя, отображаемое имя протокола.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object.ProtocolDisplayName [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *объект*  
 Объект [класса ClientNetLibInfo](../../../relational-databases/wmi-provider-configuration-classes/clientnetlibinfo-class/clientnetlibinfo-class.md) , представляющий сведения о сетевой библиотеке клиента.  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Строковое значение, определяющее отображаемое имя протокола.
