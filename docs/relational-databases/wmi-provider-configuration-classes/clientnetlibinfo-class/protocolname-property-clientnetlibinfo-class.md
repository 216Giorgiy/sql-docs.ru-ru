# <a name="protocolname-property-clientnetlibinfo-class"></a>Свойство ProtocolName (класс ClientNetLibInfo)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]Возвращает имя протокола, который связан с сетевой библиотеки клиента.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
object.ProtocolName [= value]  
```  
  
## <a name="parts"></a>Компоненты  
 *объект*  
 Объект [класса ClientNetLibInfo](../../../relational-databases/wmi-provider-configuration-classes/clientnetlibinfo-class/clientnetlibinfo-class.md) , представляющий сведения о сетевой библиотеке клиента.  
  
## <a name="property-valuereturn-value"></a>Значение свойства/возвращаемое значение  
 Строковое значение, определяющее имя протокола, связанного с сетевой библиотекой клиента.
