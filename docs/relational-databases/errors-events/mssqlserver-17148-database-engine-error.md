## <a name="details"></a>Сведения  
  
|||  
|-|-|  
|Название продукта|SQL Server|  
|Идентификатор события|17148|  
|Источник события|MSSQLSERVER|  
|Компонент|SQLEngine|  
|Символическое имя|STARTUP_SERVERUNINSTALL|
