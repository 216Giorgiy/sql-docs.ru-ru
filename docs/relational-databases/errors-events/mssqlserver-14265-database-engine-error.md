# <a name="mssqlserver14265"></a>MSSQLSERVER_14265
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
  
## <a name="details"></a>Сведения  
  
|||  
|-|-|  
|Название продукта|SQL Server|  
|Идентификатор события|14265|  
|Источник события|MSSQLSERVER|  
|Компонент|SQLEngine|  
|Символическое имя|SQLErrorNum14265|  
|Текст сообщения|Непредвиденное завершение службы MSSQLServer. Проверьте журнал ошибок SQL Server и журналы событий приложений и системы Windows, чтобы определить возможные причины.|  
  
## <a name="explanation"></a>Объяснение  
Непредвиденное завершение службы SQL Server.  
  
## <a name="user-action"></a>Действие пользователя  
Просмотрите журналы ошибок, чтобы понять, чем вызвано завершение работы службы SQL Server.
