# Категория событий Cursors
  Категория событий **Cursors** содержит классы событий, используемых для контроля действий курсоров.  
  
## В этом разделе  
  
|Раздел|Описание|  
|-----------|-----------------|  
|[Класс событий CursorClose](../../relational-databases/event-classes/cursorclose-event-class.md)|Описывает события закрытия курсора, возникающие в API-курсорах.|  
|[Класс событий CursorExecute](../../relational-databases/event-classes/cursorexecute-event-class.md)|Описывает события выполнения курсоров, возникающие в API-курсорах.|  
|[Класс событий CursorImplicitConversion](../../relational-databases/event-classes/cursorimplicitconversion-event-class.md)|Описывает события неявного преобразования курсоров, возникающие в API-курсорах или курсорах [!INCLUDE[tsql](../../includes/tsql-md.md)].|  
|[Класс событий CursorOpen](../../relational-databases/event-classes/cursoropen-event-class.md)|Описывает события открытия курсоров, возникающие в API-курсорах.|  
|[Класс событий CursorPrepare](../../relational-databases/event-classes/cursorprepare-event-class.md)|Описывает события подготовки курсоров, возникающие в API-курсорах.|  
|[Класс событий CursorRecompile](../../relational-databases/event-classes/cursorrecompile-event-class.md)|Описывает события перекомпиляции курсоров, возникающие в API-курсорах.|  
|[Класс событий CursorUnprepare](../../relational-databases/event-classes/cursorunprepare-event-class.md)|Описывает аннулирующие курсор события, возникающие в API-курсорах.|
