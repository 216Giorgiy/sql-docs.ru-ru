# <a name="ibcpsession2-ole-db"></a>IBCPSession2 (OLE DB)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
[!INCLUDE[SNAC_Deprecated](../../includes/snac-deprecated.md)]

  IBCPSession2, интерфейс является расширением IBCPSession, предоставляющий функцию-член, является альтернативой для вызова IBCPSession::BCPColFmt для каждого столбца.  IBCPSession2 наследует от интерфейса IBCPSession и добавлением одного нового метода: [IBCPSession2::BCPSetBulkMode](../../relational-databases/native-client-ole-db-interfaces/ibcpsession2-bcpsetbulkmode.md).
