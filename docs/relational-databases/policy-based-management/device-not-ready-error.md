# <a name="device-not-ready-error"></a>Ошибка отсутствия готовности устройства
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Это правило проверяет журнал системных событий на наличие ошибки с идентификатором EventId 15. Ошибка вызывается неисправностями хост-адаптера SCSI или связанными проблемами.
