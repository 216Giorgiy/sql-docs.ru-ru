# <a name="device-driver-control-error"></a>Ошибка управления драйвером устройства
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Это правило проверяет журнал системных событий на наличие ошибки с идентификатором EventId 11. Ошибка может быть вызвана повреждением драйвера устройства, проблемой оборудования, неисправностью кабельных подключений или соединения.
