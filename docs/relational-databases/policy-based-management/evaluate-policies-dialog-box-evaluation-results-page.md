# Диалоговое окно &#171;Выполнение политик&#187;, страница &#171;Результаты выполнения&#187;
  С помощью этого диалогового окна можно просмотреть результаты оценки политики и применить политики управления на основе политик к набору целей, которые не соответствуют этим политикам.  
  
## Параметры  
 **Результаты: политики**  
 Имя оцениваемой политики. Поле доступно только для чтения.  
  
 **Результаты: сообщение**  
 Ссылка на сведения, если произошла ошибка.  
  
 **Данные целевого объекта: сервер**  
 Экземпляр [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)], по отношению к которому оцениваются политики.  
  
 **Данные целевого объекта: цель**  
 Целевой объект, по отношению к которому оцениваются политики.  
  
 **Данные целевого объекта: сведения**  
 Ссылка на диалоговое окно **Подробный просмотр результатов**, в котором отображаются подробные сведения об оценке политики.  
  
 **Данные целевого объекта: сообщения**  
 Ссылка на сведения, если произошла ошибка.
