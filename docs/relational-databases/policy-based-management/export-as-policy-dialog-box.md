# <a name="export-as-policy-dialog-box"></a>Диалоговое окно «Экспортировать как политику»
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Это диалоговое окно используется для сохранения набора свойств аспекта управления на основе политик как политики. Политики можно сохранить в XML-файл или напрямую в текущий экземпляр компонента [!INCLUDE[ssDE](../../includes/ssde-md.md)].  
  
## <a name="options"></a>Параметры  
 **Имя определения политики**  
 Введите имя политики.  
  
 **Имя условия**  
 Введите имя условия.  
  
 **На локальный сервер**  
 Выберите, чтобы сохранить политику на экземпляре локального сервера.  
  
 **В файл**  
 Чтобы сохранить политику в XML-файл, следует ввести путь и имя файла либо нажать кнопку обзора (**...**).
