# <a name="storage-system-input-output-time-out"></a>Время ожидания ввода-вывода для системы хранения
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
