# <a name="results-detailed-view-dialog-box"></a>Диалоговое окно «Подробный просмотр результатов»
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] В этом диалоговом окне приводятся результаты оценки политики после ее запуска с помощью диалогового окна **Выполнение политик** и нажатия кнопки **Оценить**. Это диалоговое окно доступно только для чтения; оно помогает понять, в какой части выражения свойства может содержаться ошибка.  
  
## <a name="options"></a>Параметры  
 **AndOr**  
 Если присутствует более одного выражения свойства, показывает, являются ли выражения свойств совокупными или альтернативными.  
  
 **Результат**  
 Значок, обозначающий успех или неудачу выражения свойства.  
  
 **Поле**  
 Свойство моделируемого аспекта.  
  
 **Оператор**  
 Оператор для выражения, например: **=** или **Like**.  
  
 **Ожидаемое значение**  
 Значение поля, при котором выражение свойства будет считаться успешным.  
  
 **Фактическое значение**  
 Значение поля, обнаруженное политикой.  
  
 **Описание политики**  
 Описание политики.  
  
 **Дополнительная справка**  
 Перейдите по гиперссылке, чтобы открыть веб-страницу, связанную с данной политикой. Эта дополнительная гиперссылка справки настраивается при создании политики, она может быть пустой или недоступной.
