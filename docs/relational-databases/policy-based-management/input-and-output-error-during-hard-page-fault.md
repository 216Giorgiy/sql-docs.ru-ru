# <a name="input-and-output-error-during-hard-page-fault"></a>Input and Output Error During Hard Page Fault
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
