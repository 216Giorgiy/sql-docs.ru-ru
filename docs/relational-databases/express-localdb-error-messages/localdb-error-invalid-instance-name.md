# <a name="localdberrorinvalidinstancename"></a>LOCALDB_ERROR_INVALID_INSTANCE_NAME
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)]
    
## <a name="details"></a>Сведения  
  
|||  
|-|-|  
|Название продукта|SQL Server|  
|Идентификатор события|283|  
|Источник события|Среда выполнения локальной базы данных SQL Server 12.0|  
|Компонент|API среды выполнения локальной базы данных|  
|Текст сообщения|Имя экземпляра содержит недопустимые символы или слишком длинное. Обратитесь к документации по API.|  
  
## <a name="explanation"></a>Объяснение  
 Имя экземпляра содержит недопустимые символы или слишком длинное.  
  
## <a name="user-action"></a>Действие пользователя  
 Обратитесь к документации по API среды выполнения локальной базы данных по поводу допустимых имен экземпляров среды выполнения локальной базы данных.  Дополнительные сведения см. в разделе [SQL Server Express LocalDB Справочник по API экземпляра](../../relational-databases/express-localdb-instance-apis/sql-server-express-localdb-reference-instance-apis.md).
