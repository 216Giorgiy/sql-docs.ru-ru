# <a name="msdistributor-transact-sql"></a>MSdistributor (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  **MSdistributor** таблица содержит свойства распространителя. Эта таблица хранится в **msdb** базы данных.  
  
|Имя столбца|Тип данных|Описание|  
|-----------------|---------------|-----------------|  
|**Свойство**|**sysname**|Имя свойства|  
|**value**|**nvarchar(3000)**|Значение свойства|
