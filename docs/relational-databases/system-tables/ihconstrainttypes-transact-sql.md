# <a name="ihconstrainttypes-transact-sql"></a>IHconstrainttypes (Transact-SQL)
[!INCLUDE[tsql-appliesto-ss2008-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2008-xxxx-xxxx-xxx-md.md)]

  **IHconstrainttypes** системная таблица содержит по одной строке для каждого типа ограничений SQL Server и поддерживается для издателей, отличных от издателей SQL Server. Эта таблица хранится в базе данных распространителя.  
  
|Имя столбца|Тип данных|Description|  
|-----------------|---------------|-----------------|  
|**type**|**nvarchar(255)**|Имя поддерживаемого типа ограничений, не являющегося типом ограничений SQL Server.|
