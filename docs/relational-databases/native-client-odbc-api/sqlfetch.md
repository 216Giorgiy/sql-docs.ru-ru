# <a name="sqlfetch"></a>SQLFetch
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
[!INCLUDE[SNAC_Deprecated](../../includes/snac-deprecated.md)]

  SQLFetch поддерживает улучшенные функции даты и времени.  
  
 Сведения о том, как значения результирующих столбцов типов даты времени преобразуются в разделе [преобразования из SQL в C](../../relational-databases/native-client-odbc-date-time/datetime-data-type-conversions-from-sql-to-c.md).
