# <a name="table-valued-parameters-sql-server-native-client"></a>Возвращающие табличное значение параметры (собственный клиент SQL Server)
[!INCLUDE[appliesto-ss-asdb-asdw-pdw-md](../../../includes/appliesto-ss-asdb-asdw-pdw-md.md)]
[!INCLUDE[SNAC_Deprecated](../../../includes/snac-deprecated.md)]

  Возвращающие табличное значение параметры появились в [!INCLUDE[ssKatmai](../../../includes/sskatmai-md.md)], и обеспечивают эффективный способ передачи нескольких строк данных на сервер. Возвращающие табличное значение параметры предоставляют функциональные возможности, схожие с массивами параметров, но предлагают большую гибкость и более тесную интеграцию с [!INCLUDE[tsql](../../../includes/tsql-md.md)], и могут значительно улучшить производительность. Возвращающие табличные значения параметров также могут участвовать в операции с множествами, а массивы параметров нельзя.  
  
 Сведения о возвращающих табличное значение параметрах и ODBC см. в разделе [табличное значение параметры &#40; ODBC &#41;](../../../relational-databases/native-client-odbc-table-valued-parameters/table-valued-parameters-odbc.md).  
  
 Сведения о возвращающих табличные значения параметров и OLE DB см. в разделе [табличное значение параметры &#40; OLE DB &#41;](../../../relational-databases/native-client-ole-db-table-valued-parameters/table-valued-parameters-ole-db.md).
