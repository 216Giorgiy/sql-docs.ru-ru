# Определение фильтров
  Диалоговое окно **Определение фильтров** позволяет определять фильтры, применяемые к конфликтам данных для просмотра подмножества этих конфликтов в данной сетке. Чтобы определить фильтр, выберите оператор из **оператор** раскрывающегося списка и затем введите значение. Например, чтобы просмотреть только те конфликты, в которых проигравшей стороной является сервер **ReplTest1**, выберите **равно** из **оператор** раскрывающемся списке и введите **ReplTest1** в первом **значение** столбца.  
  
## Параметры  
 **Оператор**  
 Выберите оператор для фильтра, например **меньше или равно**.  
  
 **Значение**  
 Введите значение для фильтра. Для большинства операторов нужно только значение в первом столбце **Значение** , но для операторов **Между** и **Вне** необходимы значения в обоих столбцах **Значение** .  
  
 **Clear**  
 Нажмите эту кнопку для сброса всех ранее определенных фильтров.
