# разрешить для базы данных применение репликации (среда SQL Server Management Studio)
  Применение репликации для базы данных неявно разрешается, когда член предопределенной роли сервера **sysadmin** создает публикацию с помощью мастера создания публикаций. Является членом **sysadmin** фиксированной серверной роли также можно включить для репликации базы данных явным образом, чтобы член **db_owner** фиксированной роли базы данных можно создать одну или несколько публикаций в этой базе данных. Чтобы явно включить базу данных, используйте **баз данных публикации** Страница **Свойства издателя — \< издатель>** диалоговое окно. Дополнительные сведения о доступе к этому диалоговому окну см. в разделе [Create a Publication](../../relational-databases/replication/publish/create-a-publication.md).  
  
### Разрешение применения репликации для базы данных  
  
1.  На **баз данных публикации** Страница **Свойства издателя — \< издатель>** установите флажок **транзакций** или **слияния** флажок для каждой базы данных, нужно реплицировать. Установите флажок **Транзакционная** , чтобы разрешить для базы данных репликацию моментальных снимков.
