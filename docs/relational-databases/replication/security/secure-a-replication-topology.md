# Организация безопасности топологии репликации
  Учетные записи безопасности, имена входа и пароли для репликации предоставляются после настройки репликации и создания публикаций и подписок. Эти параметры безопасности можно изменить позже с помощью [!INCLUDE[ssManStudioFull](../../../includes/ssmanstudiofull-md.md)], хранимых процедур и объекты управления репликацией (RMO). В следующих разделах описываются способы изменения параметров безопасности репликации.
