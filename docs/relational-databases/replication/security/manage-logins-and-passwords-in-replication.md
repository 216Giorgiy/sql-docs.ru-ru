# Управление именами входа и паролями в репликации
  При настройке репликации укажите имена входа и пароли для агентов репликации. После настройки репликации можно изменить имена входа и пароли. Дополнительные сведения см. в статье [View and Modify Replication Security Settings](../../../relational-databases/replication/security/view-and-modify-replication-security-settings.md). Если изменить пароль для учетной записи, используемой агентом репликации, выполните [sp_changereplicationserverpasswords & #40; Transact-SQL & #41;](../../../relational-databases/system-stored-procedures/sp-changereplicationserverpasswords-transact-sql.md).
