# Безопасность агента &lt;AgentName&gt;
   **\< ИмяАгента> Безопасность агента** страница позволяет указать учетные записи, под которой агент распространителя (для репликации транзакций или моментальных снимков) или агент слияния (для репликации слиянием), запуск и подключаться к компьютерам в топологии репликации. Сведения о разрешениях, необходимых для агентов и рекомендации по безопасности репликации см. в разделе [модель безопасности агента репликации](../../relational-databases/replication/security/replication-agent-security-model.md) и [рекомендации по обеспечению безопасности репликации](../../relational-databases/replication/security/replication-security-best-practices.md).  
  
## Параметры  
 Нажмите кнопку свойств (**...**) в строке для каждого подписчика получить доступ к **Безопасность агента распространителя** или **Безопасность агента слияния** диалоговое окно. Щелкните **справки** в диалоговом окне, которое запускается Дополнительные сведения о разрешениях, необходимых для учетных записей, используемых агентами.  
  
 После ввода параметров в одном из диалоговых окон в сетке будут отображены сведения о соединении для подписчика.  
  
 **Агент для подписчика**  
 Имя каждого подписчика.  
  
 **Соединение с распространителем**  
 Отображается для репликации транзакций и репликации моментальных снимков. Контекст, в котором устанавливается соединение с распространителем. Локальные соединения всегда создаются с помощью контекста учетной записи [!INCLUDE[msCoName](../../includes/msconame-md.md)] Windows, под которой работает агент.  
  
-   Для принудительных подписок локальным соединением является соединение с распространителем, поэтому это поле будет отображаться всегда: **Impersonate "\< домена>\\< входа\>"** или **Олицетворение ' \< компьютер>\\< входа\>"** для принудительных подписок.  
  
-   Для подписок по запросу соединения можно также в контексте [!INCLUDE[msCoName](../../includes/msconame-md.md)] [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] входа. В поле отображается одно из следующих: **использовать имя входа ' \< входа>"**, **Impersonate" \< домена>\\< входа\>"** или **Impersonate" \< компьютер>\\< входа\>"**. [!INCLUDE[msCoName](../../includes/msconame-md.md)] рекомендует выполнять соединение в контексте учетной записи Windows.  
  
 **Соединение с издателем и распространителем**  
 Отображается для репликации слиянием. Контекст, под которым созданы соединения с издателем и распространителем. Локальные соединения всегда создаются с помощью контекста учетной записи Windows, под которой работает агент.  
  
-   Для принудительных подписок локальным соединением является соединение с издателем и распространителем, поэтому это поле будет отображаться всегда: **Impersonate "\< домена>\\< входа\>"** или **Олицетворение ' \< компьютер>\\< входа\>"** для принудительных подписок.  
  
-   Для подписок по запросу соединения также могут быть созданы в контексте имени входа [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. В поле отображается одно из следующих: **использовать имя входа ' \< входа>"**, **Impersonate" \< домена>\\< входа\>"** или **Impersonate" \< компьютер>\\< входа\>"**. [!INCLUDE[msCoName](../../includes/msconame-md.md)] рекомендует выполнять соединение в контексте учетной записи Windows.  
  
 **Соединение с подписчиком**  
 Контекст, в котором устанавливается соединение с подписчиком. Локальные соединения всегда создаются с помощью контекста учетной записи Windows, под которой работает агент.  
  
-   Для подписок по запросу локальным соединением является соединение с подписчиком, поэтому это поле будет отображаться всегда: **Impersonate "\< домена>\\< входа\>"** или **Олицетворение ' \< компьютер>\\< входа\>"** для принудительных подписок.  
  
-   Для принудительных подписок соединения также могут быть созданы в контексте имени входа [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)]. В поле отображается одно из следующих: **использовать имя входа ' \< входа>"**, **Impersonate" \< домена>\\< входа\>"** или **Impersonate" \< компьютер>\\< входа\>"**. [!INCLUDE[msCoName](../../includes/msconame-md.md)] рекомендует выполнять соединение в контексте учетной записи Windows.
