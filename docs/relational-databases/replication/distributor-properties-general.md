# Диалоговое окно &#171;Свойства распространителя&#187;, страница &#171;Общие&#187;
  На странице **Общие** диалогового окна **Свойства распространителя** можно добавлять и удалять базы данных распространителя, а также менять их свойства.  
  
 В базе данных распространителя хранятся метаданные и данные журнала для всех типов репликации, а также транзакции для репликации транзакций. В большинстве случаев достаточно одной базы данных распространителя. Но если несколько издателей используют один распространитель, рассмотрите вариант создания отдельной базы данных распространителя для каждого издателя. Это гарантирует то, что данные, проходящие через каждую базу данных распространителя, будут различаться.  
  
## Параметры  
 **Базы данных**  
  **Баз данных** сетки свойств показаны свойства имя и параметры хранения баз данных распространителя на распространителе. **Срок хранения транзакций** Длина время транзакции хранятся для репликации транзакций (срок хранения транзакций называется сроком хранения распространителя). **Срок хранения журнала** — это период времени, в течение которого метаданные журнала хранятся для всех типов репликации. Дополнительные сведения о хранении распространителя см. в разделе [сроке](../../relational-databases/replication/subscription-expiration-and-deactivation.md).  
  
 Нажмите кнопку свойств (**...**) в **базы данных** сетки свойств для запуска **Свойства базы данных распространителя** диалоговое окно.  
  
 **Создать**  
 Нажмите для создания новой базы данных распространителя.  
  
 **Delete**  
 Выбор существующей базы данных распространителя в **базы данных** сетки свойств и нажмите кнопку **Удалить** Удалить базу данных. Нельзя удалить базу данных распространителя, если существует только одна такая база данных; каждый распространитель должен иметь, по крайней мере, одну базу данных распространителя. Для удаления всех баз данных распространителя необходимо отключить распространение на компьютере. Дополнительные сведения см. в разделе [отключить публикацию и распространение](../../relational-databases/replication/disable-publishing-and-distribution.md).  
  
 **Параметры профиля по умолчанию**  
 Нажмите эту кнопку для доступа к профилям агента репликации в **Профили агента** диалоговое окно. Дополнительные сведения о профилях см. в разделе [Replication Agent Profiles](../../relational-databases/replication/agents/replication-agent-profiles.md).
