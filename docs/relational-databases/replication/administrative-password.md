# Административный пароль
  Если на **распространители** странице этого мастера выбран удаленный распространитель для этого издателя, необходимо ввести пароль для делает репликации соединения между издателем и распространителем с **distributor_admin** входа. Пароль должен совпадать с паролем, указанным на странице **Пароль распространителя** мастера настройки распространителя или на странице **Издатели** диалогового окна **Свойства распространителя** .  
  
## Параметры  
 **Пароль**  
 Введите пароль для соединения между издателем и удаленным распространителем.  
  
 **Подтвердите пароль**  
 Повторно введите пароль, чтобы подтвердить правильность ввода.
