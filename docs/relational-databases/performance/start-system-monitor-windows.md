# <a name="start-system-monitor-windows"></a>Запуск системного монитора (Windows)
[!INCLUDE[appliesto-ss-xxxx-xxxx-xxx-md](../../includes/appliesto-ss-xxxx-xxxx-xxx-md.md)] Системный монитор позволяет отслеживать использование системных ресурсов. Собирайте и просматривайте данные производительности реального времени в виде счетчиков. Вы также можете использовать счетчики для серверных ресурсов, таких как использование процессора и памяти. Кроме того, их можно использовать для многих ресурсов [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] Майкрософт, таких как блокировки и транзакции.  
  
## <a name="start-system-monitor-in-windows"></a>Запуск системного монитора в Windows  
  
В меню **Пуск** выберите пункт **Выполнить**, в диалоговом окне **Выполнить** введите **perfmon** и нажмите кнопку **ОК**.
