# <a name="spatial-data-catalog-views"></a>Представления каталога пространственных данных
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)]

  В настоящем разделе описаны представления каталога, содержащие сведения о пространственных данных.  
  
 [sys.spatial_index_tessellations &#40;Transact-SQL&#41;](../../relational-databases/system-catalog-views/sys-spatial-index-tessellations-transact-sql.md)  
 Представляет сведения о схеме тесселяции и параметрах каждого пространственного индекса.  
  
 [sys.spatial_indexes &#40;Transact-SQL&#41;](../../relational-databases/system-catalog-views/sys-spatial-indexes-transact-sql.md)  
 Представляет в главном индексе сведения о пространственных индексах.  
  
 [sys.spatial_reference_systems &#40;Transact-SQL&#41;](../../relational-databases/system-catalog-views/sys-spatial-reference-systems-transact-sql.md)  
 Перечисляет поддерживаемые системы пространственных ссылок (SRID).
