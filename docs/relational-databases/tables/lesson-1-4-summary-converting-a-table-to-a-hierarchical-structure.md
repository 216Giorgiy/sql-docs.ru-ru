# <a name="lesson-1-4---summary---converting-a-table-to-a-hierarchical-structure"></a>Занятие 1.4. Сводка. Преобразование таблицы в иерархическую структуру
[!INCLUDE[tsql-appliesto-ss2012-xxxx-xxxx-xxx-md](../../includes/tsql-appliesto-ss2012-xxxx-xxxx-xxx-md.md)] Тип данных **hierarchyid** может хранить иерархическую взаимосвязь в простом компактном формате. Соответствующие индексы могут эффективно возвращать данные по уровням иерархии, по вложенным деревьям иерархии или с помощью стандартных отдельных уточняющих запросов.  
  
В этом занятии были использованы два иерархических метода. В следующем занятии будет показано использование других подобных методов для проведения запросов и управления иерархическими данными.
