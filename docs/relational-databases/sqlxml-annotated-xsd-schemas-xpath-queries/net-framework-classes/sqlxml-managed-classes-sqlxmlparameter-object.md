# <a name="sqlxml-managed-classes---sqlxmlparameter-object"></a>Управляемые классы - SQLXML SqlXmlParameter, объект
[!INCLUDE[appliesto-ss-asdb-xxxx-xxx-md](../../../includes/appliesto-ss-asdb-xxxx-xxx-md.md)]
Объект SqlXmlParameter поддерживает следующие свойства:  
  
 Название  
 Имя параметра. Командам могут быть переданы параметры. Вызов метода CreateParameter объекта SqlXmlCommand создает объект параметра.  
  
 Значение  
 Значение параметра. Работающий пример см. в разделе [выполнение запросов SQL &#40; SQLXML управляемые классы &#41; ](../../../relational-databases/sqlxml-annotated-xsd-schemas-xpath-queries/net-framework-classes/executing-sql-queries-sqlxml-managed-classes.md).
