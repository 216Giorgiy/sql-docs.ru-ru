# Изменение адреса электронной почты пользователя (службы Master Data Services)
  В среде [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)]можно изменить адрес электронной почты, на который отправляются уведомления, и формат по умолчанию для писем с уведомлениями.  
  
## Предварительные требования  
 Чтобы выполнить эту процедуру:  
  
-   Необходимо иметь разрешение на доступ к функциональной области **Пользователи и группы** .  
  
### Изменение адреса электронной почты пользователя или формата электронной почты по умолчанию  
  
1.  В среде [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)]щелкните **Разрешения пользователей и групп**.  
  
2.  На странице **Пользователи** выберите строку для пользователя, которого необходимо изменить.  
  
3.  Нажмите **Изменить выделенного пользователя**.  
  
4.  Нажмите кнопку **Правка** над вкладкой **Общие**.  
  
5.  Обновите адрес электронной почты или формат писем.  
  
6.  Нажмите кнопку **Сохранить**.
