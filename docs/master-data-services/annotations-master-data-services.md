# Заметки (службы Master Data Services)
  Заметки в службах [!INCLUDE[ssMDSshort](../includes/ssmdsshort-md.md)]представляют собой комментарии, вводимые для указания подробных сведений о транзакциях.  
  
 Можно снабдить заметкой транзакцию для предоставления сведений о том, для чего произведено действие. Чтобы добавить связанную заметку, необходимо выбрать транзакцию. Заметки отображаются вместе с транзакциями в следующих областях:  
  
-   В функциональной области **Браузер** веб-приложения [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)] .  
  
-   В функциональной области **Управление версиями** веб-приложения [!INCLUDE[ssMDSmdm](../includes/ssmdsmdm-md.md)] .  
  
-   В MDS [!INCLUDE[ssMDSXLS](../includes/ssmdsxls-md.md)].
