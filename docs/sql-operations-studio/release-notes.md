## <a name="december-2017-december-public-preview"></a>Декабрь 2017 г. (декабрь Предварительная версия)

Дата выпуска: 12/19/2017 г.  
версия: 0.24.1

*Общедоступной предварительной версии декабря* включает в себя несколько исправлений ошибок по все функциональные области, а также следующие дополнительные возможности:

- Создать диалоговое окно правила брандмауэра для подключения к базе данных SQL Azure и хранилище данных SQL Azure теперь доступна.
- Добавлены установки Windows и Linux DEB и об/мин пакеты установки.
- Управляйте редактором visual макет панели мониторинга.
- *Как изменение скрипта* и *как выполняться сценарий* команд.
- *Выполнить текущий запрос с действительный план* команды.
- Интеграция платформы редактора VS Code 1.18.1.
- Включите файлы для загрузки неопубликованных приложений расширение VSIX.
- Поддерживает синтаксис итерации пакета «GO N».

Дополнительные сведения см. [журнал изменений](https://github.com/Microsoft/sqlopsstudio/blob/master/CHANGELOG.md).
