# <a name="keyboard-shortcuts-in-includename-sosincludesname-sosmd"></a>Сочетания клавиш в [!INCLUDE[name-sos](../includes/name-sos.md)]

Это статье описаны действия для быстрого просмотра, изменения и создания сочетаний клавиш в [!INCLUDE[name-sos](../includes/name-sos-short.md)].

Поскольку [!INCLUDE[name-sos](../includes/name-sos-short.md)] наследует его функциональность привязки ключей из кода Visual Studio, подробные сведения о расширенных настроек, используя различные раскладки клавиатуры, т. д., в [ключ привязки для Visual Studio Code](https://code.visualstudio.com/docs/getstarted/keybindings) статья. Некоторые функции соответствующие клавиши недоступен (например, расширения Keymap не поддерживаются в [!INCLUDE[name-sos](../includes/name-sos-short.md)]).


## <a name="open-the-keyboard-shortcuts-editor"></a>Откройте редактор сочетаний клавиш

Чтобы просмотреть все определенные сочетания клавиш:

Откройте **сочетания клавиш** редактор от **файл** меню: **файл** > **предпочтения**  >   **Сочетания клавиш** (**[!INCLUDE[name-sos](../includes/name-sos-short.md)]** > **предпочтения** > **сочетания клавиш** на Mac).

Помимо отображения текущей привязки клавиш, **сочетания клавиш** редактор отобразятся доступные команды, не имеющих определенные сочетания клавиш. **Сочетания клавиш** редактор позволяет легко изменить, удалить, сброс и определить новые сочетания клавиш.  


## <a name="edit-existing-keyboard-shortcuts"></a>Изменить существующие сочетания клавиш

Чтобы изменить соответствующие клавиши для существующего сочетания клавиш клавиатуры:

1. Найдите сочетание клавиш, который вы хотите изменить с помощью поля поиска или прокрутку списка.
   > [!TIP]
   > Поиск по ключу, командой, источник, т. д., чтобы вернуть все соответствующие сочетания клавиш.

1. Щелкните правой кнопкой мыши нужный элемент и выберите **соответствующие клавиши изменений**

   ![Изменение сочетания клавиш](media/keyboard-shortcuts/change-keybinding.png)

1. Нажмите нужное сочетание клавиш, нажмите клавишу **ввод** для его сохранения. 

   ![Сохранить сочетание клавиш](media/keyboard-shortcuts/save-keybinding.png)

## <a name="create-new-keyboard-shortcuts"></a>Создать новые сочетания клавиш

Чтобы создать новые сочетания клавиш:

1. Щелкните правой кнопкой мыши команду, которая нет любые соответствующие клавиши и выберите **добавить соответствующие клавиши**.

   ![Создание сочетания клавиш](media/keyboard-shortcuts/add-keybinding.png)

1. Нажмите нужное сочетание клавиш, нажмите клавишу **ввод** для его сохранения.
