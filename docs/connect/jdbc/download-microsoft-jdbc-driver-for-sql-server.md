# <a name="download-microsoft-jdbc-driver-for-sql-server"></a>Скачивание драйвера Microsoft JDBC Driver for SQL Server

![Загрузка стрелка вниз обведен](../../ssdt/media/download.png)[для загрузки драйвера JDBC](../sql-connection-libraries.md#anchor-20-drivers-relational-access)
 
## <a name="using-the-jdbc-driver-with-maven-central"></a>С помощью драйвера JDBC с центрального Maven
Драйвер JDBC можно добавить в проект Maven, добавив его в качестве зависимостей в файле POM.xml следующим кодом:

```xml
<dependency>
    <groupId>com.microsoft.sqlserver</groupId>
    <artifactId>mssql-jdbc</artifactId>
    <version>6.2.1.jre8</version>
</dependency>
```

## <a name="unsupported-drivers"></a>Неподдерживаемые драйверы  
Неподдерживаемых версий драйверов не доступна для загрузки. Мы постоянно работаем над улучшением возможностей подключения Java, Таким образом настоятельно рекомендуется работать с последней версией драйвера JDBC.
