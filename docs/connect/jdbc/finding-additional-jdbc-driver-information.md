# <a name="finding-additional-jdbc-driver-information"></a>Поиск дополнительных сведений о драйвере JDBC
[!INCLUDE[Driver_JDBC_Download](../../includes/driver_jdbc_download.md)]

  Дополнительные сведения о [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)] и [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] разработки в целом см. следующие ресурсы сети:  
  
## <a name="remarks"></a>Замечания  
  
|Ресурс|Описание|  
|--------------|-----------------|  
|[Драйвер JDBC для репозитория GitHub SQL Server](https://github.com/microsoft/mssql-jdbc)|Этот репозиторий содержит исходный код для драйвера JDBC для SQL Server. Использовать этот сайт для файла проблем и вносить в драйвере.|
|[Центр разработчиков хранилищ и доступа к данным](http://go.microsoft.com/fwlink?linkid=4173)|Этот сайт содержит документацию, технические статьи, образец кода и другие ресурсы для всех технологий доступа к данным [!INCLUDE[msCoName](../../includes/msconame_md.md)].|  
|[Форум доступа к данным SQL Server](http://go.microsoft.com/fwlink/?LinkId=70651)|Этот сайт представляет форум сообщества, посвященный доступу к данным [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] с помощью [!INCLUDE[ssNoVersion](../../includes/ssnoversion_md.md)] собственного клиента OLE DB, ODBC, ADO, MDAC, JDBC и SOAP/HTTP.|  
|[Блог JDBC](http://go.microsoft.com/fwlink/?LinkId=124746)|Этот блог используется для предоставления сведений о [!INCLUDE[jdbcNoVersion](../../includes/jdbcnoversion_md.md)] и пообщаться непосредственно с разработчиками драйвера JDBC.|  
|[Microsoft Connect](http://go.microsoft.com/fwlink/?LinkID=116905)|На этом сайте можно выполнить поиск проблем драйвера, которые были отправлены пользователями, и отправить описание своей проблемы.|
