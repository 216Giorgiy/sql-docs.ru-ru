# <a name="getcharacterstream-method-sqlserverclob"></a>Метод getCharacterStream (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Извлекает **Clob** значение, указанное объектом java.sql.Clob в виде объекта java.io.Reader.

|----------|-----------------|

|[Метод getCharacterStream &#40;long, long&#41;](../../../connect/jdbc/reference/getcharacterstream-method-long-long.md)|Возвращает данные Clob в виде объекта java.io.Reader или потока символов с указанной позицией и длиной.|
