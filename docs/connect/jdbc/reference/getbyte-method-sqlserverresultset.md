# <a name="getbyte-method-sqlserverresultset"></a>Метод getByte (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Извлекает значение заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде объекта **byte** на языке программирования Java.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Имя|Описание|  
|----------|-----------------|  
|[getByte (int)](../../../connect/jdbc/reference/getbyte-method-int-sqlserverresultset.md)|Возвращает значение индекса заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде массива типа **byte** на языке программирования Java.|  
|[getByte (java.lang.String)](../../../connect/jdbc/reference/getbyte-method-java-lang-string-sqlserverresultset.md)|Получает значение имени заданного столбца в текущей строке этого объекта [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) в виде значения типа **byte** на языке программирования Java.|
