# <a name="setint-method-sqlserverpreparedstatement"></a>setInt метод (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Устанавливает указанный параметр заданного **int** значение.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public final void setInt(int n,  
                         int value)  
```  
  
#### <a name="parameters"></a>Параметры  
 *n*  
  
 **Int** указывает номер параметра.  
  
 *value*  
  
 **Int** значение.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод setInt указывается с помощью метода setInt в интерфейсе java.sql.PreparedStatement.
