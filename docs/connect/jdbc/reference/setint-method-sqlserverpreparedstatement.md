# <a name="setint-method-sqlserverpreparedstatement"></a>Метод setInt (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Задает для указанного параметра указанное значение **int**.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public final void setInt(int n,  
                         int value)  
```  
  
#### <a name="parameters"></a>Параметры  
 *n*  
  
 Значение **int**, определяющее номер параметра.  
  
 *value*  
  
 **Int** значение.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод setInt определен с помощью метода setInt в интерфейсе java.sql.PreparedStatement.
