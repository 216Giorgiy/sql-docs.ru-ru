# <a name="setshort-method-sqlserverpreparedstatement"></a>setShort метод (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Устанавливает указанный параметр заданного **короткие** значение.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public final void setShort(int index,  
                           short x)  
```  
  
#### <a name="parameters"></a>Параметры  
 *Индекс*  
  
 **Int** указывает номер параметра.  
  
 *x*  
  
 Объект **короткие** значение.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод setShort указывается с помощью метода setShort в интерфейсе java.sql.PreparedStatement.
