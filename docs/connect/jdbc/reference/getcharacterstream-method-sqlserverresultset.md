# <a name="getcharacterstream-method-sqlserverresultset"></a>getCharacterStream метод (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Получает значение заданного столбца в текущей строке [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) объекта в виде объекта java.io.Reader.

|[Метод getCharacterStream (java.lang.String)](../../../connect/jdbc/reference/getcharacterstream-method-java-lang-string.md)|Получает значение имени заданного столбца в текущей строке [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) объекта в виде объекта java.io.Reader.|
