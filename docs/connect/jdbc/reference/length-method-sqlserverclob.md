# <a name="length-method-sqlserverclob"></a>Метод length (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает количество символов в объекте CLOB.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public long length()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Длина объекта CLOB в символах.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод Длина указывается длина методом в интерфейсе java.sql.Clob.
