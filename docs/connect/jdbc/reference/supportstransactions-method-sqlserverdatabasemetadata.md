# <a name="supportstransactions-method-sqlserverdatabasemetadata"></a>Метод supportsTransactions (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает значение, определяющее, поддерживает ли эта база данных транзакции.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public boolean supportsTransactions()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 **значение true,** , если поддерживается. В противном случае — **false**.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод supportsTransactions указывается с помощью метода supportsTransactions в интерфейсе java.sql.DatabaseMetaData.
