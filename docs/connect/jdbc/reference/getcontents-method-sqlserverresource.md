# <a name="getcontents-method-sqlserverresource"></a>Метод getContents (SQLServerResource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Получает данные из строки ошибок. Этот метод предназначен только для внутреннего использования.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
protected java.lang.Object[][] getContents()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Многомерный массив **объекта** значения.
