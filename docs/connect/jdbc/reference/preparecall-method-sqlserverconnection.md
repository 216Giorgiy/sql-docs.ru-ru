# <a name="preparecall-method-sqlserverconnection"></a>prepareCall метод (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Создает [SQLServerCallableStatement](../../../connect/jdbc/reference/sqlservercallablestatement-class.md) объект для вызова хранимых процедур базы данных.

|Имя|Description|  
|----------|-----------------|  
|[prepareCall (java.lang.String)](../../../connect/jdbc/reference/preparecall-method-java-lang-string.md)|Создает объект SQLServerCallableStatement для вызова хранимых процедур базы данных.|  
|[prepareCall (java.lang.String, int, int)](../../../connect/jdbc/reference/preparecall-method-java-lang-string-int-int.md)|Создает объект SQLServerCallableStatement, который создает [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) объекты с заданным типом и видом параллелизма.|  
|[prepareCall (java.lang.String, int, int, int)](../../../connect/jdbc/reference/preparecall-method-java-lang-string-int-int-int.md)|Создает объект SQLServerCallableStatement, который создает объекты SQLServerResultSet с заданным типом, видом параллелизма и возможностью сохранения.|
