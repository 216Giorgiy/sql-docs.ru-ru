# <a name="getinstancename-method-sqlserverdatasource"></a>Метод getInstanceName (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] имя экземпляра.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public java.lang.String getInstanceName()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Объект **строка** , содержащий экземпляр имя или значение null, если значение не задано.
