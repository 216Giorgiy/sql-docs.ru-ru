# <a name="getdescription-method-sqlserverdatasource"></a>Метод getDescription (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает описание источника данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public java.lang.String getDescription()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Объект **строка** , содержащий описание источника данных или значение null, если значение не задано.
