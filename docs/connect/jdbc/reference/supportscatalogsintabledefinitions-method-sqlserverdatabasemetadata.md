# <a name="supportscatalogsintabledefinitions-method-sqlserverdatabasemetadata"></a>Метод supportsCatalogsInTableDefinitions (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает значение, определяющее, может ли имя каталога использоваться в инструкции определения таблицы.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public boolean supportsCatalogsInTableDefinitions()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 **значение true,** , если поддерживается. В противном случае — **false**.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод supportsCatalogsInTableDefinitions указывается с помощью метода supportsCatalogsInTableDefinitions в интерфейсе java.sql.DatabaseMetaData.
