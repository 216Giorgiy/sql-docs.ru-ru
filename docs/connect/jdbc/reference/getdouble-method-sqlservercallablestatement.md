# <a name="getdouble-method-sqlservercallablestatement"></a>Метод getDouble (SQLServerCallableStatement)

|----------|-----------------|  
|[getDouble (int)](../../../connect/jdbc/reference/getdouble-method-int.md)|Получает значение указанного параметра в виде **двойные** на Java по заданному индексу параметра языка программирования.|  
|[getDouble (java.lang.String)](../../../connect/jdbc/reference/getdouble-method-java-lang-string.md)|Получает значение указанного параметра в виде **двойные** на Java по заданному имени параметра языка программирования.|
