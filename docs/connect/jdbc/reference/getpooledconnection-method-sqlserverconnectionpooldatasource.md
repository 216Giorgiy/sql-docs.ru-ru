# <a name="getpooledconnection-method-sqlserverconnectionpooldatasource"></a>Метод getPooledConnection (SQLServerConnectionPoolDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]
