# <a name="getbigdecimal-method-sqlservercallablestatement"></a>Метод getBigDecimal (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает значение указанного параметра в виде значения java.math.BigDecimal.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Название|Описание|  
|----------|-----------------|  
|[getBigDecimal (int)](../../../connect/jdbc/reference/getbigdecimal-method-int.md)|Получает значение указанного параметра в виде объекта java.math.BigDecimal с полной точностью по заданному индексу параметра.|  
|[getBigDecimal (int, int)](../../../connect/jdbc/reference/getbigdecimal-method-int-int.md)|(Устарело) Получает значение указанного параметра в виде объекта java.math.BigDecimal по заданному индексу параметра и масштабу.|  
|[getBigDecimal (java.lang.String)](../../../connect/jdbc/reference/getbigdecimal-method-java-lang-string.md)|Получает значение указанного параметра в виде объекта java.math.BigDecimal с полной точностью по заданному имени параметра.|  
|[getBigDecimal (java.lang.String, int)](../../../connect/jdbc/reference/getbigdecimal-method-java-lang-string-int.md)|(Устарело) Получает значение указанного параметра в виде объекта java.math.BigDecimal по заданному имени параметра и масштабу.|
