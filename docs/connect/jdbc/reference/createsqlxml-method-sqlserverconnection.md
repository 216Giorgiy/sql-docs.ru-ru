# <a name="createsqlxml-method-sqlserverconnection"></a>Метод createSQLXML (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Создает **SQLXML** объект без данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public java.sql.SQLXML createSQLXML()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Объект SQLXML.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Remarks  
 Этот метод createSQLXML указывается методом createSQLXML в интерфейсе java.sql.Connection.
