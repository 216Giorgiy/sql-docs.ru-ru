# <a name="createsqlxml-method-sqlserverconnection"></a>Метод createSQLXML (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Создает **SQLXML** объекта без данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public java.sql.SQLXML createSQLXML()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Объект SQLXML.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод createSQLXML задается методом createSQLXML в интерфейсе java.sql.Connection.
