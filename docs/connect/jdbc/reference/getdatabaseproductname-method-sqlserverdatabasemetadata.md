# <a name="getdatabaseproductname-method-sqlserverdatabasemetadata"></a>Метод getDatabaseProductName (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает имя данного продукта базы данных.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public java.lang.String getDatabaseProductName()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Объект **строка** , содержащее название продукта базы данных.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод getDatabaseProductName указывается с помощью метода getDatabaseProductName в интерфейсе java.sql.DatabaseMetaData.
