# <a name="getblob-method-sqlservercallablestatement"></a>Метод getBlob (SQLServerCallableStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Получает значение указанного параметра JDBC BLOB в виде большого двоичного объекта на языке программирования Java.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Имя|Description|  
|----------|-----------------|  
|[getBlob (int)](../../../connect/jdbc/reference/getblob-method-int.md)|Получает значение указанного параметра JDBC BLOB в виде большого двоичного объекта в языке программирования Java по заданному индексу параметра.|  
|[getBlob (java.lang.String)](../../../connect/jdbc/reference/getblob-method-java-lang-string.md)|Получает значение указанного параметра JDBC BLOB в виде большого двоичного объекта в языке программирования Java по заданному имени параметра.|
