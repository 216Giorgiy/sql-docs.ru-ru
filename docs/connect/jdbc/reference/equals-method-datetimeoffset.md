# <a name="equals-method-datetimeoffset"></a>Метод equals (DateTimeOffset)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Сравнивает этот **DateTimeOffset** объекта в другой **объекта**.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public boolean equals(Object o)  
```  
  
#### <a name="parameters"></a>Параметры  
 *o*  
  
 Объект для сравнения с текущим экземпляром.  
  
## <a name="return-value"></a>Возвращаемое значение  
 Возвращает true, если **DateTimeOffset** объект представляет ту же точку во времени с тем же смещением от времени GMT, как другой **DateTimeOffset**.
