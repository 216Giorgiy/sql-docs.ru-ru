# <a name="hashcode-method-datetimeoffset"></a>Метод hashCode (DateTimeOffset)

  Возвращает значение хэш-кода для объекта.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public int hashCode()  
```  
  
## <a name="return-value"></a>Возвращаемое значение
