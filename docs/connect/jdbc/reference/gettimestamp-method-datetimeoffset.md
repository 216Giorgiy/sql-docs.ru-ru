# <a name="gettimestamp-method-datetimeoffset"></a>Метод getTimestamp (DateTimeOffset)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает точку временем, представленными этим экземпляром **DateTimeOffset** объект как java.sql.Timestamp.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public java.sql.Timestamp getTimestamp()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Значение типа java.sql.Timestamp.
