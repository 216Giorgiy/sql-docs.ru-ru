# <a name="first-method-sqlserverresultset"></a>Первый метод (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Перемещение курсора к первой строке [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) объекта.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public boolean first()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 **значение true,** при перемещении курсора на первую строку. В противном случае — **false**.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Это первый метод указывается с помощью первого метода в интерфейсе java.sql.ResultSet.
