# <a name="getclientinfo-method-sqlserverconnection"></a>Метод getClientInfo (SQLServerConnection)

|----------|-----------------|

|[Метод getClientInfo &#40;java.lang.String &#41;](../../../connect/jdbc/reference/getclientinfo-method-java-lang-string.md)|Извлекает значение указанного свойства данных клиента.|
