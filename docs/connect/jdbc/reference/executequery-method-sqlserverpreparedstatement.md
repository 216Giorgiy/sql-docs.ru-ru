# <a name="executequery-method-sqlserverpreparedstatement"></a>executeQuery метод (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Выполняет запрос SQL в этом [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) объекта и возвращает [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) объект, созданный запросом.

|Название|Описание|  
|----------|-----------------|  
|[executeQuery)](../../../connect/jdbc/reference/executequery-method.md)|Выполняет запрос SQL в этом [SQLServerPreparedStatement](../../../connect/jdbc/reference/sqlserverpreparedstatement-class.md) объекта и возвращает [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) объекта, созданного запросом.|  
|[executeQuery (java.lang.String)](../../../connect/jdbc/reference/executequery-method-java-lang-string.md)|Выполняет заданную инструкцию SQL и возвращает один [SQLServerResultSet](../../../connect/jdbc/reference/sqlserverresultset-class.md) объекта.|
