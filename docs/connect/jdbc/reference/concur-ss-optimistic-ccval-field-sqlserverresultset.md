# <a name="concurssoptimisticccval-field-sqlserverresultset"></a>Поле CONCUR_SS_OPTIMISTIC_CCVAL (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Используется для указания [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] чтение и запись типа оптимистичный параллелизм без блокировки строк.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public static final int CONCUR_SS_OPTIMISTIC_CCVAL  
```  
  
## <a name="field-value"></a>Значение поля  
 **Int** значение 1010 типа.
