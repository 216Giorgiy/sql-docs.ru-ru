# <a name="typessservercursorforwardonly-field-sqlserverresultset"></a>Поле TYPE_SS_SERVER_CURSOR_FORWARD_ONLY (SQLServerResultSet)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Используется для указания [!INCLUDE[ssNoVersion](../../../includes/ssnoversion_md.md)] тип курсора быстрый однопроходный, только для чтения.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public static final int TYPE_SS_SERVER_CURSOR_FORWARD_ONLY  
```  
  
## <a name="field-value"></a>Значение поля  
 **Int** значение 2004 г.
