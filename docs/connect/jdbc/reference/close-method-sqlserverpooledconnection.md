# <a name="close-method-sqlserverpooledconnection"></a>Метод close (SQLServerPooledConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Закрывает физическое соединение, это [SQLServerPooledConnection](../../../connect/jdbc/reference/sqlserverpooledconnection-class.md) представляет.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public void close()  
```  
  
## <a name="exceptions"></a>Исключения  
 java.sql.SQLException  
  
## <a name="remarks"></a>Замечания  
 Этот метод close определяется close-метод в интерфейсе javax.sql.PooledConnection.
