# <a name="setstring-method-sqlserverclob"></a>Метод setString (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Записывает заданное значение **String** в CLOB, начиная с заданной позиции.

|[setString (long, java.lang.String, int, int)](../../../connect/jdbc/reference/setstring-method-long-java-lang-string-int-int.md)|Записывает заданную строку в объект CLOB, начиная с заданной позиции, по заданному смещению и длине.|
