# <a name="getdrivername-method-sqlserverdatabasemetadata"></a>Метод getDriverName (SQLServerDatabaseMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает имя этого драйвера JDBC.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public java.lang.String getDriverName()  
```  
  
## <a name="return-value"></a>Возвращаемое значение  
 Объект **строка** , содержащее имя драйвера JDBC.  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод getDriverName задается методом getDriverName в интерфейсе java.sql.DatabaseMetaData.
