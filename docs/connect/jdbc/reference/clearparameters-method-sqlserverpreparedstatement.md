# <a name="clearparameters-method-sqlserverpreparedstatement"></a>clearParameters метод (SQLServerPreparedStatement)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Немедленно удаляет текущие значения параметров.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public final void clearParameters()  
```  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод clearParameters указывается с помощью метода clearParameters в интерфейсе java.sql.PreparedStatement.
