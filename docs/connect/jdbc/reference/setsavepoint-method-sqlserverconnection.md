# <a name="setsavepoint-method-sqlserverconnection"></a>setSavepoint метод (SQLServerConnection)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Создает в текущей транзакции неименованную точку сохранения и возвращает новый [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) , представляющий его.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Название|Описание|  
|----------|-----------------|  
|[setSavepoint)](../../../connect/jdbc/reference/setsavepoint-method.md)|Создает в текущей транзакции неименованную точку сохранения и возвращает новый [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) , представляющий его.|  
|[setSavepoint (java.lang.String)](../../../connect/jdbc/reference/setsavepoint-method-java-lang-string.md)|Создает точку сохранения с заданным именем в текущей транзакции и возврат нового [SQLServerSavepoint](../../../connect/jdbc/reference/sqlserversavepoint-class.md) , представляющий его.|
