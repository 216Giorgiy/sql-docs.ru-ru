# <a name="setapplicationintent-method-sqlserverdatasource"></a>Метод setApplicationIntent (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Задает значение **applicationIntent** свойство соединения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
public void setApplicationIntent(java.lang.String applicationIntent);  
```  
  
#### <a name="parameters"></a>Параметры  
 *applicationIntent*  
  
 Новое значение **applicationIntent** свойство соединения.  
  
## <a name="remarks"></a>Remarks  
 Дополнительные сведения о **applicationIntent** свойство подключения, см. в разделе [заданию свойств соединения](../../../connect/jdbc/setting-the-connection-properties.md).
