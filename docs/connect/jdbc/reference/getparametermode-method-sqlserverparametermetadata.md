# <a name="getparametermode-method-sqlserverparametermetadata"></a>Метод getParameterMode (SQLServerParameterMetaData)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает режим указанного параметра.  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public int getParameterMode(int param)  
```  
  
#### <a name="parameters"></a>Параметры  
 *param*  
  
 **Int** , указывающее индекс параметра.  
  
## <a name="return-value"></a>Возвращаемое значение  
 **Int** , указывающее режим указанного параметра, который может принимать одно из следующих значений:  
  
 ParameterMetaData.parameterModeIn  
  
 ParameterMetaData.parameterModeInOut  
  
 ParameterMetaData.parameterModeOut  
  
 ParameterMetaData.parameterModeUnknown  
  
## <a name="exceptions"></a>Исключения  
 [SQLServerException](../../../connect/jdbc/reference/sqlserverexception-class.md)  
  
## <a name="remarks"></a>Замечания  
 Этот метод getParameterMode указывается с помощью метода getParameterMode в интерфейсе java.sql.ParameterMetaData.
