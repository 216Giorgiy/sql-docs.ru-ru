# <a name="position-method-sqlservernclob"></a>Метод position (SQLServerNClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает позицию символа указанного **NClob** объекта или подстроки в **NClob** по указанной начальной позиции.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Имя|Description|  
|----------|-----------------|  
|[Поместите метод &#40;java.sql.NClob, долго &#41;](../../../connect/jdbc/reference/position-method-java-sql-nclob-long.md)|Возвращает позицию символа, с которой указанный **NClob** объекта *searchstr* появляется в этом **NClob** объекта.|  
|[Поместите метод &#40;java.lang.String, долго &#41; &#40; SQLServerNClob &#41;](../../../connect/jdbc/reference/position-method-java-lang-string-long-sqlservernclob.md)|Возвращает позицию символа, в которой указанная подстрока *searchstr* отображается в **NCLOB** значение, представляемое этим **NClob** объекта.|
