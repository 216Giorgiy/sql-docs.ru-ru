# <a name="setpassword-method-sqlserverdatasource"></a>Метод setPassword (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Задает пароль, который будет использоваться для подключения к [!INCLUDE[ssNoVersion](../../../includes/ssnoversion-md.md)].  
  
## <a name="syntax"></a>Синтаксис  
  
```  
  
public void setPassword(java.lang.String password)  
```  
  
#### <a name="parameters"></a>Параметры  
 *password*  
  
 Значение типа **String**, содержащее пароль.
