# <a name="setmultisubnetfailover-method-sqlserverdatasource"></a>Метод setMultiSubnetFailover (SQLServerDataSource)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Задает значение **multiSubnetFailover** свойство соединения.  
  
## <a name="syntax"></a>Синтаксис  
  
```vb  
public void setMultiSubnetFailover(boolean multiSubnetFailover);  
```  
  
#### <a name="parameters"></a>Параметры  
 *MultiSubnetFailover*  
  
 Новое значение **multiSubnetFailover** свойство соединения.
