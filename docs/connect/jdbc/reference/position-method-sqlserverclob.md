# <a name="position-method-sqlserverclob"></a>Метод position (SQLServerClob)
[!INCLUDE[Driver_JDBC_Download](../../../includes/driver_jdbc_download.md)]

  Возвращает позицию символа указанного объекта CLOB или подстроки в объекте CLOB по заданной начальной позиции.  
  
## <a name="overload-list"></a>Список перегрузок  
  
|Имя|Description|  
|----------|-----------------|  
|[положение (java.sql.Clob, long)](../../../connect/jdbc/reference/position-method-java-sql-clob-long.md)|Возвращает позицию символа указанного объекта CLOB в CLOB по заданной начальной позиции.|  
|[положение (java.lang.String, long)](../../../connect/jdbc/reference/position-method-java-lang-string-long.md)|Возвращает позицию символа указанной подстроки в CLOB по заданной начальной позиции.|
