# <a name="download-the-microsoft-drivers-for-php-for-sql-server"></a>Загрузка драйверов Майкрософт для PHP для SQL Server

[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]

## <a name="windows"></a>Windows

Следующие версии драйверов для PHP для Windows доступны для загрузки:

- [5.2 драйверы Майкрософт для PHP для SQL Server](https://www.microsoft.com/en-us/download/details.aspx?id=56729)
- [4.3 драйверы Майкрософт для PHP для SQL Server](https://www.microsoft.com/en-us/download/details.aspx?id=55642)
- [Драйверы Microsoft 4.0, 3.2, 3.1 и 3.0 для PHP для SQL Server](https://www.microsoft.com/en-us/download/details.aspx?id=20098)

## <a name="linux-and-macos"></a>Linux и macOS

Драйверы для PHP, легко загружаются и установить с помощью нагрузка PECL на macOS и Linux. В разделе [Linux и macOS учебника установки](installation-tutorial-linux-mac.md) подробные сведения. Если необходимо вручную установить драйверы для PHP в ОС Linux и macOS следующие версии доступны для загрузки:

- [5.2 драйверы Майкрософт для PHP для SQL Server](https://github.com/Microsoft/msphpsql/releases/tag/v5.2.0)
- [4.3 драйверы Майкрософт для PHP для SQL Server](https://github.com/Microsoft/msphpsql/releases/tag/v4.3.0)
