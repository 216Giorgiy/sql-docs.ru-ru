# <a name="step-1-configure-development-environment-for-php-development"></a>Шаг 1. Настройка среды разработки для разработки PHP
[!INCLUDE[Driver_PHP_Download](../../includes/driver_php_download.md)]




* Определите, какая версия драйвера PHP будет использоваться, описанных здесь в зависимости от вашей среды: [требования к системе для драйвера SQL PHP](../../connect/php/system-requirements-for-the-php-sql-driver.md)
* Загрузите и установите соответствующий драйвер PHP здесь: [загрузить драйвер Microsoft PHP](https://www.microsoft.com/download/details.aspx?id=20098)  
* Загрузите и установите соответствующий драйвер ODBC здесь: [загрузить драйвер ODBC для SQL Server](../../connect/odbc/download-odbc-driver-for-sql-server.md)  
* Настройте драйвер PHP и веб-сервер для вашей операционной системе.

### <a name="windows"></a>Windows  
  

* Настройка загрузки драйвера PHP, как указано ниже: [Загрузка драйвера SQL PHP](../../connect/php/loading-the-php-sql-driver.md) 
* Настройка IIS для размещения приложений PHP, как указано ниже: [настройки IIS для драйвера SQL PHP](../../connect/php/configuring-iis-for-php-sql-driver.md)

### <a name="linux-and-mac"></a>Linux и Mac


*   Настройка, загрузка драйвера PHP и настройка веб-сервера для размещения приложений PHP, описанных здесь: [PHP Linux и Mac драйверы установки учебника](../../connect/php/installation-tutorial-linux-mac.md)
