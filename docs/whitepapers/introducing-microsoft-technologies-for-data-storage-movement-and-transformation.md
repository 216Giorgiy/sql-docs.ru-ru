# Введение в технологии Майкрософт для хранения, перемещения и преобразования данных
**Аннотация:** в этой вводной статье описаны технологии Майкрософт для хранения, перемещения и преобразования данных. Эти технологии обеспечивают работу бизнес-аналитики, позволяя вашей организации хранить гетерогенные качественные данные для анализа и формирования отчетов в облаке или локально, а также обращаться к ним.  
  
Обзор технологий бизнес-аналитики Майкрософт см. в статье [Introducing Microsoft BI Reporting and Analysis Tools](https://msdn.microsoft.com/en-us/library/dn655131.aspx) (Введение в инструменты для формирования отчетов и анализа бизнес-аналитики Майкрософт).
