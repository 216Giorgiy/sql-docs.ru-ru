## <a name="bkmk_Prerequisites"></a>Предварительные требования

Чтобы использовать эти учебники, необходимо иметь 2017 г. SQL Server, и необходимо явно установить и затем включите компонент службы обучения машины (в базе данных). 

2017 г. SQL Server поддерживает языки R и Python, но не установлен или включен по умолчанию. Для запуска Python необходимо включить extensibility framework, и выберите Python, как язык для установки. 

### <a name="post-installation-configuration-tips"></a>Советы по настройке после установки

После запуска программы установки SQL Server, может потребоваться выполнить некоторые дополнительные действия, чтобы обеспечить взаимодействие Python и SQL Server:

+ Включение возможности выполнения внешнего сценария, запустив `sp_configure 'external scripts enabled', 1`.
+ Перезапустите сервер. 
+ Откройте **служб** панель, чтобы проверить, запущен ли панель запуска. 
+ Убедитесь в наличии необходимых разрешений на службу, которая обращается к внешней среде выполнения. Дополнительные сведения см. в разделе [Enable неявной проверки подлинности](../r/add-sqlrusergroup-to-database.md).
+ Открытие порта в брандмауэре для SQL Server и включите необходимые сетевые протоколы.
+ Убедитесь в наличии необходимых разрешений для подключения к серверу для чтения данных, а также для создания объектов базы данных, необходимые для образца вашей учетной записи пользователя Windows или имя входа SQL.

Для некоторых распространенных проблем в статье: [Устранение неполадок службы машины обучения](../machine-learning-troubleshooting-faq.md)

### <a name="resource-management"></a>Управление ресурсами

На одном компьютере можно установить R и Python, но для выполнения и может потребоваться значительное количество ресурсов. Если возникли ошибки «недостаточно памяти» или выполнения заданий обучения компьютера является основной намеченное использование сервера, можно уменьшить объем памяти, выделенной с компонентом database engine. Дополнительные сведения см. в разделе [управление и мониторинг Python в SQL Server](../python/managing-and-monitoring-python-solutions.md).
