# Пошаговое руководство по обработке и анализу данных
В этом пошаговом руководстве вы создадите готовое решение для прогнозирующего моделирования с помощью служб [!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)].  
  
Это руководство основано на общедоступном наборе данных по работе такси в Нью-Йорке. Используя сочетание кода на языке R, данных [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] и пользовательских функций SQL, вы создадите модель классификации, которая сообщает вероятность получения чаевых таксистом за конкретную поездку. Вы также развернете модель R в [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] и используете данные на сервере для формирования оценок на основе модели.  
  
Этот пример можно легко адаптировать для решения других реальных проблем, таких как прогнозирование отклика клиентов на кампании по сбыту или прогнозирование расходов посетителей мероприятий. Так как модель можно вызывать из хранимой процедуры, ее можно легко внедрить в приложение.  
  
**Целевая аудитория**  
  
Это пошаговое руководство предназначено для разработчиков на языке R. Вы также должны быть знакомы с основными операциями с базами данных, такими как создание баз данных и таблиц, импорт данных в таблицы и запрос таблиц с помощью [!INCLUDE[tsql](../../includes/tsql-md.md)].  Вам будут предоставлены скрипты SQL и R для выполнения.  
  
На тот случай, если вы не знакомы с языком R, но хорошо разбираетесь в базах данных, в этом учебнике содержатся общие сведения об интеграции R в корпоративные рабочие процессы с помощью служб [!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)]. Писать код на языке R не требуется — все скрипты предоставляются. Однако для выполнения команд требуется какая-либо интегрированная среда разработки R.  
  
**Предварительные требования**  
  
Для прохождения этого учебника требуется доступ к экземпляру [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)] с установленными службами [!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)] . В локальной среде подготовьте рабочую станцию для обработки и анализа данных, которая может подключаться к экземпляру [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)], а затем установите необходимые библиотеки R.  
  
Дополнительные сведения см. в разделе [Предварительные требования для работы с пошаговыми руководствами по обработке и анализу данных (службы SQL Server R Services)](../../advanced-analytics/r-services/prerequisites-for-data-science-walkthroughs-sql-server-r-services.md).

## <a name="notes"></a>Примечания  
Так как это пошаговое руководство призвано познакомить разработчиков на языке R со службами [!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)], по возможности все действия выполняются с помощью языка R. Это не означает, что язык R непременно является лучшим средством для выполнения каждой задачи. Во многих случаях [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] может обеспечивать более высокую производительность, особенно для таких задач, как агрегирование данных и формирование характеристик. Для выполнения таких задач новые возможности [!INCLUDE[ssCurrent](../../includes/sscurrent-md.md)], такие как оптимизированные для памяти индексы columnstore, могут быть особенно полезны.
