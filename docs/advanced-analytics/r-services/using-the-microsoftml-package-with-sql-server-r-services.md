# Использование пакета MicrosoftML со службами R Services SQL Server
Пакет **MicrosoftML**, который входит в состав Microsoft R Server и SQL Server vNext CTP 1.0, включает в себя несколько алгоритмов машинного обучения, разработанных корпорацией Майкрософт и поддерживающих многоядерную обработку и быструю потоковую передачу данных. Пакет также содержит преобразования для обработки текста и добавления функций.

### <a name="new-machine-learning-algorithms"></a>Новые алгоритмы машинного обучения


-  **Быстрый линейный.** Линейное средство обучения на базе стохастического парного координатного подъема, которое можно использовать для двоичной классификации или регрессии. Модель поддерживает регуляризацию L1 и L2.

- **Быстрый на базе дерева.** Ускоренный алгоритм на базе дерева принятия решений, ранее известный как FastRank, который был разработан для использования в Bing. Это одно из самых популярных и быстрых средств обучения. Поддерживает двоичную классификацию и регрессию.

- **Быстрый на базе леса.** Модель логистической регрессии на основе метода случайного леса. Этот вариант похож на функцию `rxLogit` в RevoScaleR, но поддерживает регуляризацию L1 и L2. Поддерживает двоичную классификацию и регрессию.

- **Логистическая регрессия** Модель логистической регрессии аналогична функции `rxLogit` в RevoScaleR и дополнительно поддерживает регуляризацию L1 и L2. Поддерживает двоичную или многоклассовую классификацию.

- **Нейронная сеть** Модель нейронной сети для двоичной классификации, многоклассовой классификации и регрессии. Поддерживает ускорение GPU и настраиваемых сверточных сетей.

- **Одноклассовый метод опорных векторов** Модель обнаружения аномалий на основе метода опорных векторов, которую можно использовать для двоичной классификации в несбалансированных наборах данных.

## <a name="transformation-functions"></a>Функции преобразования

Пакет **MicrosoftML** также содержит указанные ниже функции, которые можно использовать для преобразования данных и извлечения компонентов.

- `featurizeText()`
 
  Создает некоторое число n-грамм в заданной текстовой строке. 

  Функция включает возможность определения используемого языка, разметки и нормализации текста, удаления стоп-слов и создания функций из текста. 

- `categorical()`

  Формирует словарь из категорий и преобразует каждую из них в вектор индикатора. 
 
- `categoricalHash()`

  Преобразует категориальные значения в массив индикатора посредством хеширования значения и использования хеша в качестве индекса в контейнере.  

- `selectFeatures()` 

  Выбирает подмножество функций из заданной переменной посредством подсчета нестандартных значений или вычисления оценки взаимной информации относительно метки. 

Подробные сведения об этих новых возможностях см. в разделе [Функции MicrosoftML](https://msdn.microsoft.com/microsoft-r/microsoftml/microsoftml).

## <a name="support-for-microsoftml-in-r-services"></a>Поддержка MicrosoftML в службах R Services

Пакет **MicrosoftML** полностью интегрирован с конвейером обработки данных, используемым пакетом **RevoScaleR**. Сейчас вы можете использовать пакет **MicrosoftML** в любом контексте вычислений на основе Windows, включая службы R Services SQL Server.
