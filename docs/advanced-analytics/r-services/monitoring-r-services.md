# Наблюдение за службами R
  Этот раздел содержит сведения об общих задачах мониторинга при использовании [!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)]:  
  
-   Определение ресурсов сервера, используемый R  
  
-   Наблюдение за состоянием инструменты R  
  
-   Определение задания, которые выполняются с помощью среды выполнения R  
  
-   Идентификация удаленного задания R, использующие [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] экземпляр контекста вычисления  
  
-   Сопоставление длительные запросы с задания R  
  
-   Завершение процессов R  
  
 Здесь также описываются средства, которые установлены на [!INCLUDE[ssNoVersion](../../includes/ssnoversion-md.md)] компьютере как часть установки R и списки динамических административных представлений и расширенных событий, связанных с [!INCLUDE[rsql_productname](../../includes/rsql-productname-md.md)].
