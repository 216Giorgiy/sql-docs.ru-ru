# R Server (Standalone)
  **Сервер Microsoft R (автономный)** является частью платформы аналитики корпоративного уровня, предоставляемые Майкрософт R.  
  
 Он расширяет R, предоставляя поддержку, масштабируемости и безопасности. [!INCLUDE[rsql_platform](../../includes/rsql-platform-md.md)] также устраняет ограничения в памяти R с открытым исходным путем добавления параллельного и частями обработки данных, позволяя пользователям анализировать данные в масштабе предприятия и использовать данные, намного больше, чем может поместиться в памяти.

 Дополнительные сведения о настройке Windows server для поддержки приложений R см. в разделе [создать отдельный сервер R](../../advanced-analytics/r-services/create-a-standalone-r-server.md).

+ [Подготовка виртуальной машины сервера R](../../advanced-analytics/r-services/provision-the-r-server-only-sql-server-2016-enterprise-vm-on-azure.md)
