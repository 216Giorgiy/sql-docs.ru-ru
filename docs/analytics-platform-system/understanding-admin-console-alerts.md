# <a name="understanding-admin-console-alerts-in-analytics-platform-system"></a>Основные сведения об оповещениях консоль администрирования в система платформы аналитики
Оповещения отображаются в устройстве **консоли администрирования** и **System Center Operations Manager** (SCOM). Используйте этот список оповещений, чтобы определить, какие оповещения требуют дополнительного изучения.  
  
Сведения о подключении к **консоли администрирования** с помощью Internet Explorer, в разделе [отслеживать устройства с помощью консоли администрирования &#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-the-admin-console.md). Сведения о **SCOM**, в разделе [отслеживать устройства с помощью System Center Operations Manager &#40;система платформы аналитики&#41;](monitor-the-appliance-by-using-system-center-operations-manager.md)  
  
Дополнительные сведения о получении сведений о предупреждениях, с помощью Transact\-SQL, в разделе [отслеживать устройства с помощью системных представлений &#40;Analytics Platform System&#41;](monitor-the-appliance-by-using-system-views.md).  
  
## <a name="types-of-alerts"></a>Типы оповещений  
Имена предупреждений, которые указывают на НОРМАЛЬНОЕ состояние обычно не требуется исследование. Имена предупреждений, которые содержат слова NON_CRITICAL иногда требуется действие. Исследование является обязательным для всех других типов оповещений.
