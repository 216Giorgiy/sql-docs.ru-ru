# <a name="information-to-obtain-from-your-ihv"></a>Сведения о получении из вашего Оборудования
Если вашего оборудования независимых поставщиков Оборудования позволяет ваш новый SQL Server PDW appliance, они также предоставлять сведения о конфигурации оборудования устройства, они выполнена на устройстве. Необходимо будет эти сведения для управления вашим устройством.  
  
Ниже приведены сведения, которые обычно требуется, от к независимому поставщику Оборудования. В некоторых случаях требуется дополнительные или другие сведения. Обратитесь к независимому поставщику Оборудования, чтобы убедиться, что вся необходимая информация была передана вам с доставкой устройства.  
  
|||  
|-|-|  
|**Сведения или документа**|**Description**|  
|Спецификация (BOM)|Ведомости материалов список компонентов, включенных в устройстве. Эта информация необходима для подтверждения того, что все компоненты были доставлены.<br /><br />**Важно:** ведомости материалов должно включать весовые коэффициенты для каждого из узлов устройства и для каждого завершения для монтажа в стойку. Эта информация важна при планировании как обрабатывать и перемещать компоненты устройства, а за обеспечение центра обработки данных может вместить устройства. Если ваш Спецификации включает весовые коэффициенты узлов, убедитесь, что для получения этих сведений из вашего Оборудования для всех узлов.|  
|Схемы кабелей|Кабелей приведены способы подключения к сети, управления питанием и другие кабели для каждого устройства для установки в стойку. Эти схемы необходимы при установке устройства в центре обработки данных, и каждый раз необходимо удалить или заменить компонент.|  
|Требования к racking устройства|Прежде чем устройства можно установить в центре обработки данных, необходимо знать, если центр данных соответствует требованиям к воздушный поток и длина кабеля требования для устройства, а также размер и питания для компонентов. См. также спецификации (Спецификация) выше сведения об устройстве вес запчастей, который также является обязательным.|
