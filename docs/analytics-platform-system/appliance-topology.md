# <a name="appliance-topology"></a>Топология устройства
Отображает сводную информацию на весь Analytics Platform System.  
  
Разверните элементы этого раздела, чтобы настроить сведения, применимые ко всей устройства. Выберите топологию хранилища параллельных данных или топологии HDInsight разделы для настройки сведения, относящиеся к только что области устройства.
