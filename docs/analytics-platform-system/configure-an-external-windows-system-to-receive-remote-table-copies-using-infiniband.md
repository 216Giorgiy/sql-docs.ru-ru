# <a name="configure-an-external-windows-system-to-receive-remote-table-copies-using-infiniband---parallel-data-warehouse"></a>Настройка внешней системы Windows для получения копии удаленной таблицы с помощью InfiniBand - Parallel Data Warehouse
Описывает, как приобрести и настроить систему Windows не является специализированным, подключенных с помощью сети InfiniBand для использования с функцией копирования удаленной таблицы в SQL Server PDW. Система Windows будет размещаться база данных SQL Server, который получает копию удаленной таблицы из базы данных SQL Server PDW. Он приобретается отдельно от устройства, подключен к сети InfiniBand устройства.  
  
> [!NOTE]  
> Подключение через сети InfiniBand не требуется с помощью функции копирования удаленной таблицы. Подключение через сеть Ethernet возможна, если пропускная способность Ethernet вашим потребностям.  
  
В этом разделе описывается один из шагов конфигурации для настройки копирования удаленной таблицы. Список всех шагов см. в разделе [удаленной копии таблицы](remote-table-copy.md)  
  
## <a name="before-you-begin"></a>Перед началом  
Перед настройкой внешней системы Windows, необходимо выполнить следующее:  
  
1.  Приобретения или предоставить системы Windows, который получит удаленных копий.  
  
2.  Для установки в стойку системы Windows в стойку управления (если имеется достаточно свободного пространства) или как можно ближе к устройству, чтобы вы могли подключаться к сети InfiniBand устройства.  
  
3.  Приобрести InfiniBand кабели и сетевой адаптер InfiniBand у поставщика оборудования устройства. Мы рекомендуем сетевой адаптер с двумя портами для обеспечения отказоустойчивости при получении экспортированных данных. Два порта сетевого адаптера, рекомендуется, но не является обязательным.  
  
## <a name="HowToWindows"></a>Настройки Windows внешние системы для получения копии удаленной таблицы  
Чтобы настроить внешней системы Windows, выполните следующие действия:  
  
1.  Установка сетевого адаптера InfiniBand в системе Windows.  
  
2.  Подключите сетевой адаптер InfiniBand главный выключатель InfiniBand в стойку элемента управления с помощью кабелей InfiniBand.  
  
3.  Установите и настройте Windows драйвер для сетевого адаптера InfiniBand.  
  
    InfiniBand драйверов для Windows разработанный Alliance OpenFabrics, Консорциум InfiniBand поставщиков.  Драйвер завершенных, выбрав свой адаптер InfiniBand. В противном случае драйвер можно загрузить из www.openfabrics.org.  
  
4.  Настройка IP-адрес для каждого порта для адаптера. SMP систем должны использовать статические IP-адреса из диапазона адресов, зарезервированных для этой цели. Настройте первый порт в соответствии со следующими параметрами:  
  
    -   Сетевой IP-адрес: 172.16.132.x  
  
    -   Маска подсети: 255.255.128.0  
  
    -   Узел IP-диапазон: 1 – 254  
  
    Для сетевых адаптеров InfiniBand с двумя портами настройте второй порт в соответствии со следующими параметрами:  
  
    -   Сетевой IP-адрес: 172.16.132.x  
  
    -   Маска подсети: 255.255.128.0  
  
    -   Узел IP-диапазон: 1 – 254  
  
5.  Если используется адаптер два порта или несколько внешних систем Windows установлено соединение с устройством, назначьте каждой системе номер другой узел в каждой IP-подсети.
